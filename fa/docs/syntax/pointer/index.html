ЩҫЩҲЫҢЩҶШӘШұ

Щ…ШҜЩ„ ЩҶЩҲШ№ ШӯШ§ЩҒШёЩҮ ШөШұЫҢШӯ WaveвҖӢ

Ш·ШұШ§ШӯЫҢ ЩҫЩҲЫҢЩҶШӘШұ ШҜШұ Wave ШЁШұ Ш§ШіШ§Ші Щ…ШҜЩ„ ЩҶЩҲШ№ ШӯШ§ЩҒШёЩҮ ШөШұЫҢШӯ Wave Ш§ШіШӘ. Ш§ЫҢЩҶ Щ…ШҜЩ„ ЩҮШҜЩҒШҙ ШӘШ№ШұЫҢЩҒ Ш§ШҙШ§ШұЩҮвҖҢЪҜШұЩҮШ§ ЩҲ ШўШұШ§ЫҢЩҮвҖҢЩҮШ§ ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ ЩҶЩҲШ№ ШӯШ§ЩҒШёЩҮ ШөШұЫҢШӯ ШҜШұ ШіШ·Шӯ ШІШЁШ§ЩҶ Ш§ШіШӘШҢ ЩҶЩҮ ШӘШұЩҒЩҶШҜЩҮШ§ЫҢ ШҜШіШӘЩҲШұЫҢ ЫҢШ§ Ш§ЩҶШӘШІШ§Ш№вҖҢЩҮШ§ЫҢ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢШ§ЫҢ.

ШЁШ§ Ш§ЫҢЩҶ Ш·ШұШ§ШӯЫҢШҢ ШҜШұ WaveШҢ Ш§ШҙШ§ШұЩҮвҖҢЪҜШұ ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ ЩҶЩҲШ№ ptr<T> ШЁЫҢШ§ЩҶ Щ…ЫҢвҖҢШҙЩҲШҜ ШӘШ§ ШұЩҲШҙЩҶ Ъ©ЩҶШҜ Ъ©ЩҮ Ш§ЫҢЩҶ ЩҶЩҲШ№ШҢ ШўШҜШұШіЫҢ Ш§ШІ ШӯШ§ЩҒШёЩҮ Ш§ШіШӘ Ъ©ЩҮ Щ…ЩӮШҜШ§ШұЫҢ Ш§ШІ ЩҶЩҲШ№ Ш®Ш§Шө T ШұШ§ ЩҶЪҜЩҮ Щ…ЫҢвҖҢШҜШ§ШұШҜ. Ш§ЫҢЩҶ ШұЩҲЫҢЪ©ШұШҜ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ Ъ©ЩҮ Ш§ШҙШ§ШұЩҮвҖҢЪҜШұ ШЁШ®ШҙвҖҢЩҮШ§ЫҢЫҢ Ш§ШІ ШіЫҢШіШӘЩ… ШӘШ§ЫҢЩҫ ШЁШ§ШҙШҜШҢ ЩҶЩҮ Ш№Щ…Щ„ЪҜШұ ЫҢШ§ ШҜШіШӘЩҲШұ ШІШЁШ§ЩҶЫҢШҢ ШӘШ§ ШіШ§Ш®ШӘШ§Шұ ШӯШ§ЩҒШёЩҮ ШұШ§ ШЁЩҮвҖҢШөЩҲШұШӘ ШҙЩҮЩҲШҜЫҢвҖҢШӘШұ ЩҲ ЫҢЪ©ЩҫШ§ШұЪҶЩҮвҖҢШӘШұ ЩҶЩ…Ш§ЫҢШҙ ШҜЩҮШҜ.

ШҜШұ WaveШҢ Ш§ШҙШ§ШұЩҮвҖҢЪҜШұЩҮШ§ ШҜШ§ШұШ§ЫҢ ШӘШ§ЫҢЩҫ ШөШұЫҢШӯ ШЁЩҮ ШҙЪ©Щ„ ptr<T> ЩҮШіШӘЩҶШҜ. ШЁШұШ§ЫҢ ШЁЩҮвҖҢШҜШіШӘ ШўЩҲШұШҜЩҶ ШўШҜШұШі Ш§ШІ & ЩҲ ШЁШұШ§ЫҢ Ш§ШұШ¬Ш§Ш№вҖҢШҜЩҮЫҢ Щ…Ш¬ШҜШҜ Ш§ШІ deref Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.

Ш§Ш№Щ„Ш§ЩҶ ЩҲ Щ…ЩӮШҜШ§ШұШҜЩҮЫҢ Ш§ЩҲЩ„ЫҢЩҮвҖӢ

var x: i32 = 10;
var p: ptr<i32> = &x;

ЩҶЩҲШ№ Ш§ШҙШ§ШұЩҮвҖҢЪҜШұЩҮШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШӘЩҲШҜШұШӘЩҲ ШЁШ§ШҙШҜ.

var p1: ptr<i32> = &x;
var p2: ptr<ptr<i32>> = &p1;

Ш§ШұШ¬Ш§Ш№вҖҢШҜЩҮЫҢ Щ…Ш¬ШҜШҜвҖӢ

var x: i32 = 10;
var p: ptr<i32> = &x;

println("{}", deref p); // 10
deref p = 20;
println("{}", x);       // 20

ЩӮЩҲШ§Ш№ШҜ Щ„ЫҢШӘШұШ§Щ„ nullвҖӢ

null ЫҢЪ© Щ„ЫҢШӘШұШ§Щ„ ШұШіЩ…ЫҢ Щ…ЫҢвҖҢШЁШ§ШҙШҜ. Ш§ЫҢЩҶ ЫҢЪ© ШҙЩҶШ§ШіЩҮ ЩҶЫҢШіШӘ ЩҲ ЩҶЩ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ЩҶШ§Щ… Щ…ШӘШәЫҢШұ Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙЩҲШҜ.

ЩӮЩҲШ§Ш№ШҜ Ш§ШөЩ„ЫҢ:

  • null ШӘЩҶЩҮШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЩҮ Щ…ЩӮШөШҜ ptr<T> Ш§Ш®ШӘШөШ§Шө ЫҢШ§ШЁШҜ.
  • ШЁЩҮ ЩҶЩҲШ№вҖҢЩҮШ§ЫҢ ШәЫҢШұШ§ШҙШ§ШұЩҮвҖҢЪҜШұЫҢ Щ…Ш§ЩҶЩҶШҜ i32ШҢ boolШҢ array<...> ЩҶЩ…ЫҢвҖҢШӘЩҲШ§ЩҶ ШӘШ®ШөЫҢШө ШҜШ§ШҜ.
  • ЩҶЩ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШҙШ§ШұЩҮвҖҢЪҜШұ ШұШ§ ШЁШ§ Щ„ЫҢШӘШұШ§Щ„вҖҢЩҮШ§ЫҢ Ш№ШҜШҜ ШөШӯЫҢШӯЫҢ Щ…Ш§ЩҶЩҶШҜ (0, 123, -1 ЩҲ ШәЫҢШұЩҮ) Щ…ЩӮШҜШ§ШұШҜЩҮЫҢ Ш§ЩҲЩ„ЫҢЩҮ Ъ©ЩҶЫҢШҜ. ШЁЩҮвҖҢШ·ЩҲШұ ШөШұЫҢШӯ Ш§ШІ null Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.
var p: ptr<i32> = null;
var arrp: ptr<array<i32, 3>> = null;

// var n: i32 = null;  // ERROR
// var b: bool = null; // ERROR

ШӯШіШ§ШЁ Ш§ШҙШ§ШұЩҮвҖҢЪҜШұвҖӢ

Wave Ш§ШІ Щ…ШӯШ§ШіШЁШ§ШӘ Ш§ШҙШ§ШұЩҮвҖҢЪҜШұЫҢ ШІЫҢШұ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

  • ptr + int: ЩҫЫҢШҙШұЩҲЫҢ Ш§ШҙШ§ШұЩҮвҖҢЪҜШұ Щ…ШЁШӘЩҶЫҢ ШЁШұ GEP
  • ptr + int: Ш№Щ…Щ„ЫҢШ§ШӘ Щ…ШҙШ§ШЁЩҮ
  • int + ptr: Ш№ЩӮШЁвҖҢШұЩҲЫҢ Ш§ШҙШ§ШұЩҮвҖҢЪҜШұ Щ…ШЁШӘЩҶЫҢ ШЁШұ GEP
  • ptr - ptr: Щ…ШӯШ§ШіШЁЩҮ ШӘЩҒШ§ЩҲШӘ ШЁШ§ЫҢШӘ ШҜШұ i64

ЩҶЪ©ШӘЩҮ:

  • ptr<T> +/- n ШЁШұШ§ШіШ§Ші Ш§ЩҶШҜШ§ШІЩҮ T (sizeof(T)) ШӯШұЪ©ШӘ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
  • ШЁЩҮвҖҢШ№ШЁШ§ШұШӘ ШҜЫҢЪҜШұШҢ ptr<i32> + 3 ШЁЩҮвҖҢШ§ЩҶШҜШ§ШІЩҮ +12 ШЁШ§ЫҢШӘ ШӯШұЪ©ШӘ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
var base: ptr<i32> = 0x1000 as ptr<i32>;

var p1: ptr<i32> = base + 3; // 0x1000 + 12
var p2: ptr<i32> = 2 + base; // 0x1000 + 8
var p3: ptr<i32> = base - 1; // 0x1000 - 4

var diff: i64 = p1 - base;   // 12 (byte diff)

Щ…ЩӮШ§ЫҢШіЩҮ Ш§ШҙШ§ШұЩҮвҖҢЪҜШұвҖӢ

Ш§ШҙШ§ШұЩҮвҖҢЪҜШұЩҮШ§ ШұШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ ШЁШұШ§ЫҢ Щ…ЩӮШ§ЫҢШіЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ.

if (p == null) { ... }
if (p != null) { ... }
if (p1 == p2) { ... }

Ш§ШұШӘШЁШ§Ш· ШЁШ§ ШўШұШ§ЫҢЩҮвҖҢЩҮШ§вҖӢ

ШўШұШ§ЫҢЩҮ Ш§ШҙШ§ШұЩҮвҖҢЪҜШұ:

var a: i32 = 10;
var b: i32 = 20;
var arr: array<ptr<i32>, 2> = [&a, &b];
println("{} {}", deref arr[0], deref arr[1]);

Ш§ШҙШ§ШұЩҮвҖҢЪҜШұ ШўШұШ§ЫҢЩҮ:

var p: ptr<array<i32, 3>> = &[1, 2, 3];
if (p != null) {
    println("{}", deref p[1]);
}

ЩҶЪ©Ш§ШӘ Ш§ЫҢЩ…ЩҶЫҢвҖӢ

ШҜШұ ШӯШ§Щ„ ШӯШ§Ш¶ШұШҢ Wave ШҜШ§ШұШ§ЫҢ Щ…ШҜЩ„ Ш§ЫҢЩ…ЩҶЫҢ ШЁШұШ§ЫҢ Ш§ШҙШ§ШұЩҮвҖҢЪҜШұЩҮШ§ Щ…ШЁШӘЩҶЫҢ ШЁШұ Щ…Ш§Щ„Ъ©ЫҢШӘ/Ш·ЩҲЩ„ Ш№Щ…Шұ Щ…Ш§ЩҶЩҶШҜ Rust ЩҶЫҢШіШӘ. ШЁЩҶШ§ШЁШұШ§ЫҢЩҶШҢ Ш§ШұШ¬Ш§Ш№вҖҢШҜЩҮЫҢ Щ…Ш¬ШҜШҜ null ШЁЩҮвҖҢШ·ЩҲШұ Ш®ЩҲШҜЪ©Ш§Шұ Щ…ШіШҜЩҲШҜ ЩҶЩ…ЫҢвҖҢШҙЩҲШҜ. Ш§Щ„ЪҜЩҲЫҢ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ ЪҶЪ©вҖҢЪ©ШұШҜЩҶ deref ШЁЩҮвҖҢШөЩҲШұШӘ ШөШұЫҢШӯ ЩӮШЁЩ„ Ш§ШІ null ШӘЩҲШөЫҢЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.
