Hello Wave प्रिंट करना

यह दस्तावेज़ Wave भाषा में सबसे बुनियादी आउटपुट प्रोग्राम लिखने के तरीके को समझाता है।

उदाहरण कोड​

fun main() {
    println("Hello Wave");
}

विवरण​

  • fun main()

    Wave प्रोग्राम के प्रवेश बिंदु का फ़ंक्शन है। यह सबसे पहले निष्पादित होता है।

  • println()

    यह एक अस्थायी प्रिंट फ़ंक्शन है जो स्ट्रिंग आउटपुट करता है, और आउटपुट के बाद एक नई पंक्ति (\n) जोड़ता है।

  • ; (सेमीकोलन)

    Wave में सभी कथन सेमीकोलन से समाप्त होते हैं।

निष्पादन परिणाम​

Hello Wave

अतिरिक्त उदाहरण​

Wave स्ट्रिंग इंटरपोलेशन का समर्थन करता है।

fun main() {
    var name: str = "Wave";
    println("Hello, {}!", name);
}

आउटपुट:

Hello, Wave!

यह उदाहरण Wave मानक लाइब्रेरी के बुनियादी आउटपुट फ़ंक्शन के उपयोग को दर्शाता है।
