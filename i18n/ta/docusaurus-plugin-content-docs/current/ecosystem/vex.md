# வெக்ஸ் தொகுப்பு மேலாளர்

## சுருக்கம்

வெக்ஸ் வேவ் நிரலாக்க மொழிக்கான தனிச்சொத்து மேலாளர் மற்றும் கட்டமைப்பு அமைப்பாகும்.
வெக்ஸ் மூல உரை சார்ந்த பராமரிப்பு, கட்டமைப்பு அமைப்பு, இலக்கு தளம் குறித்த வளர்ச்சியை, தொகுதிகள் நிறுவல் மற்றும் விநியோகம் போன்ற திட்ட மேலாண்மையை ஆதரிக்கிறது.
வெளியீட்டு மொழி அல்லது அமைப்புகளுடன் பொருந்தக்கூடியவற்றை கருத்தில் கொள்ளாமல், வைவு சூழலில் இயங்கிட வடிவமைக்கப்பட்டுள்ளது.

## வடிவமைப்பு குறிக்கோள்

வெக்ஸ் பின்வரும் குறிக்கோளின் அடிப்படையில் வடிவமைக்கப்பட்டுள்ளது:

வேவ் தனியம் வடிவம்: வைவு செயல்பாடுகளை மட்டும் குறிக்கோளாகக் கொண்டுள்ள, வாக்களின் இலக்கமுழு, தொகுதி அமைப்பு, செயலாக்க சூழலில் சேர்க்கப்பட்டுள்ளது.

- நேரடியான கட்டளை கட்டமைப்பு: சிக்கலான கட்டமைப்பு உரு இன்றி ஒரு கட்டளையில் முக்கிய ஆக்கங்களை செய்ய முடியும்.
- பல இலக்கு ஆதரவு: இயக்க அமைப்பின் மற்றும் கட்டமைப்புக்கு ஏற்ப இலக்கு கோப்புகளை மாற்றுவது எளிதாக.
- WSON அடித்தளம் அமைப்பு பராமரிப்பு: அனைத்து திட விளக்கம் சேர தயார் WSON (வேவ் ஒழுங்காக்கல் பொருள் குறிப்பன) வடிவத்தில் உருவாக்கப்பட்டுள்ளது.
- நிலையான உருவாக்க மற்றும் விநியோகம்: செயல் கோப்புகள் நிலையாக உருவாக்கப்படுகின்றன, வெளியர்கமள சந்தர்ப்பத்தில் சார்ந்தாது சொந்தமாகச் செய்யலாம்.
