# Vex தொகுப்பு மேலாளர்

## கண்ணோட்டம்

Vex என்பது Wave நிரலாக்க மொழிக்கான பிரத்யேக தொகுப்பு மேலாளர் மற்றும் உருவாக்க அமைப்பு ஆகும்.
Vex ஒட்டுமொத்த திட்ட நிர்வாகத்தை ஆதரிக்கிறது, இதில் மூல குறியீடு சார்பு மேலாண்மை, உருவாக்க அமைப்புகள், இலக்கு இயங்குதள பதவி மற்றும் தொகுதி நிறுவல் மற்றும் வரிசைப்படுத்தல் ஆகியவை அடங்கும்.
வெளிப்புற மொழிகள் அல்லது அமைப்புகளுடன் பொருந்தக்கூடிய தன்மை கருதப்படாது, மேலும் இது Wave சுற்றுச்சூழல் அமைப்பிற்குள் மட்டுமே செயல்படும் வகையில் வடிவமைக்கப்பட்டுள்ளது.

## வடிவமைப்பு இலக்குகள்
Vex பின்வரும் இலக்குகளை மனதில் கொண்டு வடிவமைக்கப்பட்டது:

Wave க்காக பிரத்யேகமாக வடிவமைக்கப்பட்டது: Wave திட்டத்தில் மட்டுமே இலக்காகக் கொண்டது, இது Wave இன் தொடரியல், தொகுதி அமைப்பு மற்றும் செயல்படுத்தும் சூழலுக்கு உகந்ததாக உள்ளது.

* உள்ளுணர்வு கட்டளை அமைப்பு: சிக்கலான பில்ட் ஸ்கிரிப்டுகள் இல்லாமல், ஒரே கட்டளையுடன் முக்கிய பணிகளைச் செய்ய ஒழுங்கமைக்கப்பட்டது.
* பல-இலக்கு ஆதரவு: இயக்க முறைமை மற்றும் கட்டமைப்பைப் பொறுத்து நீங்கள் எளிதாக உருவாக்க இலக்குகளை மாற்றலாம்.
* WSON அடிப்படையிலான உள்ளமைவு மேலாண்மை: அனைத்து திட்ட கட்டமைப்பு தகவல்களும் WSON (Wave வரிசைப்படுத்தல் பொருள் குறிப்பீடு) வடிவத்தில் வரையறுக்கப்பட்டுள்ளது.
* நிலையான உருவாக்கம் மற்றும் வரிசைப்படுத்தல்: இயங்கக்கூடியவை நிலையான முறையில் கட்டமைக்கப்படுகின்றன மற்றும் வெளிப்புற இயக்க நேரத்தை நம்பாமல் சுயாதீனமாக பயன்படுத்தப்படலாம்.
