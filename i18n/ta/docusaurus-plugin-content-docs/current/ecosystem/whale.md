# வெயில் தொகுதி கருவிப் பட்டி

## சுருக்கம்

வெயில், வைவு நிரலாக்ககண்டுவிப்பதற்கான தனிச்சொத்து கருவிப் கட்டமைப்பாகும்.
வெயில் வைவில் எழுதப்பட்ட மூல உரைகள் புரிவது, வருங்காலம் மாற்ற வசதியான கிளை மற்றும் இலக்கு தளத்தில் வருகிறது என்பதைச் செய்துண்கிறது.
இந்த கருவியமைப்பும் வெவ்வேறு மொழி ஆதரவு அல்லது வெளியர்க்குவை ஓப்புசீட்டதிற்கான தனியடையாளி சூதாக்ச்சி விவகாரத்தை புறக்கண்டுள்ளது.

## வடிவமைப்பு குறிக்கோள்

வெயில் முக்கிய வடிவமைப்பு குறிக்கோளை அடிப்படை நிலை கூறுகிறது:

- வைவு ஆர்வ ஆதரவு: வேயில் வைவு மொழிக்கே உத்தேசமான ஆதரவு வழங்குகிறது, மற்றொரு மொழிகளோடு ஒருங்கிணைக்கப்படவில்லை.
- தொகுதி அமைப்பு: ஒவ்வொரு அமையவும் தன்னிச்சையாக இருக்கும் தொகுதி உருவாக்கப்பட்டுள்ளது, தேவைப்படும் போது இணைக்கிறது அல்லது அன்னைக் கடந்து செல்கிறது.
- சுயாதீன IR பயன்பாடு: வேலே LLVM IR விட்டுவிடவும், அல்லது பிற வெளிப்புற IR களை பயன்படுத்தாமல், அதன் சொந்த இடைமுகக் காட்சியை வரையறுக்கின்றது.
- பல இலக்கு தளம் ஆதரவு: இயக்க முறைமைகள் மற்றும் மென்பொருள் வடிவமைப்புகளுடன் தொடர்பில்லாமல் பல்வேறு சூழல்களை இலக்காகக் கொண்டு கட்டமைக்க முடியும்.
- பிரத்யேக மையக்கோணம்: குழுமத்தின் முழு செயல்முறையை மேம்படுத்துபவர் விரிவானவாறு கட்டுப்படுத்த முடியும்.
- வெளிப்புற சார்ந்த தன்மைகளை நீக்குதல்: Whale வெளிப்புற C/C++ இயக்க நேரம் அல்லது தொகுப்பாளருக்கு சார்ந்துள்ளதில்லை.

## இலக்குகள் ஆதரவு

வேலே பின்வரும் இலக்கு சூழல்களுக்கு ஆதரவு வழங்கும் குறிக்கோளாக உள்ளது:

- இயக்க முறைமை:
  - Linux
  - Windows
  - macOS
  - UEFI (BIOS நீக்கம்)
  - WaveOS (சுய உருவாக்க OS)
- வடிவமைப்பு:
  - x86_64 (AMD64)
  - ARM64
  - மற்றவை தொகுதிகளைச் சேர்த்து விரிவாக்கக்கூடியவை

## வெளிப்புற இணைப்பு(FFI)

வேல் தொழில்நுட்ப ரீதியாக FFI (ஆகமீன் செயல்பாட்டு இடைமுகம்)க்கு ஆதரவு அளிக்க உருவாக்கப்பட்டுள்ளது, ஆனால் Wave தொடர்வு வெளிப்புற மொழிகளுடன் பரிந்துரைக்கப்படவில்லை மற்றும் நிலையான முறையில் வழங்கப்படவில்லை.
வேவ் தனது சொந்த மொழியில் அனைத்து செயல்பாடுகளையும் செயல்படுத்த உருவாக்கப்பட்டது.

## விரிவாக்கக் கோளை

Whale பின்வரும் வகையிலே விரிவாக்கப்படலாம்:

- புதிய இயக்க முறைமையுடன் அல்லது வடிவமைப்புடன் தொடர்புடைய தொகுதிகளைச் சேர்த்தல்
- பயனர் வரையறுக்கப்பட்ட செயலுத்திறன் முறைகணிதத்தைச் சேர்த்தல்
- கட்டமைப்பு சுயவிவரங்கள் மற்றும் இணைக்குநர் அமைப்புகளை தனிப்பயனாக்குதல்
- சுய இயக்க வடிவத்தை வரையறுத்தல்
