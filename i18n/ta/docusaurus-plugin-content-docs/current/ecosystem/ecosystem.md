# Wave சுற்றுச்சூழல் அமைப்பு

## கண்ணோட்டம்

Wave சுற்றுச்சூழல் அமைப்பு நிரலாக்க மொழிகளை விட அதிகம்.
Wave பல்வேறு கருவிகள், நூலகங்கள் மற்றும் மேம்பாட்டுத் தத்துவங்களை ஒருங்கிணைக்கும் சூழலை வழங்குகிறது.
இது ஒரு விரிவான வளர்ச்சி சுற்றுச்சூழல் அமைப்பாகும், இது புரோகிராமர்கள் எந்தத் துறையிலும் ஆக்கப்பூர்வமாக வேலை செய்ய உதவுகிறது.

## Wave சுற்றுச்சூழல் அமைப்பின் கலவை
### Wave
Wave சுற்றுச்சூழல் அமைப்பின் மையமானது Wave நிரலாக்க மொழியாகும்.
C மற்றும் Rust இன் தத்துவத்தின் அடிப்படையில், இது குறைந்த-நிலை கட்டுப்பாடு மற்றும் உயர்-வரிசை உற்பத்தித்திறன் இடையே சமநிலையை ஏற்படுத்த வடிவமைக்கப்பட்டுள்ளது.

* **எளிதான கற்றல் வளைவு**: C மற்றும் Rust இடையே நடுத்தர சிரமம்.
* ** சக்திவாய்ந்த அம்சங்கள்**: சுட்டிகள், அணிவரிசைகள் மற்றும் வரிசைப்படுத்தல் போன்ற பல்வேறு தரவு வகைகளை ஆதரிக்கிறது.

### நிலையான நூலகம்
Wave நிலையான நூலகங்களின் செல்வத்தை வழங்குகிறது, இதனால் புரோகிராமர்கள் தனி வெளிப்புற நூலகங்களை நம்ப வேண்டிய அவசியமில்லை.

* **வன் அலை**: வன்பொருள் கட்டுப்பாடு
* **http**: HTTP மற்றும் WebSocket தொடர்பு
* **iosys**: கோப்பு மற்றும் துண்டு I/O
* **sys**: கணினி கட்டுப்பாடு
* **wson**: உயர் செயல்திறன் தரவு வரிசைப்படுத்தல் வடிவம்
* **குவாண்டம்**: குவாண்டம் கம்ப்யூட்டிங் ஆதரவு

### சமூகம் மற்றும் சுற்றுச்சூழல் தத்துவம்
Wave டெவலப்பர்களுடனான தொடர்புக்கு மதிப்பளிக்கிறது, மொழி மட்டுமல்ல.

* **டெவலப்பர்-மைய வடிவமைப்பு**: பல்வேறு மொழிகள் மற்றும் கலாச்சாரங்களை மதிப்பது.
* **சிறுபான்மை மற்றும் இறந்த மொழிகளுக்கான ஆதரவு**: உலகளாவிய மொழியியல் பன்முகத்தன்மையை தழுவுதல்.
* **எதிர்கால ஆதார தொழில்நுட்பம்**: குவாண்டம் கம்ப்யூட்டிங் மற்றும் அடுத்த தலைமுறை பாதுகாப்பு தரநிலைகளுக்கான ஆதரவு.

### கருவி சங்கிலி
Wave முழுமையான மேம்பாட்டு கருவிகள் மூலம் உற்பத்தித்திறனை அதிகரிக்கிறது.

* **Wave தொகுப்பு மேலாளர் (Vex)**: பிளாட்ஃபார்ம்-சுயாதீனமான உருவாக்கம் மற்றும் வரிசைப்படுத்தல்.
* **Wave பிழைத்திருத்தி**: உள்ளுணர்வு பிழைத்திருத்தம்.

## Wave சுற்றுச்சூழல் அமைப்பின் இலக்குகள்
Wave சுற்றுச்சூழல் அமைப்பு பின்வரும் மதிப்புகளைச் சுற்றி உருவாகிறது:

1. **பன்முகத்தன்மையை உள்ளடக்கிய மொழி**: அனைத்து டெவலப்பர்களும் தங்கள் சொந்த மொழி மற்றும் கலாச்சாரத்தின்படி Wave ஐப் பயன்படுத்துவதற்கு துணைபுரிகிறது.
2. **அனைத்து வளர்ச்சிப் பகுதிகளின் ஒருங்கிணைப்பு**: வலை, வன்பொருள், AI மற்றும் நெட்வொர்க் போன்ற பல்வேறு துறைகளில் பயன்படுத்தப்படுகிறது.
3. **எதிர்காலத்திற்குத் தயாராகுதல்**: குவாண்டம் கம்ப்யூட்டிங் மற்றும் பிந்தைய குவாண்டம் குறியாக்கம் போன்ற புதிய தொழில்நுட்பங்களுக்கு பதிலளிப்பது.
