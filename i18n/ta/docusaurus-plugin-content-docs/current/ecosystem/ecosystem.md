# Wave பரிட்சைகூறு

## சுருக்கம்

Wave பரிட்சைகூறு எளிய நிரலாக்க மொழியை மிஞ்சியுள்ளது.
Wave பல்வேறு கருவிகள், நூலகங்கள் மற்றும் மேம்பாட்டு தத்துவம் இணைந்த சூழலாக இருக்கின்றது, எதில் இருந்தாலும் நிரலர் படைப்பாற்றலுடனான வேலை செய்ய உதவுகிறது.

## Wave பரிட்சைகுறியின் அமைப்பு

### Wave

Wave பரிட்சைகுறியின் மையம் Wave நிரலாக்க மொழியாகும்.
C மற்றும் Rust தத்துவங்களின் அடிப்படையில், குறைந்த நிலை கட்டுப்பாடு மற்றும் உயர்ந்த உற்பத்தித்தன்மையிடையே சமநிலையை அடைவதை நோக்கமாகக் கொண்டு வடிவமைக்கப்பட்டுள்ளது.

- **எளிய கற்றல் வளைவு**: C மற்றும் Rust இடையே மத்தியமச்சிக்கல்.
- **வலுவான செயல்பாடுகள்**: பாய்ன்டர், வரிசை, தொடருதல் போன்ற பல்வேறு தரவுவகைகள் ஆதரவு.

### நிலையான நூலகம்

Wave நிரலர்கள் வெளிப்புற நூலகங்களின் மீதான சார்பில்லாமல் இருக்க நிலைபேறு நூலகங்களை வழங்குகிறது.

- **hardwave**: கருவியியல் கட்டுப்பாடு
- **http**: HTTP மற்றும் WebSocket தொடர்பு
- **iosys**: கோப்பு மற்றும் வருக்கு I/O
- **sys**: முறைமைக் கட்டுப்பாடு
- **wson**: உயர் செயல்திறன் தரவு தொடரமைப்பு வடிவமைப்பு
- **quantum**: குவாண்டம் கணினி ஆதரவு

### சமூகமும் பரிட்சைகுறி தத்துவமும்

Wave மொழியுடன் மட்டுமன்றி, மேம்பாட்டாளருடனான பரஸ்பர தொடர்பையும் முக்கியமாகக் கருதுகிறது.

- **மேம்பாட்டாளர் மையக்குரல்**: பல்வேறு மொழிகளை மற்றும் பண்பாட்டை மதிப்பு.
- **சொல் குறைவுகளான மொழிகள் மற்றும் மரித்த மொழிகளின் ஆதரவு**: உலகளாவிய மொழி பரந்தமையை ஆதரிக்கிறது.
- **எதிர்கால நோக்கிய தொழில்நுட்பம்**: குவாண்டம் கணினி மற்றும் அடுத்த தலைமுறை பாதுகாப்பு நிலைப்பாடு ஆதரவு.

### கருவிகள் சங்கிலி

Wave மேம்பாட்டு கருவிகளின் முழுமையினால் உற்பத்தித்தன்மையை அதிகரிக்கிறது.

- **Wave தொகுப்பாளி மேலாளர் (Vex)**: தளம் சாரா கட்டமைப்பு மற்றும் விநியோகம்.
- **Wave பிழைத்திருத்தி**: உடனடிய கிழித்தல்.

## Wave பரிட்சைகுறியின் இலக்கம்

Wave பரிட்சைகுறை பின்வரும் மதிப்புகளை மையமாக வைத்து வளர்கிறது:

1. **பனுவல் ஆதரவு மொழி**: எல்லா மேம்பாட்டாளரும் தங்கள் மொழி மற்றும் பண்பாட்டிற்கு ஏற்ப Wave பயன்படுத்த முடியும்.
2. **அனைத்து மேம்பாட்டு பகுதிகளின் இணைப்பு**: வலை, கருவியியல், AI, வலையமைப்பு போன்ற பல்வேறு துறைகளில் பயன்படுத்தப்படுகின்றது.
3. **எதிர்காலத்திற்கு தயார்**: குவாண்டம் கணினி, அடுத்த தலைமுறை பின்-குவாண்டம் மறையியல் போன்ற புதிய தொழில்நுட்ப முகாமேற்பு.
