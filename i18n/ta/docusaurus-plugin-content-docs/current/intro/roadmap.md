# Wave + Whale ஒருங்கிணைந்த வளர்ச்சி பாதை v2

இந்த ஆவணமானது ஒரு தளமாக இருப்பதாக விளக்கம் வீட்டு வளத்தை காந்தத்துடன் முடியும்.
அவர், அப்போது சங்கலிக்குரிய ஒரு மதிப்பு தூக்கத்திற்கு ஒரு கவனம் பெற முடியும்.

மறுபடியும், இரண்டு உள்ளங்கள்.

```matlab
முன்-அல்பா → முன்-பீட்டா → ஆல்பா → பிட்ட் → ஆர்.சி → வெளியீடு
```

அவர் வடிவமைப்படையை தெரித்து வரும்வரை.

---

## முன்-பீட்டா கட்டம்

முன்பு-பில棌க்கு கைதத்தை மீண்டும் முதலில் உறவுகளமாக வைத்த முயற்சிக்கின்றது.
இங்கு அவர்கள் வெளியிட்டதற்கு அவர்களைத் தேர்ந்தெடுக்குவதற்கான முயற்சி.

நாம் இங்கு வாய்ப்பு பெறலாம்.
அவர்களால், அவர்கள் முன்னணி வெதும்பரிய கலத்தை இந்தியாவில் நிரூபிக்கப்பட்டுள்ளது.
அவர் விவரத்தைச் செய்கின்றாலும், அவர் முதலில் மேம்பாட்டு சர்வதை அங்கீகாரம் அளிக்கின்றனர்.

இங்கு, களர்களுக்குமா மீண்டாலும் மற்றுமிடங்களுற்று இசைகிறார்கள்.
இவை, உத்திரவாதமாகவும் அங்கே ஒரே மாதிரியா.

அதனால், ஆலியாமுறவு செய்ய இப்படித்தான் பின்வற்றுதற்கு

---

## அல்பா கட்டம்

அடிப்படையால் இது, ஷேலின் செயல்முறை ஆய்வாகும்.
அந்தவற்றின் வரைவு மிகவும் சங்கரிக்காகவும் வசதியாகவே இருக்கிறது.

அதில் அவர் வழியே, அவர் சில புதிய தொகுக்கப்படும் என ஒரு உருக் பெறுவார்.

```bash
wavec run main.wave --backend=whale
wavec run main.wave --backend=llvm
```

இந்த கட்டத்தில், Whale IR அமைப்பைப் வடிவமைத்து விவரிக்கின்றோம்.
Instruction, Value, Block போன்று முக்கிய கூறுகளை எடுத்துக்கொண்டு
Wave AST யை Whale IR ஆக மாற்றும் IR உற்பத்தியாளரை உருவாக்குகின்றோம்.

Whale க்கு குறியீடு உருவாக்கி, assembly அல்லது binary வடிவத்தில் இயக்கக் கூடியதாக உருவாக்குகின்றோம்.
LLVM இல் மாற்றக்கூடிய மற்றும் அசாதாரணமான வகைகள், உதாரணமாக `i1024` போன்ற பெரிய முழு எண்ணின்முள்ள வகை
மேலும் முன்னணி இடத்தை Whale க்கு மீறிய функ்பம் உருவாக்கப்படுகிறது.

Whale Backend இல் Hello World பிரச்சினை நீங்கவேண்டும்,
மாறிகள் குறிப்பு, அழைப்புகளில் குறிப்பு, வாருங்கள், IR Debugging ஆதிகாரிகள் அனைவரும் மீறவேண்டும்.
Wave → Whale IR மாற்றம் வாய்மையாக நிகழ்கிறது.

## பீட்டா கட்டம்

Beta கட்டத்தின் நோக்கம் Whale க்கு முழுமையாக மாறுவது மற்றும் LLVM சார்ந்தசெயல்களைக் கொணடுத்து விடுவிக்கவுள்ளது.
இந்த கட்டத்திற்கு பிறகு Wave தொகுப்பு மற்றும் இயக்கம் மட்டும் Whale ஐப் பயன்படுத்துகிறது.

LLVM தொடர்பான சாப்புக்கள் மற்றும் உருப்படிகள் கொண்டே நீக்குகின்றன,
குறியீட்டின் உருவாக்கம் மற்றும் இயக்க பாதை Whale அடிப்படையில் சுலபமாக உள்ளது.
IR உருவாக்கத்திலிருந்து செயல்படுத்துதல் வரைச் சொத்துக்களைச் சின்னமாகவும் விரைந்து நடத்தவேண்டும்.

Whale IR க்கான சிறந்த பயன்பாட்டைப் பரினையுறை சுற்றி உருவாக்கிக்கொள்கிறோம்,
குறியீட்டு உருவாக்கத்தையும் மெதுபலம் වන விரைவுகளை முன்னேற்றுகின்றோம்.
Wave இன் அனைத்து நியாயங்கள் இந்த கட்டத்தில் Whale பின்புறத்தின் அடிப்படையில் சரியான விவராக்க வேண்டியவை.

சோதனை மற்றும் விரும்பிய தேர்வுகள், முழுமையான அதேவகை தேர்வுகளில் செயல்படுத்துகிறோம்,
WSON மற்றும் பட்டியல் வாழும் இல்லாத வாடி பயன்படுத்துகிறோம்.

## RC (வெளியீட்டு kandidat) கட்டம்

RC கட்டத்தின் நோக்கம் Wave க்கு காரணியாக உருவாக்கத்தை ஆரம்பிக்க வேண்டும்.
இந்த கட்டத்தில், Wave தொகுப்பாளரின் Rust படிகளைக் குறிப்பிட்டு கொண்டு, குறிபெயர் தொகுப்பாளரைக் கட்டமைப்பதற்கு ஆரம்பிக்கிறோம்.

Whale ஐ அடிப்படையாகக் கொண்டு Wave IR உற்பத்தியாளரை மீண்டும் எழுதி,
உரைமுறைகள் மற்றும் std / core நூலகங்களை Wave குறியீட்டுடன் சவாத்தரமாக்குகிறோம்.
இந்த செயல்முறை மூலம் Whale தற்காலிகமரீதியாக மாற்றல்குழுக்ககக் கொண்டுள்ளது.

Bootstrapping வெற்றிகரமாக இருந்தால், முதலாவது Wave-native நிரலாக்கக்குழு உருவாகும்.

## வெளியீட்டு கட்டம் (v0.0.1)

Release கட்டம் Wave இன் அதிகாரப்பூர்வ முதல் வெளியீட்டை குறிக்கின்றது.
இந்த நேரத்தில் Wave மற்றும் Whale முழுமையாக உள்ளே சிக்கல்களை உருவாக்குகின்றன.

Release அமைப்புகள் Wave மொழி மற்றும் அடிப்படை நூலகம்,
Whale காசோலைப் தொழில்நுட்பத்தை அங்கூறிய கொள்கங்களை உட்படுகிறது.

இந்த கட்டத்தின் Wave இரண்டு வேறு Wave குறியீட்டுடன் நிரலாக்கிய நிரல்களுடன் இருக்க வேண்டும்,
Whale அளவே வேலை கருத்திற்கே்ல இருப்ப ஒருவராகவே இருக்க வேண்டும்.
Vex மூலம் கட்டமைப்பு மற்றும் வெளியீட்டு நிகழ்வுகளை நம்பத்தகவு செய்து,
`vex build --windows` போன்ற சேவைகளுக்கு ஒரு குறுக்கிட தயாரா தொடங்குகிறது.

## Developement Meta Strategy

Wave + Whale உருவாக்கம் முகப்பில் சரியான ஆராய்ச்சி மூலம் வருகின்றது.
Whale உருவாக்கமாக, ஒரு வெளியீட்டு அரசையை உருவாக்குபவை கல்லூரியில் உருவாக்கப்பாட்டுக்கான நிலைகாட்டி வழங்க法ாக செய்கின்றன.

Alpha கட்டத்தில், `--backend` தேர்வின் மூலம் காலாண்மையை அமைய மொழிக்கு மிகுந்த முக்கியத்துவம் உள்ளது,
LLVM மற்றும் Whale இருக்கும் மொழியின் சம்பவங்களை சரி பார்க்கலாம்.

RC முறையாகக் கட்டமைப்புகொட்ட கடந்த விவாதங்களில் மறுபட்டதாகவே இருந்தாலும்,
Wave உறுப்புக்களை கொள்கிறது, அவர்கள் முதலில் மிகுசென்ற கட்டமைப்பு அமைக்கவேண்டும்.
