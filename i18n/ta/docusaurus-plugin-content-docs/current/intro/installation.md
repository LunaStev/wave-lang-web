# நிறுவல்

## நிறுவல் முறை

வழங்கப்பட்ட நிறுவல் ஸ்கிரிப்ட் மூலம் Wave ஐ நிறுவ முடியும்.
கீழே உள்ள கட்டளையை முனையத்தில் இயக்கினால், Wave கம்பைலரின் (`wavec`) குறிப்பிட்ட பதிப்பு தானாகவே நிறுவப்படும்.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version <version>
```

நிறுவல் ஸ்கிரிப்ட் கணினி சூழலைச் சரிபார்த்து, Wave ஐ இயக்கத் தேவையான சார்புகளையும் கம்பைலரையும் தானாகவே அமைக்கிறது.
நீங்கள் ஒரு பதிப்பைக் குறிப்பிடவில்லை என்றால், குறிப்பிட்ட அளவுகோல்களின்படி சமீபத்திய நிலையான பதிப்பு அல்லது இயல்புநிலை பதிப்பு நிறுவப்படும்.

## நிறுவல் உதாரணம்

சமீபத்திய பதிப்பை நிறுவ, இயக்கவும்:

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- latest
```

நீங்கள் ஒரு குறிப்பிட்ட பதிப்பை நிறுவ விரும்பினால், `--version` விருப்பத்தைப் பயன்படுத்தவும்.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version v0.1.3-pre-beta
```

இரவு கட்டங்கள் போன்ற விரிவான பதிப்புகளைக் குறிப்பிடவும் முடியும்.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version v0.1.3-pre-beta-nightly-2025-07-11
```

## நிறுவலின் போது என்ன நடக்கும்

Wave ஐ சாதாரணமாக இயக்குவதற்கு நிறுவல் ஸ்கிரிப்ட் தானாகவே பல படிகளைக் கையாளுகிறது.
முதலில், `apt-get` மூலம் LLVM 14 தொடர்பான தேவையான தொகுப்புகளை நிறுவவும்.
`/usr/lib/libllvm-14.so` க்கு ஒரு குறியீட்டு இணைப்பை உருவாக்கவும், இதன் மூலம் எதிர்கால அமைப்புகள் LLVM ஐ நம்பத்தகுந்த வகையில் குறிப்பிட முடியும்.

Wave `LLVM_SYS_140_PREFIX` சூழல் மாறியை அமைக்கிறது, இதனால் கம்பைலர் LLVM ஐ சரியாகக் கண்டறிய முடியும்.
இந்த அமைப்பு `~/.bashrc` இல் சேர்க்கப்பட்டு எதிர்கால முனைய அமர்வுகள் முழுவதும் தொடரும்.

அடுத்து, Wave தொகுப்பின் (`.tar.gz`) பயனர் குறிப்பிட்ட பதிப்பைப் பதிவிறக்கி அன்சிப் செய்யவும்.
அன்ஜிப் செய்த பிறகு, `wavec` இயங்கக்கூடிய கோப்பை `/usr/local/bin` இல் நிறுவவும்.
`wavec` கட்டளையை கணினியில் எங்கும் கிடைக்கும்படி கட்டமைக்கவும்.

நிறுவல் முடிந்ததும், `wavec --version` கட்டளையைப் பயன்படுத்தி அது சரியாக நிறுவப்பட்டுள்ளதா என்பதைச் சரிபார்க்கவும்.

## நிறுவலைச் சரிபார்க்கவும்

நிறுவல் முடிந்ததும், கீழே உள்ள கட்டளையை இயக்குவதன் மூலம் Wave கம்பைலர் சரியாக நிறுவப்பட்டுள்ளதா என்பதை நீங்கள் சரிபார்க்கலாம்.

```bash
wavec --version
```

கட்டளையை இயக்கும் போது நிறுவப்பட்ட Wave இன் பதிப்புத் தகவல் காட்டப்பட்டால், அது சாதாரணமாக நிறுவப்படும்.

---

## Wave அகற்றும் வழிகாட்டி (`uninstall.sh`)

உங்கள் கணினியிலிருந்து Wave ஐ அகற்ற விரும்பினால், நீங்கள் வழங்கிய அகற்றும் ஸ்கிரிப்டைப் பயன்படுத்தலாம்.
நிறுவலின் போது சேர்க்கப்பட்ட கோப்புகள் மற்றும் அமைப்புகளை சுத்தம் செய்வதற்கு இந்த ஸ்கிரிப்ட் பொறுப்பாகும்.

### எப்படி நீக்குவது

பின்வரும் கட்டளையை முனையத்தில் இயக்கவும்:

```bash
curl -fsSL https://wave-lang.dev/uninstall.sh | bash
```

அகற்றுதல் முடிந்ததும், wavec கட்டளை இனி பயன்படுத்தப்படாது.
Wave உடன் தொடர்புடைய இயங்கக்கூடிய கோப்புகள் மற்றும் அமைப்புகள் உங்கள் கணினியிலிருந்து நீக்கப்படும்.
