# நிறுவல்

## நிறுவும் முறை

வேவ் வழங்கப்படும் நிறுவல் ஸ்கிரிப்ட் மூலம் எளிமையாக நிறுவ முடியும்.
டெர்மினல் இல் கீழ்க்கண்ட கட்டளையை இயக்கினால் குறிப்பிடப்பட்டுள்ள வேவ் தொகுப்பாளரின் (`wavec`) பதிப்பு தானாக நிறுவப்படும்.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version <version>
```

நிறுவல் ஸ்கிரிப்ட் அமைப்பு சூழலை சரிபார்க்கும் வாக்கில், வேவ் இயக்கத்திற்குத் தேவையான சார்பு மற்றும் தொகுப்பாளர் தானாக அமைந்துவிடும்.
பதிப்பை குறிப்பிடில் இல்லை என்றால், தற்போதைய நிலையான பதிப்பு அல்லது குறிப்பிடப்பட்ட அளவுகோலின் அடிப்படையில் அமைப்பட்ட தொடக்கப்பதிப்பு நிறுவப்படும்.

## நிறுவல் எடுத்துக் காண்பிக்கும்

தற்போதைய பதிப்பைக் கட்டமைக்க கீழ்க்காணும் போல இயக்கவும்.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- latest
```

குறிப்பிட்ட பதிப்பை நிறுவ விரும்பினால், `--version` விருப்பத்தை பயன்படுத்தவும்.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version v0.1.3-pre-beta
```

நைட்லி கட்டமைப்பில் போன்ற மேலும் விவரமான பதிப்பை குறிப்பிடுதல் கூட சாத்தியமானது.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version v0.1.3-pre-beta-nightly-2025-07-11
```

## நிறுவலை செயல்படும் போது செய்யப்படும் பணிகள்

நிறுவுதல் ஸ்கிரிப்ட் வேவ் சாதாரணமாக செயல்படும் பல படிகளை தானாகச் செய்யும்.
முதலில் LLVM 14-க்கு தேவையான கட்டளைகளைக் `apt-get` மூலம் நிறுவுக.
அத்தியாயம் LLVM மிகச் சமன்படுத்தப்பட்ட முறையில் குறிப்பிடுவதற்காக `/usr/lib/libllvm-14.so` கடையில் சின்ன தொடர்பைப் உருவாக்குங்கள்.

DSLllvm-ம் சீரான அறிவுறுத்தல் எழுத்தோடு சங்க்வித்திய வேண்டும். இந்த அமைப்பு `~/.bashrc`வில் அனுப்பப்பட்ட பின்னர், அந்த டெர்மினல் வழக்கத்தில் இருந்தாலும் கொண்டிருக்கின்றது.

தற்போதைய கட்டமைப்பு சீட்டையில் குறிப்பிடப்பட்டுள்ள வேவ் பெட்டகத்தை (`.tar.gz`) பதிவிறக்கம் செய்வதும் அவர்கள்தான் அந்த உருப்படிக்கு திரும்பினால். பின்னர் தரவை வடிகட்டி எடுத்தல்.
திரும்புதல் முடிந்ததும் `wavec` இயக்கக் கோப்பினைப் `/usr/local/bin` உடல் கட்டமைத்து, அவை அங்கிருந்து `wavec` கட்டளையை பயன்படுத்தும்பொருட்டு அமைக்கின்றது.

நிறுவலை முடிந்து `wavec --version` எனும் கட்டளையை அனுபவித்து இயங்கினால் உறுதி செய்யும்.

## நிறுவலை உறுதி செய்க

நிறுவல் முடிந்த பிறகு, கீழேயுள்ள கட்டளைகளை இயக்கி வேவ் தொகுப்பாளர் சரியாக நிறுவப்பட்டதைப் பரிசோதிக்கலாம்.

```bash
wavec --version
```

கட்டளைக்கு பின்னர் வெளியீடு செய்யப்பட்ட பதிப்பு தகவல் வெளியிடப்படுகிறது என்றால், அது சாதாரணமாக நிலை படுத்துவதனால் வருகிறது.

## 웨이브 அகற்றல் வழிகாட்டி (`uninstall.sh`)

முறைமையிலிருந்து ਵੇਵ மேலும் நீக்க விரும்பினால், வழங்கிய நீக்கத்தின் ஸ்கிரிப்ட்டுகளைப் பயன்படுத்தலாம்.
இந்த ஸ்கிரிப்ட் நிறுவும் சுழற்சியில் சேர்க்கப்பட்ட கோப்புகளும் அமைப்புகளையும் வரிசைக்கட்டும்.

### அகற்றும் முறை

டெர்மினலில் கீழ்க்கண்ட உத்தரவை இயக்கு.

```bash
curl -fsSL https://wave-lang.dev/uninstall.sh | bash
```

நீக்கம் முடிக்கப்பட்ட போது wavec கட்டளையை விலக்க முடியும்,
சோம்பல் தொடர்புடைய செயல்பாடு மற்றும் காத்தல் அமைபுத்தகவல்களும் நீக்கப்பட்டு முடிவுபெற்றது.
