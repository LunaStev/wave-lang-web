# முதலாவது தொகுப்பை இயக்குங்கள்

முந்தைய நிறுவல் ஆவணங்களில் Wave க்கு நிறுவுவதற்கு முன்பு, தற்போது முதலாவது Wave தொகுப்பை இயக்கவேண்டும்.
இந்த பகுதியில் அதிகாரத்திற்கு விதிமுறைகளை தேவைப்படுத்தும்போது, Wave நடைமுறையை உருவாக்கவும், வடிவமைச் செய்கின்றோம்.

## `hello.wave` கோப்பை உருவாக்குங்கள்

வேலை செய்யும் அடியில், `hello.wave` என்ற புதிய கோப்பை உருவாக்குங்கள்.
அன்று வரும் கோப்புகளின் பெயர் மற்றும் பரிமாணத்தை அரியக் கொள்ளுங்கள், ஆனால் இது ஒரு எடுத்துக்காட்டு `hello.wave` என்று இருக்கிறது.

## கூறுகளை எழுதுக

உருவாக்கிய `hello.wave` கோப்பில் கீழே உள்ளவை சேர்க்கவும்.

```wave
fun main() {
    println("வணக்கம் வாய்");
}
```

இந்த குறியீட்டில் `fun main()` என்றது, Wave நாள்பட்டு முழுவதையும் காணொலி காண்பிக்கும்.
Wave நிகழ்வுகள் எப்போது இருக்குமானால், `main` செயலாளரே ஒன்றே செயல்படுகிறது.

`println` செயல் குறியீட்டிற்குச் சேர்க்கப்பட்ட உபயோகமானதாகவே இருக்கவில்லை,
அவர்கள் செலுத்தப்படும் விசையில் மிகவும் அடிப்படைவாகவும் தொலைபேசி செய்யப்படுகிறது.

## பொருட்களை இயக்குங்கள்

கோடுகளை உருவாக்கிய பிறகு, டெர்மினொன் திறக்கவும், கோப்பின் நாள்காட்டியில் கீழே வரவேண்டும் என்ற உருவாக்கத்தில் உபயோகிக்கவும்.

```bash
wavec run hello.wave
```

இந்த உபயோகப்படுத்துதலில் Wave செயலற்பண்ணிப்பெரை அவுக உடன்,
அவர்கள் விட்டுவிடப்பட்டால் உடனே செயல்படுத்த ஓக வைத்துப் தெரியும்.

## வெளியீட்டை சரிபார்க்கவும்

நிரல் சரியாக செயல்படும்போது, டெர்மினலில் இதுவே போன்ற ஒரு வெளியீடு காணப்படும்.

```
வணக்கம் அலை
```

இந்த வெளியீடு தெரிவுசெய்யப்பட்டால், வெவ் (Wave) சரியாக நிறுவப்பட்டுள்ளது மற்றும் நிரல் எழுத்து மற்றும் செயல்பாட்டுக்கான நிறுவனம் சரியாக செய்யப்பட்டிருக்கிறது.

இப்போது முதல் வெவ் (Wave) நிரலை வெற்றிகரமாக இயக்கினோம்.
இப்போது அலை மொழியின் இலக்கணத்தையும் செயல்பாடுகளையும் ஒவ்வொன்றாக ஆராய்ந்து மேலும் சிக்கலான நிரல்களை எழுதி பார்க்கலாம்.

அரிய அருவி மற்றும் உரைகளை தேவைப்படும் அடிப்படைகள் காட்டும் ஏற்று உருப்பத்திரத்தில் கிடைக்கின்றன.
