# ஹலோ Wave ஐ அச்சிடவும்

Wave மொழியில் மிக அடிப்படையான வெளியீட்டு நிரலை எவ்வாறு எழுதுவது என்பதை இந்த ஆவணம் விளக்குகிறது.

---

## எடுத்துக்காட்டு குறியீடு

```wave
fun main() {
    println("Hello Wave");
}
```

---

## விளக்கம்
- `fun main()`

    Wave இது நிரலின் நுழைவு புள்ளி செயல்பாடு ஆகும். ஓடும்போது இது முதலில் அழைக்கப்படுகிறது.

- `println()`

    ஒரு தற்காலிக வெளியீட்டு செயல்பாட்டுடன் ஒரு சரத்தை வெளியிடுகிறது, மேலும் வெளியீட்டிற்குப் பிறகு ஒரு வரி இடைவெளியை (`\n`) சேர்க்கிறது.

- `;` (அரைப்புள்ளி)

    Wave இல் உள்ள அனைத்து அறிக்கைகளும் அரைப்புள்ளியுடன் முடிக்கப்படுகின்றன.

---

## மரணதண்டனை முடிவு

```text
Hello Wave
```

---

## கூடுதல் உதாரணங்கள்
Wave சரம் இடைக்கணிப்பை ஆதரிக்கிறது.

```wave
fun main() {
    var name: str = "Wave";
    println("Hello, {}!", name);
}
```

வெளியீடு:

```text
Hello, Wave!
```

---

> இந்த எடுத்துக்காட்டு Wave நிலையான நூலகத்தின் அடிப்படை வெளியீட்டு செயல்பாடுகளின் பயன்பாட்டை நிரூபிக்கிறது.
