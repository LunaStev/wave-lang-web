# தகரம்

Wave இரண்டு சிறுகுறிப்புகளை ஆதரிக்கிறது:

- ஒரு வரி கருத்து: `//`
- கருத்தைத் தடு: `/* ... */`

## ஒரு வரி கருத்து

`//` க்குப் பிறகு எதுவும் வரியின் இறுதி வரை புறக்கணிக்கப்படும்.

```wave
var x: i32 = 10; // line comment
x += 5;          // still works
```

## தொகுதி சிறுகுறிப்பு

`/*` மற்றும் `*/` இடையே உள்ள உள்ளடக்கங்களை புறக்கணிக்கவும்.

```wave
var y: i32 = 1 /* inline block */ + 2;
```

பிளாக் கருத்துகள் பல வரிகள் மற்றும் கூடுகளை ஆதரிக்கின்றன.

```wave
/* outer
   /* inner */
   outer end
*/
```

## சரங்கள் மற்றும் கருத்து குறிகள்

சரத்திற்குள் உள்ள `/*`, `*/`, `//` ஆகியவை கருத்து தொடக்கம்/முடிவு என கருதப்படுவதில்லை.

```wave
var marker: str = "/*//*/";
```

## சிறுகுறிப்பு பிழை

பிளாக் கருத்து மூடப்படவில்லை என்றால், தொகுத்தல் பிழை (`E1002`) ஏற்படும்.

```wave
/* not closed
```

கம்பைலர் தொடக்க இடம், காரணம் மற்றும் திருத்தம் குறிப்புகளை வெளியிடுகிறது.
