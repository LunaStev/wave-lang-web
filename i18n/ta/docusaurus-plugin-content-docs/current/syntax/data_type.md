# தரவியல் வகை

இந்த ஆவணம் வெவ் (Wave) நிரல் மொழியின் மூலம் வழங்கப்படும் பல்வேறு தரவுக் கோடிகளை விளக்குகிறது.
Wave பல்வேறு தரவுக் கோடிகளை மூலம் மதிப்புகளை சேமித்து செயலாக்க முடியும், ஒவ்வொரு தரவுக் கோடிக்கும் அதன் தரவுகளைப் பிரதிபலிக்கும் முறை மற்றும் நினைவக செயலாக்க முறையை தெளிவாக வரையறுக்கிறது.

தரவுக் கோடிகளை தெளிவாக வரையறுக்குவது வெவ் (Wave) க்கான முக்கிய வடிவியல் பார்வைகளில் ஒன்றாகும்.
Wave \*\*முழுமையான அமை என்ற \*\* கண்பதிப்பு முறை. எல்லாவற்றே `var`/`fun` உருப்பதிகளை மற்றும் மாறிலிகள் போதிய அமைவு தேவைப்படுகிறது, அளவீடுகள் மூலமாக மதிப்பீடுகளைக் கொண்டுவர முடியாது. இதனால், `var x = 1;` போன்று, ஆபட்டை ஏற்படுத்த முடியாது.
இதன் மூலம் குறியீட்டின் நோக்கத்தை தெளிவாக வெளிப்படுத்துவதற்கு, தொகுப்பிற்கான நேரத்தில் பிழைகளை முன்கூட்டியே கண்டறியவும், சிறந்த நினைவக உபயோகத்தை மற்றும் நிலையான செயல்பாட்டை உறுதி செய்யவும் முடியும்.

---

## முழுமை வகை

முழுமை வகை அனைத்துப் பலன்களைச் சேமிக்கப் பயன்படுத்தப்படுகிறது.
வைவில் அடிப்படையாக `i32` (சினம் உடைய 32 பிட்டுப் பூரணங்கள்) மற்றும் `u32` (சினம் இன்றி 32 பிட்டுப் பூரணங்கள்) அடிக்கிறார்கள், ஆனால் தேவைஎனில் வாரி பலன்களின் பிட்டுப் அளவை மிகவும் சென்னையாக வரையறுக்கலாம்.

சினம் உடைய முழுமை வகை `i8` துவக்கம் `i1024` வரை வழங்கப்படுகிறது, சினம் இல்லாத முழுமை வகை `u8` துவக்கம் `u1024` வரை பெறலாம்.
இது மூலம் எளிய கணக்கிடுதல் முதல் பெரிய அளவிலான முழுமையாக கலல் ஆராய்ச்சி, குறியீட்டு செயலாக்கம், குறைந்த அளவு அமைப்புக் குறியீடுகளை தேவைப்படும் விருப்பங்களை பூர்த்தி செய்ய முடியும்.

கீழ்க்காணும் முழுமை வகையைப் பயன்படுத்தும் எளிதான உதாரணம்.

```wave
var a: i32 = 100;
var b: u32 = 200;
```

## ஊழல்வளைவாடு வகை

ஊழல்வளைவாடு வகை உண்மையான எண்ணைப் பூட்டுவதற்காக பயன்படுத்தப்படுகிறது.
Wave அடிப்படையாகக் கணக்கீட்டுச் செயல்களில் `f32` என்ற ஊழல்வளைவாடு வகையை பயன்படுத்துகிறது, மேலும் மேலதிகக் கண்காணிப்பு தேவைப்பட்டால் பெரிய அளவின் வகையைத் தேர்ந்தெடுக்கலாம்.

Wave `f32` முதல் `f128` வரை பரவிய ஊழல்வளைவாடு வகைகள் வழங்குகிறது, உடனே நாடுகாணல் தன்மை மற்றும் செயல்திறனைச் தேர்வு செய்வதற்கு பயனருக்கு தீர்வு உள்ளது.
இதன்மூலம் பொதுவான எண் கணக்கீடுகள் முதல் அரிதான விஞ்ஞான கணக்கீடுகள் வரை பல்வேறு விதமான எண்ணியல் செயல்களைச் சேவையாக வழங்க முடியும்.

கீழே பசுமாப்னன் களத்தால் செயல்தகவுபின்பட்ட உதாரணம்.

```wave
var pi: f32 = 3.14;
var e: f64 = 2.71828;
```

## அருகு வகை

அருகு வகை இடைத்தாயிலை படிக்க சக்தியாக உள்ளது.
Wave அடியூட்டு `str` என்று கூறுப்புனர்களைப் பயன்படுத்தி அழகிடும் அச்சுக்களை உருவாக்குகின்றன, அனுப்பிய சில உரைகள் பெரிய விலைபாராட்டில் இடமளிக்கின்றன.

அழைப்புகள் அதற்கொன்றிய செயற்றுறை, உட்பட, உள்குறிகாட்டிகள், உரை அடித்தமைகளை வாசிக்க, அளவீடுகளை வென்றுள்ள எண்ணான உள்புற ஷியர்கள்.

கீழ்க்காணும் இணிமையின் அடிப்படையான பயன்முறை.

```wave
var text: str = "வணக்கம் அலை";
```

## பொய்யி உண்மை வகை

பொய்யி உண்மை வகை உண்மையான அல்லது பொய்யான மதிப்புகளை வெளிப்படுத்தும் தரவுக் கோடியே ஆகிறது.
Wave `bool` வகையைப் பயன்படுத்துகிறது, மதிப்பு `true` அல்லது `false` செுரிமை உள்ளது.

பொய்யி உண்மை வகை காரியங்கள் மற்றும் உருப்பதிகளில் முக்கிய இடத்தில் செயல்படுகிறது, நிரலின் பீடுகளை கட்டுப்படுத்துகிறது.

```wave
var isActive: bool = true;
var isAvailable: bool = true;
```

## எழுத்து வகை

எழுத்து வகை ஒரே ஒரு எழுத்து சேமிக்க பயன்படுத்தப்படுகிறது.
`char` நெடியில் உள்ளது, உட்புற அணிவைகள் இடம்பெறும்கூறுகள் அல்லது பிற பொருட்களை மாற்றாது.

எழுத்து உருதிகள் மலர்களில் முதல் வேலை தனிமைகிரகங்கள் உள்ளது.

```wave
var letter: char = 'A';
```

## பைட் வகை

பைட் வகை 1 பைட்டுப் பழுக்குகளை உள்ளே போட்ட கையாள்கிறது.
இந்த வகை நியுயார்க் கையாள்தல்களை வகுப்பை பொறுப்பேற்கும் வகைக்கு, பின்னவுள்ள நெ காரண்முரட்டு மற்றும் பைலான செலவு தகவல்களைப் பொதுவாக வழங்குகின்றனர்.

Wave `byte` பற்றி பைட் வகையை உருவாக்குகிறது.

```wave
var byteData: byte = 0xFF;
```

## 포인터 타입

포인터 타입은 메모리 주소를 직접 참조하는 데 사용됩니다.
Wave에서는 `ptr<T>` 형태로 포인터 타입을 선언합니다.

포인터는 저수준 메모리 접근이 필요한 경우에 사용되며, 시스템 프로그래밍이나 성능이 중요한 코드에서 주로 활용됩니다.

```wave
var ptr: ptr<T> = &someVariable;
```

## `null` 리터럴

Wave에서 `null`은 정식 리터럴입니다.

- `null`은 식별자가 아닙니다. (`var null = ...` 형태 불가)
- `null`은 오직 `ptr<T>` 타입에만 대입할 수 있습니다.

```wave
var p: ptr<i32> = null;  // OK

// var n: i32 = null;    // ERROR
// var b: bool = null;   // ERROR
```

## 배열 타입

배열 타입은 동일한 데이터 타입의 여러 값을 순차적으로 저장하는 데 사용됩니다.
Wave에서는 `array<타입, 크기>` 형태로 배열을 선언하며, 배열의 크기를 컴파일 타임에 명확히 지정합니다.

이를 통해 메모리 구조가 명확해지고, 안정적인 접근이 가능합니다.

```wave
var numbers: array<i32, 5> = [1, 2, 3, 4, 5];
```

각 데이터 타입은 용도와 특성에 맞는 범위와 크기를 선택할 수 있도록 설계되어 있습니다.
적절한 데이터 타입을 선택하면 메모리를 효율적으로 관리할 수 있으며, 코드의 안정성과 가독성 또한 크게 향상됩니다.
