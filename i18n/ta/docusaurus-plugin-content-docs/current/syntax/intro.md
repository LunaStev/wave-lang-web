# வகை மற்றும் அளவு

## அறிமுகம்

Wave நிரல் மொழியின் அடிப்படை வடிவமைப்புக் கொள்கைத் தரத்தே வழங்கும் திறனைத் தரவுத் தலையாகவும் தர்க்கர்களிடையே சமமையாகவும் பூராவு சேமிக்கவும், பயனுள்ளவும் மற்றும் நெகிழ்வதற்கான மென்பொருள் வளர்ச்சி சூழல் வழங்குகிறது.
இந்த பகுதி Wave வை அமைப்பதற்கான அடிப்படை பகுதிகளை உள்ளடக்கிய செயல்வழிகளையும் மாறிகளையும் போதிக்கும்.

செயல்பாடுகள் நிரலின் செயல்முறைகளை உருவாக்குவதற்கான அடிப்படை அலகுக்களாக செயல்படும், மாறிகள் அந்த நகருக்கே தேவையான தரவுகளைச் சேமிக்கவும் கையாள்வதற்கான வேலை செய்வதாக இருக்கின்றன.
செயல்பாடுகள் மற்றும் மாறிகளை பரிசீலித்தலும் பயன்படுத்துவதற்கான வழிமுறைகளை உறுதியாக புரிந்தால், Wave என்னும் மொழியின் கட்டமைப்பு மற்றும் வடிவமைப்பு நோக்கங்களை மேலும் அழுத்தமாகப் பயன்படுத்தலாம்.

## செயல்பாடு

Wave இல் செயல்பாடு ஒரு தனித்தனி முறையில் இயக்கக்கூடிய மறுசுழற்சி செய்யக்கூடிய குறியீட்டு தொகுப்பாகும்.
எதிர்வழியின் செயல்கள் அல்லது கணக்குகளை ஒரே மொத்தமாகக் காண்பித்துப்பார்க்கலாம், நிரலின் நீங்கத்தும் வேண்டுமென்றால் அழைக்கப்படும்.

செயல்பாட்டைப் பயன்படுத்துவதால் மீண்டும் மீண்டும் உள்ள குறியீட்டை குறைக்கலாம், நிரலைக் கழிக்கவும், கேள்விக்கொள்கை மற்றும் பராமரிப்பு மேம்படுத்தலாம்.
மேலும் கணக்கீட்டுக் கையாள்வதற்கான வேலைகளையும், தகவல்களைப் பரிசுடாஐ பார்க்கும் வழிகளையும், மூலக் கோட்டிலிருந்து விவரங்களைப் பிரிக்கும் பல்வேறு நோக்கத்தில் பயன்படும்.

Wave இல் செயல்பாடு எனப்படும் `fun` பொருள்களால் வரையறுக்கப்படும், செயல்பாடு பெயர் மற்றும் உபயோகமான பதில்கள் மற்றும் குறியீட்டு உபாதிகள் `{}` மதில்களால் மூடியுள்ள செயல்தீர் ஆகிறது.

### வகை வரையறுத்தல்

Wave இல் மிகவும் அடிப்படையான செயல்பாடு வரையறுக்கப்படும் வடிவம் இதுவரை உள்ளது.

```wave
fun main() {
    // தரவிலிருந்து குறியீடு உற்பத்தியிலிருந்து மேலிடு
}
```

`main` செயல்பாடு நிரலின் முன்னிலை என்றாலும், Wave நிரல்கள் ஒன்றில் குறைந்தது ஒரு `main` செயல்பாட்டை அடிப்பட்டிருக்க வேண்டும்.
நிரல் இந்த செயல்பாட்டிலிருந்து செயல்பாட்டைப் தொடங்குகிறது.

செயல்பாடுகள் தேவையின்படி பயனர்கள் இருப்பதாகக் கொண்டெல்லாம் பயன்படத்தே அல்லது கணக்கீட்டு விளைவுகளை வழங்குவது மட்டுமல்ல, அழைக்கும் இடத்தை நோக்கி வெளியிடலாம்.
வெளியீட்டு மக்கள் தருவதற்கான வரையறைகள் இருக்கின்றன, செயல்பாட்டின் வெளியாணை வழங்கும்போது அதை அறிவுறுத்துவது.

### உதாரணம்: எளிய செயல்பாடு

இந்த எடுத்துக்காட்டு இரண்டு முழு எண்களை எடுத்துக் கொண்டு, அவற்றின் கூட்டத்தை வெளியிடும் சின்ன செயலி.

```wave
fun தேதி(ஒரு :i32, b :i32) -> i32 {
    return a + b;
}

fun முதன்மை() {
    var முடிவு = தேதி(5, 7);     // add செயல்பாட்டைப் பதிவகற்ற
    println(முடிவு);           // அச்சிடுக: 12
}
```

இந்த எடுத்துக்காட்டில் `add` செயலி `a` மற்றும் `b` என்ற இரண்டு முழு எண் அளவீடுகளைப் பெற்று, அவற்றை சேர்த்து, முடிவைக் கட்டுப்படுத்துகிறது.
`main` செயலியில், `add` செயலியை அழைத்து, பெற்ற மதிப்பை ஒரு மாறியில் சேமித்து அச்சிடுகிறோம்.

இப்படி செயலிகள் குறிப்பிட்டச் செயல்களை மூடிய அறிவிப்பு அளிக்கச் செய்கின்றன, அதனால், இந்த செயலிகள் நிகழ்நிலைகளில் மீண்டும் பயன்பட உதவுகின்றன.

## மாறி

மாறிகள் நிரலில் தரவுகளை சேமிக்கவும் கையாள உதவுவதற்காகப் பயன்படுத்தப்படுகின்றன.
Wave மாறிகளை அறிவிப்பில் ஒரு மாறி மற்றும் மாறிலி மாறிகளை நன்கு வேற்றுமைப்படுத்துவதற்காக வடிவமைக்கப்பட்டுள்ளது, பகுப்பாய்வுக் கொள்ளும் குறியீட்டின் உதவியால் தரவுக்கான மாற்றத்திற்கான நோக்கங்களை வெளிப்படுத்துவதற்கு.

இதனால், நிகழ்வு மாற்றங்களுக்கு மேலான துல்லியமான ஒத்திகை கிடைக்கின்றது, திட்டமிடாத மதிப்பு மாற்றத்திற்கான பிழைகள் குறைக்கப்படுகின்றன.

### மாறும் மாறி

Wave இல், மாறிகள் அடிப்படையாக முறையாக மாற்றக்கூடியவை.
அதாவது, ஒரு முறை அறிவிக்கப்படும் பிறகு நிகழ்ச்சி மேற்கொண்டு சுட்டிய மதிப்புகளை மாற்ற முடியும்.

மாறி மாறிகளை var விசையைப் பயன்படுத்தி அறிவிக்கிறோம்.

```wave
var x :i32 = 10;
x = 20;
```

மேலே உள்ள குறியீட்டில் `x` அளவை துவக்க மதிப்பில் `10` என்ற மதிப்பை இருக்கும், பிறகு அதை `20` என்ற மதிப்பாக மாற்ற முடியும்.
இப்படி பயன்பட வேண்டிய தரவுகளுக்கான மாறிகளை மாற்றுபடுத்துகிறோம்.

### மாறாத மாறி

மாறியை மாற்றம் (immutable) என்ற அளவிற்காக அறிவித்தால், ஆரம்ப மாறிக்குப் பிறகு அந்த மதிப்பை மாற்ற முடியாது.
பார்வை மாறிகள் let விசையைப் பயன்படுத்தி அறிவிக்கப்படுகின்றன.

```wave
let y :i32 = 5;
// y = 10;   // பிழை: மாற்ற முடியாத மாறிகள் மதிப்புகளை மாற்ற முடியாது.
```

இணை மாறிகள் மதிப்புகள் மாற்றப்படாது என்பதற்கான உறுதியைக் கொண்டுள்ளதாக இருக்கிறது, இதனால் திட்டத்தின் நிலைத்தன்மை மற்றும் முன்னேற்றம் அதிகரிக்கிறது.
மதிப்பில் மாற்றங்கள் தேவைப்படாத குறியீட்டு தரவுகளுக்கான மாறிகளைப் பயன்படுத்த கனரகம் பரிந்துரைக்கப்படுகிறது.

Wave இல் `let` விசையுடன் `mut` ஐப் பயன்படுத்தி, வெளிப்படையாக மாற்றுகூடியதைக் கிடைக்க செய்கின்றது.

```wave
let mut y :i32 = 5;
y = 10;
```

ஆம் என்றால், மாறி `let` என்பதற்காக அறிவிக்கப்பட்டு, ஆனால் `mut` விசையானாலும் மதிப்புகளை மாற்ற முடியும்.

### மாறி குறித்த எடுத்துக்காட்டுகள்

இதுவே மாறி மற்றும் மாற்றுக்கு உருப்படிகளை அறிவிப்பதற்கான பல வடிவங்கள்.

```wave
var x :i32 = 10;
let y :f64 = 3.14159;
var name :str = "Wave";
let is_active :bool = true;
```

இந்த எடுத்துக்காட்டில், `x` மற்றும் `name` என்பது மாறி மாறுகள், `y` மற்றும் `is_active` என்பது மாறாதவையாகும்.
Wave பரிமாணங்கள் `var` மற்றும் `let` என்பவற்றை தீவிரமாக பற்றியிருப்பதே இக்கோட் அளவுகோலுக்குப் பெற்று தரவுகளை மாற்றுகூடியவற்றைக் செலுத்துகின்றது.

உறுதியாக மாறி மற்றும் மாறாத அளவுகளை வெளிப்படுத்துவதால் தரவுகளினால் ஆரோக்கியம் காப்பாற்றப் பெற்று நிலைத்துக் கொள்ளலாம்.
