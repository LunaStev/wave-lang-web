# இலக்கணம்

இந்த ஆவணம் Wave நிரலாக்க மொழியின் ஒட்டுமொத்த இலக்கணத்தை விவரிக்கிறது.
Wave தற்போது உருவாக்கத்தில் உள்ள மொழி என்பதால், சில தொடரியல் மற்றும் அம்சங்கள் இன்னும் முழுமையாக செயல்படுத்தப்படாமல் இருக்கலாம் அல்லது எதிர்காலத்தில் மாறலாம்.
ஆயினும்கூட, Wave தற்போது இலக்காகக் கொண்டிருக்கும் இலக்கண அமைப்பு மற்றும் அடிப்படைக் கருத்துகளைப் புரிந்துகொள்ளும் நோக்கத்துடன் இந்த ஆவணம் எழுதப்பட்டது.

இந்த ஆவணம் Wave இன் தற்போதைய நிலையைப் பற்றிய யோசனையை வாசகர்களுக்கு மட்டும் வழங்காது;
மொழி எவ்வாறு வடிவமைக்கப்பட்டு விரிவடைகிறது என்பதைப் பற்றிய ஒட்டுமொத்தப் படத்தைப் பெறலாம்.

Wave குறைந்த-நிலை கட்டுப்பாடு மற்றும் உயர்-நிலை சுருக்கத்தை இணைக்கும் மொழியாக வடிவமைக்கப்பட்டுள்ளது.
சிஸ்டம் புரோகிராமிங் போன்ற குறைந்த அளவிலான பகுதிகளிலிருந்து வலை மேம்பாடு, செயற்கை நுண்ணறிவு மற்றும் பிளாக்செயின் போன்ற உயர்-நிலை பயன்பாட்டுப் பகுதிகள் வரை பரந்த அளவிலான பயன்பாட்டை செயல்படுத்துவதே வடிவமைப்பு குறிக்கோள்.
இந்த நோக்கத்திற்காக, Wave செயல்திறன் மற்றும் கட்டுப்பாட்டை வலியுறுத்துகிறது, அதே நேரத்தில் உற்பத்தித்திறனை சமரசம் செய்யாத இலக்கணம் மற்றும் கட்டமைப்பைப் பின்பற்றுகிறது.

Wave ஒரு சக்திவாய்ந்த நிலையான நூலகம் மற்றும் ஒருங்கிணைந்த உருவாக்கம் மற்றும் கருவி கட்டமைப்பை மனதில் கொண்டு வடிவமைக்கப்பட்டுள்ளது.
சிக்கலான அமைப்பில் நேரத்தை செலவிடுவதை விட, முக்கிய தர்க்கத்தை செயல்படுத்துவதில் டெவலப்பர்கள் கவனம் செலுத்துவதை இது நோக்கமாகக் கொண்டுள்ளது.

Wave இன் தொடரியல் C அல்லது Rust க்கு சில ஒற்றுமைகளைக் கொண்டுள்ளது, ஆனால் இது எளிய சாயல்களை நோக்கமாகக் கொண்டிருக்கவில்லை.
இது பரிச்சயமான இலக்கணக் கூறுகளை அடிப்படையாகக் கொண்டது, ஆனால் மொழியை விரைவாகக் கற்றுக்கொள்வதற்கும் உண்மையான வளர்ச்சியில் திறமையாகப் பயன்படுத்துவதற்கும் உதவும் வகையில் மறுசீரமைக்கப்பட்டுள்ளது.
ஒரு வெளிப்படையான வகை அமைப்பு, யூகிக்கக்கூடிய நடத்தை மற்றும் குறியீட்டின் நோக்கத்தை வெளிப்படுத்தும் தொடரியல் ஆகியவை முக்கியமான வடிவமைப்பு அளவுகோல்கள்.

இந்த ஆவணம் Wave இன் அடிப்படை தொடரியல் மற்றும் முக்கிய மொழி அம்சங்களை நடைமுறை எடுத்துக்காட்டுகளுடன் அறிமுகப்படுத்துகிறது.
கட்டுப்பாட்டு அறிக்கைகள் பிரிவில் `if`, `while`/`for` உடன் `match` அறிக்கைகள் உள்ளன.
ஒவ்வொரு பகுதியும் தனிப்பட்ட இலக்கண கூறுகளை விளக்குகிறது, மேலும் Wave மொழியின் கட்டமைப்பு மற்றும் வடிவமைப்பு தத்துவத்தை நீங்கள் புரிந்து கொள்ள உதவும் வகையில் ஒழுங்கமைக்கப்பட்டுள்ளது.
