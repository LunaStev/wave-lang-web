# இயக்குபவர்

இந்த ஆவணம் தற்போதைய கம்பைலரின் அடிப்படையில் கிடைக்கக்கூடிய ஆபரேட்டர்களை ஒழுங்கமைக்கிறது.

## எண்கணிதம்

| ஆபரேட்டர் | விளக்கம் |
| --- | --- |
| `+` | கூடுதலாக |
| `-` | கழித்தல் |
| `*` | பெருக்கல் |
| `/` | பிரிவு |
| `%` | ஓய்வு |

## ஒப்பிடு

| ஆபரேட்டர் | விளக்கம் |
| --- | --- |
| `==` | சமம் |
| `!=` | வேறுபட்ட |
| `<` | சிறிய |
| `<=` | குறைவாக அல்லது சமமாக |
| `>` | பெரிய |
| `>=` | விட அதிகமாகவோ அல்லது சமமாகவோ |

## தர்க்கம்

| ஆபரேட்டர் | விளக்கம் |
| --- | --- |
| `&&` | தருக்க மற்றும் |
| `\|\|` | தருக்க அல்லது |
| `!` | தர்க்கம் இல்லை |

## பிட்

| ஆபரேட்டர் | விளக்கம் |
| --- | --- |
| `&` | பிட்வைஸ் மற்றும் |
| `\|` | பிட் அல்லது |
| `^` | பிட்வைஸ் XOR |
| `~` | பிட் இல்லை |
| `<<` | இடதுபுறம் மாற்றவும் |
| `>>` | வலது மாற்றம் |

## கல்லூரி சேர்க்கை

| ஆபரேட்டர் | விளக்கம் |
| --- | --- |
| `=` | அடிப்படை பணி |
| `+=` | சேர்த்த பிறகு பணி |
| `-=` | கழித்த பிறகு ஒதுக்கீடு |
| `*=` | பெருக்கல் பிறகு பணி |
| `/=` | பிரிவுக்குப் பிறகு மாற்று |
| `%=` | மீதமுள்ள பிறகு பணி |

## unary/pointer/cast

| ஆபரேட்டர்/திறவுச்சொல் | விளக்கம் |
| --- | --- |
| `++`, `--` | முன்/பின்னோக்கி அதிகரிப்பு/குறைவு |
| `&x` | முகவரி பெற |
| `deref p` | சுட்டி dereference |
| `expr as T` | வெளிப்படையான நடிகர்கள் |

## சுட்டி எண்கணிதம்

| வெளிப்பாடு | முடிவுகள் |
| --- | --- |
| `ptr<T> + int` | `ptr<T>` (GEP இயக்கம்) |
| `int + ptr<T>` | `ptr<T>` (GEP இயக்கம்) |
| `ptr<T> - int` | `ptr<T>` (GEP இயக்கம்) |
| `ptr<T> - ptr<T>` | `i64` (பைட் வித்தியாசம்) |
| `ptr == ptr`, `ptr != ptr` | சுட்டி ஒப்பீடு |

## ஒதுக்கப்பட்ட அல்லது செயல்படுத்தப்படாத பொருட்கள்

இலக்கண டோக்கன்கள் உள்ளன, ஆனால் சிலவற்றை தற்போது வெளிப்பாடு செயல்பாடுகள் ஆதரிக்கவில்லை.
எடுத்துக்காட்டு: `??`, `?:`, `in`, `is`, `!&`, `!|`, `~^`.
