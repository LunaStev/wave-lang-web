# የመጀመሪያ ፕሮግራምዎን በማሄድ ላይ

ከቀዳሚው የመጫኛ ጽሑፍ Wave ን ከጫኑ የመጀመሪያውን Wave ፕሮግራምዎን በቀጥታ ለማሄድ ጊዜው አሁን ነው።
ይህ ክፍል ቀላል ምሳሌን በመጠቀም የWave ፕሮግራምን ደረጃ በደረጃ በመጻፍ እና በማስኬድ ሂደት ውስጥ ያሳልፈዎታል።

## `hello.wave` ፋይል ይፍጠሩ

በመጀመሪያ፣ በእርስዎ የስራ ማውጫ ውስጥ `hello.wave` የሚባል አዲስ ፋይል ይፍጠሩ።
የፋይሉ ስም እና ቅጥያ በነጻ ሊገለጽ ይችላል ነገርግን `hello.wave`ን እንደ ምሳሌ እንጠቀማለን።

## ኮድ ጻፍ

ከዚህ በታች ያለውን ኮድ በፈጠረው `hello.wave` ፋይል ውስጥ ይፃፉ።

```wave
fun main() {
    println("Hello Wave");
}
```

በዚህ ኮድ `fun main()` የ Wave ፕሮግራም ማስፈጸሚያ ነጥብን ያመለክታል።
የWave ፕሮግራም ሁልጊዜ ከ`main` ተግባር ጀምሮ ይሰራል።

የ`println` ተግባር ሕብረቁምፊ ወደ መደበኛ ውፅዓት የሚያወጣ ተግባር ነው።
በስክሪኑ ላይ ጽሑፍ ሲታይ በብዛት ጥቅም ላይ ይውላል።

## ፕሮግራሙን አሂድ

ኮዱን መፃፍ ከጨረሱ በኋላ ተርሚናል ይክፈቱ እና ፋይሉ ካለበት ማውጫ የሚከተለውን ትዕዛዝ ያስኪዱ።

```bash
wavec run hello.wave
```

ይህ ትዕዛዝ Wave ማጠናከሪያ የምንጭ ፋይሉን ካጠናቀረ በኋላ ነው የሚሰራው።
ወዲያውኑ እንዲፈጽሙ ይመራዎታል።

## ውጤቱን ያረጋግጡ

ፕሮግራሙ በመደበኛነት የሚሰራ ከሆነ በተርሚናል ውስጥ የሚከተለውን ውፅዓት ያያሉ።

```
Hello Wave
```

ይህን ውፅዓት ካዩ፣ Wave በትክክል ተጭኗል።
ይህ ማለት ፕሮግራሙ በትክክል ተጽፎ ተፈፀመ ማለት ነው.

አሁን የመጀመሪያውን Wave ፕሮግራምዎን በተሳካ ሁኔታ አከናውነዋል።
ከአሁን ጀምሮ የ Wave አገባብ እና ተግባራትን አንድ በአንድ በመመልከት ተጨማሪ ውስብስብ ፕሮግራሞችን መፃፍ ይችላሉ።

ትክክለኛ የትዕዛዝ አማራጮች (`-O*`፣ `--debug-wave`፣ `--link`፣ `--dep-root`፣ `--dep`)
በ`wavec` CLI ሰነድ ውስጥ ማረጋገጥ ይችላሉ።
