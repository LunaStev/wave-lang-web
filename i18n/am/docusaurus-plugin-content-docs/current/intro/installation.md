# መጫን

## የመጫኛ ዘዴ

Wave በቀላሉ በተሰጠው የመጫኛ ስክሪፕት ሊጫን ይችላል።
ከዚህ በታች ያለውን ትዕዛዝ በተርሚናል ውስጥ ካስኬዱ የተገለጸው የ Wave ማጠናከሪያ (`wavec`) ስሪት በራስ-ሰር ይጫናል.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version <version>
```

የመጫኛ ስክሪፕቱ የስርዓት አካባቢን ይፈትሻል እና Wave ን ለማስኬድ የሚያስፈልጉትን ጥገኞች እና ማቀናበሪያ በራስ ሰር ያዘጋጃል።
አንድን ስሪት ካልገለጹ የቅርብ ጊዜው የተረጋጋ ስሪት ወይም በተጠቀሰው መስፈርት መሰረት ነባሪ ስሪት ይጫናል.

## የመጫኛ ምሳሌ

የቅርብ ጊዜውን ስሪት ለመጫን፣ ያሂዱ፡-

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- latest
```

የተወሰነ ስሪት መጫን ከፈለጉ የ `--version` አማራጭን ይጠቀሙ።

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version v0.1.3-pre-beta
```

እንደ የምሽት ግንባታዎች ያሉ ተጨማሪ ዝርዝር ስሪቶችን መግለጽም ይቻላል.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version v0.1.3-pre-beta-nightly-2025-07-11
```

## በመጫን ጊዜ ምን ይከሰታል

Wave በመደበኛነት እንዲሠራ ለማድረግ የመጫኛ ስክሪፕቱ ብዙ ደረጃዎችን በራስ-ሰር ያስተናግዳል።
በመጀመሪያ ከLLVM 14 እስከ `apt-get` ድረስ የሚፈለጉትን ጥቅሎች ይጫኑ።
የወደፊት ስርዓቶች LLVMን በአስተማማኝ ሁኔታ ማጣቀስ እንዲችሉ ወደ `/usr/lib/libllvm-14.so` ምሳሌያዊ አገናኝ ይፍጠሩ።

Wave አጠናቃሪው LLVM በትክክል እንዲያገኝ የ`LLVM_SYS_140_PREFIX` አካባቢን ተለዋዋጭ ያዘጋጃል።
ይህ ቅንብር ወደ `~/.bashrc` ታክሏል እና በወደፊቱ የተርሚናል ክፍለ-ጊዜዎች ላይ ይቀጥላል።

በመቀጠል በተጠቃሚ የተገለጸውን የWave ጥቅል (`.tar.gz`) ስሪት ያውርዱ እና ይክፈቱ።
ዚፕውን ከከፈቱ በኋላ የ`wavec` executable ፋይልን ወደ `/usr/local/bin` ይጫኑ።
የ`wavec` ትዕዛዝ በሲስተሙ ውስጥ በማንኛውም ቦታ የሚገኝ እንዲሆን ያዋቅሩት።

መጫኑ ከተጠናቀቀ በኋላ የ`wavec --version` ትዕዛዝን በመጠቀም በትክክል መጫኑን ያረጋግጡ።

## መጫኑን ያረጋግጡ

መጫኑ ከተጠናቀቀ በኋላ የ Wave ማቀናበሪያ በትክክል መጫኑን ከዚህ በታች ያለውን ትዕዛዝ በማሄድ ማረጋገጥ ይችላሉ።

```bash
wavec --version
```

የተጫነው Wave የስሪት መረጃ ትዕዛዙን ሲፈጽም ከታየ በመደበኛነት ተጭኗል።

---

## Wave የማስወገጃ መመሪያ (`uninstall.sh`)

Waveን ከስርዓትዎ ማስወገድ ከፈለጉ የቀረበውን የማስወገጃ ስክሪፕት መጠቀም ይችላሉ።
ይህ ስክሪፕት በመጫን ሂደት ውስጥ የታከሉ ፋይሎችን እና ቅንብሮችን የማጽዳት ሃላፊነት አለበት።

### እንዴት ማስወገድ እንደሚቻል

በተርሚናል ውስጥ የሚከተለውን ትዕዛዝ ያሂዱ

```bash
curl -fsSL https://wave-lang.dev/uninstall.sh | bash
```

አንዴ ማስወገዱ ከተጠናቀቀ በኋላ የwavec ትዕዛዝ ስራ ላይ አይውልም።
ከWave ጋር የተገናኙ ተፈጻሚነት ያላቸው ፋይሎች እና ቅንብሮች ከእርስዎ ስርዓት ይሰረዛሉ።
