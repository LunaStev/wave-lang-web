# ሰላም አትም

ይህ ሰነድ በWave ቋንቋ በጣም መሠረታዊ የሆነውን የውጤት ፕሮግራም እንዴት እንደሚፃፍ ያብራራል።

---

## ምሳሌ ኮድ

```wave
fun main() {
    println("Hello Wave");
}
```

---

## መግለጫ
- `fun main()`

    Wave ይህ የፕሮግራሙ የመግቢያ ነጥብ ተግባር ነው። ሲሮጥ መጀመሪያ ይባላል።

- `println()`

    ጊዜያዊ የውጤት ተግባር ያለው ሕብረቁምፊ ያወጣል እና ከውጤቱ በኋላ የመስመር መግቻ (`\n`) ይጨምራል።

- `;` (ሴሚኮሎን)

    በWave ውስጥ ያሉ ሁሉም መግለጫዎች በሰሚኮሎን ይቋረጣሉ።

---

## የማስፈጸሚያ ውጤት

```text
Hello Wave
```

---

## ተጨማሪ ምሳሌዎች
Wave የሕብረቁምፊ ግንኙነትን ይደግፋል።

```wave
fun main() {
    var name: str = "Wave";
    println("Hello, {}!", name);
}
```

ውጤት፡

```text
Hello, Wave!
```

---

> ይህ ምሳሌ የWave መደበኛ ቤተ-መጽሐፍትን መሠረታዊ የውጤት ተግባራት አጠቃቀም ያሳያል።
