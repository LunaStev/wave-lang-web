# Vex ጥቅል አስተዳዳሪ

## አጠቃላይ እይታ

Vex ለWave ፕሮግራሚንግ ቋንቋ ራሱን የቻለ የጥቅል አስተዳዳሪ እና የግንባታ ስርዓት ነው።
Vex አጠቃላይ የፕሮጀክት አስተዳደርን ይደግፋል፣ የምንጭ ኮድ ጥገኝነት አስተዳደርን፣ የግንባታ ቅንብሮችን፣ የታለመ መድረክን መሰየምን፣ እና ሞጁሉን መጫን እና ማሰማራትን ጨምሮ።
ከውጭ ቋንቋዎች ወይም ስርዓቶች ጋር ተኳሃኝነት አይታሰብም፣ እና በWave ሥነ-ምህዳር ውስጥ ብቻ እንዲሠራ የተቀየሰ ነው።

## የንድፍ ግቦች
Vex የተነደፈው የሚከተሉትን ግቦች ግምት ውስጥ በማስገባት ነው።

ለWave ብቻ የተነደፈ፡ የታለመው ለWave ፕሮጀክት ብቻ፣ ለWave አገባብ፣ ሞጁል መዋቅር እና የማስፈጸሚያ አካባቢ የተመቻቸ ነው።

* ሊታወቅ የሚችል የትዕዛዝ መዋቅር፡ ያለ ውስብስብ የግንባታ ስክሪፕቶች በአንድ ትዕዛዝ ቁልፍ ተግባራትን ለማከናወን የተደራጀ።
* ባለብዙ ዒላማ ድጋፍ፡ እንደ ስርዓተ ክወና እና አርክቴክቸር መሰረት የግንባታ ኢላማዎችን በቀላሉ መቀየር ትችላለህ።
* በWSON ላይ የተመሰረተ የውቅር አስተዳደር፡ ሁሉም የፕሮጀክት ውቅር መረጃ በWSON (Wave Serialization Object Notation) ቅርጸት ይገለጻል።
* የማይንቀሳቀስ ግንባታ እና ማሰማራት፡- ፈጻሚዎች በስታስቲክስ የተገነቡ ናቸው እና በውጫዊ የስራ ጊዜ ላይ ሳይመሰረቱ በተናጥል ሊሰማሩ ይችላሉ።
