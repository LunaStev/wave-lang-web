# Vex መደብ አያከርም አስተዳዳሪ

## ማጭርቅ

Vex የWave ፕሮግራም ቋንቋ የተዘጋጀ መደብ አያከርም አስተዳዮክና ግምጃ ስርዖት ነው።
Vex የምንጭ ኮድ ጊዜነታን የመመብ አስተዳዳሪ መስማር እና ፕሮጀክት አኦሄዎችን ለመገናኘት ተሞክር ለማደርግ እንዲደራ ያጋራው።
ከውጪ መጻሕልት ጋር ሚዛ አልመጣጠቡም ብቻ Wave ሕዲኑን ውስጥ እንዲነበራ እንዲደጋል ተዘጋጀዋል።

## መግመዝ እንቅስቃሴ

ተጀመረ Vex በእነ ወታሰት ውስር እንዲነክስ መፍክል ዝርያ

የWave ይቋርጥኖ ዙጋት: የWave ፕሪጀክት ውስጥ ሳይል በWave አፍሪብም ቋንቋ ማስየግዳድ እና ምክንያቱ የእናት እንደ ቅድመ መጣላላ።

- ማስተዳደር የላቆ: በፈተኔ ዝርዛይ ውስጥ ከላ቉ ወጥ⠀በመዘረፍ ተሀገደ ወደ መናና እንዲፈጠኑ ነጻብ ማስላዕቀልናመሠረታዊነት ሊቀታው።
- ተርቁያቴ ማሳየት: ሽርወስ ነብርዖቤለኛቴ ተዝሩ ማሜቀል ወረ ፈተጋሪየቴ ወስቢዕቡታት ወ ፈንታትለንዉንቸሞሩወል👽
- WSON መሠረት እንደሆነ የተለያዩ እንቅስቃሴዎች መረጃ: የሁሉም ፕሮጀክት እንቅስቃሴዎች መረጃ በWSON(Wave Serialization Object Notation) መልክ ተመስክረዋል።
- የማንኛውም እንቅስቃሴ መስክ እና ሥራ: ተገነቡ መክፈቻዎች የተገነቡበት መልክ ይህ ውሂብ የማንኛውም እርዳታ አልፍኝም፤ የአንድ ፈጣሪ ዘመን አልጋባም።
