Wave OS ከሃርድዌር ቅርብ **ግምገማ በመስጠት** ፣ እንዲመረጥና እንዲፈቃድ ማቅላትን የምታበረዝብላቸዉ።

- **ምኴቱ ሃርድዌር**፡ በWave ቋንቋ የተሰኮ ኮዶችን በመጠቀም ስርዓት እንዲኖር ወንጪጸዉ።
