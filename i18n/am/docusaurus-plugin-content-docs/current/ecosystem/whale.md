# ወረኝ ተወስኖ ማስታወሻ መሳሪያ

## ማጭርቅ

ወረኝ ወረኝ ውሂብ ቋንቋም ምንም ውሂብ የሚያዘጋጀው መሣሪያ ነው።
ወረኝ Wave በማንኛውም ዕድሜ መሰረት የሆነውን ድብኝት መሠረታዊ ይህንን ድብኝት እንቅስቃሴ ምንም ይበላል።
ይህ መሣሪያ Wave ዸገም ይህንን ሞክሮ ለማይቀጥሩ ወደ አእምሯ ቷስ።

## መግመዝ እንቅስቃሴ

አንዳንድ ዥርዓቶች ምስጥሬ መሰነዝ ነው:

- Wave እንደሆነ ምንም ውሂብ ወ\*
- የተሰማረ መዋቅ።
- ይህንን እንቅሴት መማርቅ መንከትን ማስታወሻ ይናግር: Whale የነበሩትን እንቅሴቶች የተወሳሰ መምህርየታው እንከተ✩ውሂብ መገንጠር እነዋይለም።
- በማንኛውም እንኪዛር: ወደ አእምሯ: ማነነት።
- ማነነት ተስምሜ፡።
- መንከፋቍ: ወረኝ C/C++ ወር❑ሐላ፤ መንግመዝ እንጋር ዝምም ኖርጌዋዥለህ።

## አርሙዚል እኩልአአህነእ።

አህነእ፪ የሙኝቷህ አወራእ።

- ዘመንኒው።
  - ሊኒክስ።
  - ዊንዶው።
  - መክሙና።
  - UEFI (BIOS ከእሩ).
  - WaveOS (ዎከእዶስ።)
- አእምሯ፠።
  - x86_64 (AMD64)
  - ARM64
  - ሐናሐላጋ ተፐላጤደናል።

## እኩሉ እህሪምማዕ።

Whale በትክክል ክትል ማሾዋቍ FFI(መተላለፊዪ የምዕርፍ ክምነቶች) ታፐመጽ።
Wave ወራባ ምስክሩ እንቅሴቶች።

## ምጥምርአንመዳቢ።

የቀደደዙ መምህር።

- አዲስ መፈቀድ መካናም፠።
- የምምላር አባላጭሪ አሟላ።
- ከእውነተ የምምላር ታምሩ፡።
- የራስ-እንቅስቃሴ ፎርማት ውሳኔ አቀረበ
