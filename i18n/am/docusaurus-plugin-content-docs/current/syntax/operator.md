# ኦፕሬተር

ይህ ሰነድ አሁን ባለው ኮምፕሌተር ላይ በመመስረት ትክክለኛውን ኦፕሬተሮች ያደራጃል.

## አርቲሜቲክ

| ኦፕሬተር | መግለጫ |
| --- | --- |
| `+` | መደመር |
| `-` | መቀነስ |
| `*` | ማባዛት |
| `/` | ክፍል |
| `%` | እረፍት |

## አወዳድር

| ኦፕሬተር | መግለጫ |
| --- | --- |
| `==` | እኩል |
| `!=` | የተለየ |
| `<` | ትንሽ |
| `<=` | ያነሰ ወይም እኩል |
| `>` | ትልቅ |
| `>=` | ይበልጣል ወይም እኩል |

## አመክንዮ

| ኦፕሬተር | መግለጫ |
| --- | --- |
| `&&` | ምክንያታዊ AND |
| `\|\|` | ምክንያታዊ ወይም |
| `!` | አመክንዮ አይደለም |

## ትንሽ

| ኦፕሬተር | መግለጫ |
| --- | --- |
| `&` | bitwise AND |
| `\|` | ቢት OR |
| `^` | bitwise XOR |
| `~` | ትንሽ አይደለም |
| `<<` | ወደ ግራ ፈረቃ |
| `>>` | የቀኝ ፈረቃ |

## የኮሌጅ መግቢያዎች

| ኦፕሬተር | መግለጫ |
| --- | --- |
| `=` | መሠረታዊ ምደባ |
| `+=` | ከተደመር በኋላ ምደባ |
| `-=` | ከተቀነሰ በኋላ ምደባ |
| `*=` | ከተባዛ በኋላ ምደባ |
| `/=` | ከመከፋፈል በኋላ መተካት |
| `%=` | ከቀሪው በኋላ ምደባ |

## ያልተለመደ / ጠቋሚ / ውሰድ

| ኦፕሬተር/ቁልፍ ቃል | መግለጫ |
| --- | --- |
| `++`, `--` | የፊት/ወደ ኋላ መጨመር/መቀነስ |
| `&x` | አድራሻ ያግኙ |
| `deref p` | ጠቋሚ ማጣቀሻ |
| `expr as T` | ግልጽ ቀረጻ |

## የጠቋሚ አርቲሜቲክ

| አገላለጽ | ውጤቶች |
| --- | --- |
| `ptr<T> + int` | `ptr<T>` (የጂኢፒ እንቅስቃሴ) |
| `int + ptr<T>` | `ptr<T>` (የጂኢፒ እንቅስቃሴ) |
| `ptr<T> - int` | `ptr<T>` (የጂኢፒ እንቅስቃሴ) |
| `ptr<T> - ptr<T>` | `i64` (የባይት ልዩነት) |
| `ptr == ptr`, `ptr != ptr` | የጠቋሚ ንጽጽር |

## የተያዙ ወይም ያልተተገበሩ ዕቃዎች

የሰዋሰው ቶከኖች አሉ፣ ነገር ግን አንዳንዶቹ በአሁኑ ጊዜ በመግለፅ ስራዎች አይደገፉም።
ምሳሌ፡ `??`፣ `?:`፣ `in`፣ `is`፣ `!&`፣ `!|`፣ `~^`።
