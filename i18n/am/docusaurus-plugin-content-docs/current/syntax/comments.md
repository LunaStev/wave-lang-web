# ቆርቆሮ

Wave ሁለት ማብራሪያዎችን ይደግፋል፡-

- የአንድ መስመር አስተያየት፡ `//`
- አስተያየት አግድ፡ `/* ... */`

## አንድ መስመር አስተያየት

ከ`//` በኋላ ያለው ማንኛውም ነገር እስከ መስመሩ መጨረሻ ድረስ ችላ ይባላል።

```wave
var x: i32 = 10; // line comment
x += 5;          // still works
```

## አግድ ማብራሪያ

በ`/*` እና `*/` መካከል ያለውን ይዘቶች ችላ ይበሉ።

```wave
var y: i32 = 1 /* inline block */ + 2;
```

አስተያየቶችን አግድ በርካታ መስመሮችን እና መክተቻዎችን ይደግፋሉ።

```wave
/* outer
   /* inner */
   outer end
*/
```

## ሕብረቁምፊዎች እና አስተያየት ምልክቶች

በሕብረቁምፊው ውስጥ `/*`፣ `*/`፣ `//` የአስተያየት መጀመሪያ/መጨረሻ ተደርጎ አይወሰድም።

```wave
var marker: str = "/*//*/";
```

## የማብራሪያ ስህተት

የማገጃው አስተያየት ካልተዘጋ፣ የማጠናቀር ስህተት (`E1002`) ይከሰታል።

```wave
/* not closed
```

አቀናባሪው የመነሻ ቦታውን፣ መንስኤውን እና የማስተካከያ ፍንጮችን ያወጣል።
