# コメント

Waveは2種類のコメントをサポートしています。

- 1行コメント: `//`
- ブロックコメント: `/* ... */`

## 1行コメント

`//` の後の内容は行末まで無視されます。

```wave
var x: i32 = 10; // line comment
x += 5;          // still works
```

## ブロックコメント

`/*` と `*/` の間の内容が無視されます。

```wave
var y: i32 = 1 /* inline block */ + 2;
```

ブロックコメントは複数行とネストをサポートします。

```wave
/* outer
   /* inner */
   outer end
*/
```

## 文字列とコメント記号

文字列内の `/*`, `*/`, `//` はコメント開始・終了として扱われません。

```wave
var marker: str = "/*//*/";
```

## コメントエラー

ブロックコメントが閉じられないとコンパイルエラー（`E1002`）が発生します。

```wave
/* not closed
```

コンパイラは開始位置、原因、修正ヒントを一緒に出力します。
