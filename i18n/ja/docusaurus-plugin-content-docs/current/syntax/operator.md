# 演算子

이 문서는 현재 컴파일러 기준으로 실제 사용 가능한 연산자를 정리합니다.

## 산술

| 演算子 | 説明  |
| --- | --- |
| `+` | 加算  |
| `-` | 減算  |
| `*` | 乗算  |
| `/` | 除算  |
| `%` | 나머지 |

## 비교

| 演算子  | 説明        |
| ---- | --------- |
| `==` | 同じ        |
| `!=` | 違い        |
| `<`  | 小さい       |
| `<=` | 小さいまたは等しい |
| `>`  | 大きい       |
| `>=` | 大きいまたは等しい |

## 논리

| 演算子        | 説明    |
| ---------- | ----- |
| `&&`       | 論理AND |
| \\\`\\ | 論理OR  |
| `!`        | 論理NOT |

## 비트

| 演算子        | 説明      |
| ---------- | ------- |
| `&`        | ビット AND |
| \\\`\\ | ビット OR  |
| `^`        | ビット XOR |
| `~`        | ビット NOT |
| `<<`       | 左シフト    |
| `>>`       | 右シフト    |

## 대입

| 演算子  | 説明       |
| ---- | -------- |
| `=`  | 기본 대입    |
| `+=` | 덧셈 후 대입  |
| `-=` | 뺄셈 후 대입  |
| `*=` | 곱셈 후 대입  |
| /=   | 나눗셈 후 대입 |
| `%=` | 나머지 후 대입 |

## 단항 / 포인터 관련

| 연산자/키워드    | 説明       |
| ---------- | -------- |
| `++`, `--` | 전위/후위 증감 |
| `&x`       | 주소 획득    |
| `deref p`  | 포인터 역참조  |

포인터의 경우 현재 `==`, `!=` 비교 중심으로 사용하며, 포인터 산술은 지원하지 않습니다.

## 예약 또는 미구현 항목

문법 토큰은 존재하지만 현재 표현식 연산으로는 지원되지 않는 항목이 있습니다.  
예: `??`, `?:`, `in`, `is`, `!&`, `!|`, `~^`.
