# Условное выражение IF

## Введение

В этом разделе рассматриваются синтаксис и использование одного из управляющих операторов языка Wave — условного выражения IF.
Условное выражение IF — это базовая структура управления, которая выполняет определённый блок кода только в том случае, если условие оказывается истинным.

Благодаря этому программа может выполнять различные действия в зависимости от условий и ситуаций, ни ограничиваясь только последовательным выполнением сверху вниз.
Условное выражение IF является ключевым элементом практически всех программ и необходимо для реализации логических ветвлений и управления потоком.

## Основная структура

Сначала условное выражение IF оценивает условие, и если оно истинно (True), то выполняется блок кода, записанный в фигурных скобках `{}`.
Если условие ложно (False), то этот блок пропускается, и выполнение переходит к следующему коду.

Основная структура условного выражения IF в языке Wave выглядит следующим образом.

```wave
if (условие) {
    // Код, выполняемый если условие истинно
}
```

В условных выражениях можно свободно использовать операторы сравнения и логические операторы.
Например, можно сравнивать значения с помощью операторов сравнения, таких как `==`, `!=`, `<`, `>`, `<=`, `>=`, а также сочетать несколько условий с помощью логических операторов, таких как `&&`, `||`, `!`.

Результат условия всегда должен оцениваться как истинный или ложный, и если условие оказывается ложным, код внутри блока IF не выполняется.

## Пример

Ниже приведён пример самого простого условного выражения IF.

```wave
var temperature :i32 = 30;

if (temperature > 25) {
    println("Погода жаркая.");
}
```

В этом коде оценивается, больше ли значение переменной `temperature`, чем 25.
Если условие истинно, выводится сообщение `"날씨가 덥습니다."`, а если условие ложно, ничего не происходит.

Таким образом, условное выражение IF используется тогда, когда необходимо выполнить код только при выполнении определённого условия.

## Условное выражение IF-ELSE.

Если есть код, который должен выполняться даже когда условие не является истинным, можно добавить блок ELSE к условному выражению IF.
Условное выражение IF-ELSE представляет собой структуру, которая выполняет один из двух блоков кода в зависимости от результата условия.

Базовая структура выглядит следующим образом.

```wave
if (условие) {
    // Код, выполняемый если условие истинно
} else {
    // Код, выполняемый если условие ложно
}
```

Если условие истинно, выполняется блок IF, если ложно — блок ELSE.
Выполняется только один из двух блоков, и одновременное выполнение невозможно.

Ниже приведён пример использования условного выражения IF-ELSE.

```wave
var score :i32 = 70;

if (score >= 60) {
    println("Вы прошли!");
} else {
    println("Вы не прошли.");
}
```

В этом коде выводятся разные сообщения в зависимости от того, больше или равно ли `score` 60.
Если условие истинно, выводится `"합격입니다!"`, в противном случае — `"불합격입니다."`.

## Вложенные условные выражения

Условные выражения IF можно использовать внутри других условных выражений IF, и это называется вложенными условными выражениями IF.
Вложенные условные выражения IF полезны, когда необходимо последовательно оценить несколько уровней условий.

Следующий пример демонстрирует вложенное условное выражение IF, которое выводит различные результаты в зависимости от оценки.

```wave
var score :i32 = 85;

if (score >= 60) {
    if (score >= 90) {
        println("Отличный результат!");
    } else {
        println("Вы прошли.");
    } 
} else {
    println("Вы не прошли.");
}
```

В этом коде сначала проверяется, больше ли оценка 60.
Если она меньше 60, сразу выводится `"불합격입니다."`.
Если она 60 или выше, условие проверяется снова: если оценка 90 или выше, выводится `"우수한 성적입니다!"`, в противном случае — `"합격입니다."`.

Таким образом, использование вложенных условных выражений IF позволяет поэтапно выразить сложные условные ветвления.

## Резюме

Условное выражение IF — это базовый оператор управления, который оценивает условия и контролирует поток выполнения программы.
Совместное использование с блоком ELSE позволяет однозначно определить действия в случае ложности условия, а использование вложенных условных выражений IF позволяет обрабатывать сложные ветвления, объединяющие несколько условий.

Грамотное использование условных выражений IF позволяет сделать поток выполнения программы более логичным и ясным.
