# Оператор

이 문서는 현재 컴파일러 기준으로 실제 사용 가능한 연산자를 정리합니다.

## 산술

| Оператор | Описание  |
| -------- | --------- |
| `+`      | Сложение  |
| `-`      | Вычитание |
| `*`      | Умножение |
| `/`      | Деление   |
| `%`      | 나머지       |

## 비교

| Оператор | Описание         |
| -------- | ---------------- |
| `==`     | Равно            |
| `!=`     | Не равно         |
| `<`      | Меньше           |
| `<=`     | Меньше или равно |
| `>`      | Больше           |
| `>=`     | Больше или равно |

## 논리

| Оператор   | Описание       |
| ---------- | -------------- |
| `&&`       | Логическое И   |
| \\\`\\ | Логическое ИЛИ |
| `!`        | Логическое НЕ  |

## 비트

| Оператор   | Описание      |
| ---------- | ------------- |
| `&`        | Побитовое И   |
| \\\`\\ | Побитовое ИЛИ |
| `^`        | Побитовое XOR |
| `~`        | Побитовое НЕ  |
| `<<`       | Сдвиг влево   |
| `>>`       | Сдвиг вправо  |

## 대입

| Оператор | Описание |
| -------- | -------- |
| `=`      | 기본 대입    |
| `+=`     | 덧셈 후 대입  |
| `-=`     | 뺄셈 후 대입  |
| `*=`     | 곱셈 후 대입  |
| `/=`     | 나눗셈 후 대입 |
| `%=`     | 나머지 후 대입 |

## 단항 / 포인터 / 캐스트

| 연산자/키워드     | Описание |
| ----------- | -------- |
| `++`, `--`  | 전위/후위 증감 |
| `&x`        | 주소 획득    |
| `deref p`   | 포인터 역참조  |
| `expr as T` | 명시적 캐스트  |

## 포인터 연산

| 표현식                        | 결과                                   |
| -------------------------- | ------------------------------------ |
| `ptr<T> + int`             | `ptr<T>` (GEP 이동) |
| `int + ptr<T>`             | `ptr<T>` (GEP 이동) |
| `ptr<T> - int`             | `ptr<T>` (GEP 이동) |
| `ptr<T> - ptr<T>`          | `i64` (바이트 차이)    |
| `ptr == ptr`, `ptr != ptr` | 포인터 비교                               |

## 예약 또는 미구현 항목

문법 토큰은 존재하지만 현재 표현식 연산으로는 지원되지 않는 항목이 있습니다.
예: `??`, `?:`, `in`, `is`, `!&`, `!|`, `~^`.
