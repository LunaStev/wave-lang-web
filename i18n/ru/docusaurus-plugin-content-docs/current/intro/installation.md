# Установка

## Способ установки

Wave можно легко установить с помощью предоставленного скрипта установки.
После выполнения следующей команды в терминале будет автоматически установлен указанный вами выпуск компилятора Wave (`wavec`).

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version <version>
```

Скрипт установки проверяет окружение системы и автоматически настраивает зависимости и компилятор, необходимые для выполнения Wave.
Если версия не указана, будет установлен последний стабильный выпуск или базовая версия, соответствующая указанным критериям.

## Пример установки

Для установки последней версии выполните следующую команду.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- latest
```

Если вы хотите установить конкретную версию, используйте опцию `--version`.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version v0.1.3-pre-beta
```

Вы также можете указать более детализированную версию, такую как nighty build.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version v0.1.3-pre-beta-nightly-2025-07-11
```

## Операции, выполняемые при установке

Скрипт установки автоматически выполняет несколько этапов, чтобы Wave мог работать корректно.
Сначала установите необходимые пакеты, связанные с LLVM 14, с помощью `apt-get`.
Затем создайте символическую ссылку на `/usr/lib/libllvm-14.so`, чтобы система могла стабильно ссылаться на LLVM.

Установите переменную окружения `LLVM_SYS_140_PREFIX`, чтобы компилятор Wave мог правильно находить LLVM. Эта настройка добавляется в `~/.bashrc` и сохраняется для последующих сеансов терминала.

Затем загрузите и распакуйте пакет Wave в формате `.tar.gz` указанной версии.
После распаковки установите исполняемый файл `wavec` в `/usr/local/bin`, чтобы команда `wavec` была доступна из любой точки системы.

После завершения установки проверьте корректность установки, выполнив команду `wavec`.

## Проверка установки

После установки вы можете проверить корректность установки компилятора Wave, выполнив следующую команду.

```bash
wavec --version
```

Если при выполнении команды выводится информация о версии установленного Wave, значит установка прошла успешно.

## Руководство по удалению Wave (`uninstall.sh`)

Если вы хотите удалить Wave из системы, вы можете использовать предоставленный скрипт удаления.
Этот скрипт предназначен для удаления файлов и настроек, добавленных в процессе установки.

### Способ удаления

Выполните следующую команду в терминале.

```bash
curl -fsSL https://wave-lang.dev/uninstall.sh | bash
```

После завершения удаления команда `wavec` больше не будет доступна, и исполняемые файлы и настройки, связанные с Wave, будут удалены из системы.
