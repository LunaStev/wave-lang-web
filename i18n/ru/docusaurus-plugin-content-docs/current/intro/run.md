# Запуск первой программы

Если Вы уже установили Wave по прежней документации, то вы теперь можете запустить первую программу на Wave.
В этом разделе постуление выполняем прощую программу на Wave и посмотрим на лучшие примеры отдельно.

## Создађем файл `hello.wave`

Начнем с создания нового файла с именем `hello.wave` в корневой каталог.
Вы можете свободно задать имя и расширение файла, но в этом примере мы создам `hello.wave`.

## Написание кода

Запишите код, приведенный ниже, в созданном файле `hello.wave`.

```wave
fun main() {
    println("Hello Wave");
}
```

Функция `fun main()` в данном коде обозначает точку начала выполнения Wave программы.
Программы Wave постоянно запускаются с функции `main`.

Функция `println` извлекает строку на экран и часто применяется для вывода текста в консоли.

## Запуск программы

Если вы написали код, откройте терминал и переместитесь в директорию с файлом, затем выполните следующую команду.

```bash
wavec run hello.wave
```

Эта команда скомпилирует ваш исходник на Wave, затем отражает для него в реальное время и сразу его исполняет.

## Проверка вывода

Когда программа будет выполнена успешно, в терминале появится следующий вывод.

```
Hello Wave
```

Если этот вывод виден, то Wave установлен и программная подготовка и запуск успели.

Теперь первая программа на Wave успешно запущена.
После этого мы можем постепенно ознакомиться с синтаксисом и функциями Wave и пробуем виполнять более сложные программы.

Точные опции команд (`-O*`, `--debug-wave`, `--link`, `--dep-root`, `--dep`) можно найти в документации CLI `wavec`.
