# ইন্সটলেশন

## ইনস্টল পদ্ধতি

ওয়েভ সরবরাহিত ইনস্টলেশন স্ক্রিপ্টের মাধ্যমে সহজেই ইনস্টল করা যায়।
টার্মিনালে নিম্নলিখিত কমান্ডগুলি চালালে নির্দিষ্ট সংস্করণের ওয়েভ কম্পাইলার (`wavec`) স্বয়ংক্রিয়ভাবে ইনস্টল হয়ে যায়।

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version <version>
```

ইনস্টলেশন স্ক্রিপ্ট সিস্টেম পরিবেশ পরীক্ষা করার পর, ওয়েভ চালানোর জন্য প্রয়োজনীয় ডিপেন্ডেন্সি ও কম্পাইলার স্বয়ংক্রিয়ভাবে কনফিগার করে।
যদি সংস্করণ নির্দিষ্ট না করা হয় তবে সাম্প্রতিক স্থিতিশীল সংস্করণ বা নির্দিষ্ট মানদণ্ড অনুযায়ী ডিফল্ট সংস্করণ ইনস্টল হবে।

## ইনস্টলেশনের উদাহরণ

সাম্প্রতিক সংস্করণ ইনস্টল করতে হলে নিচে দেখানো মতো কমান্ড চালান।

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- latest
```

যদি নির্দিষ্ট সংস্করণ ইনস্টল করতে চান তবে `--version` অপশন ব্যবহার করুন।

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version v0.1.3-pre-beta
```

নাইটলি বিল্ডের মতো আরও বিস্তারিত সংস্করণও নির্দিষ্ট করা সম্ভব।

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version v0.1.3-pre-beta-nightly-2025-07-11
```

## ইনস্টলের সময় সম্পন্ন কার্য

ইনস্টলেশন স্ক্রিপ্ট ওয়েভকে স্বাভাবিকভাবে চালানোর জন্য অনেকগুলো ধাপ স্বয়ংক্রিয়ভাবে পরিচালনা করে।
প্রথমে `apt-get` এর মাধ্যমে LLVM 14 এর সাথে সম্পর্কিত প্রয়োজনীয় প্যাকেজগুলি ইনস্টল করুন।
এরপর সিস্টেমে LLVM কে স্থিতিশীলভাবে উল্লেখ করার জন্য `/usr/lib/libllvm-14.so` এর একটি প্রতীকী লিঙ্ক তৈরি করুন।

ওয়েভ কম্পাইলার যেন সঠিকভাবে LLVM খুঁজে পায় তার জন্য `LLVM_SYS_140_PREFIX` পরিবেশ ভেরিয়েবল সেট করুন; এই সেটিংটি `~/.bashrc` তে যোগ করা হয় এবং পরবর্তী টার্মিনাল সেশনেও কার্যকর থাকে।

তারপর ব্যবহারকারী নির্দিষ্ট সংস্করণের Wave প্যাকেজ (`.tar.gz`) ডাউনলোড করে আনজিপ করুন।
আনজিপ করার পর `wavec` এক্সিকিউটেবল ফাইলটিকে `/usr/local/bin` এ ইনস্টল করে সিস্টেম যে কোন জায়গায় `wavec` কমান্ড ব্যবহার করা যায়।

ইনস্টলেশন সম্পন্ন হলে `wavec` কমান্ডের মাধ্যমে সঠিকভাবে ইনস্টল হয়েছে কিনা যাচাই করুন।

## ইনস্টলেশন নিশ্চিতকরণ

ইনস্টলেশন সম্পন্ন হলে, নিচের কমান্ড রান করে Wave কম্পাইলার সঠিকভাবে ইনস্টল হয়েছে কিনা যাচাই করা যায়।

```bash
wavec --version
```

কমান্ড চালানোর সময় যদি ইনস্টল করা Wave এর সংস্করণ তথ্য প্রদর্শিত হয় তবে ইনস্টলেশন সঠিকভাবে সম্পন্ন হয়েছে।

## Wave অপসারণ নির্দেশিকা (`uninstall.sh`)

যদি সিস্টেম থেকে Wave অপসারণ করতে চান, তাহলে সরবরাহিত আনইনস্টলেশন স্ক্রিপ্ট ব্যবহার করা যেতে পারে।
এই স্ক্রিপ্ট ইনস্টলেশনের সময় যোগ করা ফাইল এবং সেটিংস গুলো সরানোর কাজ করে।

### অপসারণ পদ্ধতি

টার্মিনালে নিম্নলিখিত কমান্ডগুলি চালান।

```bash
curl -fsSL https://wave-lang.dev/uninstall.sh | bash
```

অপসারণ সম্পন্ন হলে, wavec কমান্ড আর ব্যবহার করা যাবে না এবং Wave সম্পর্কিত এক্সিকিউটেবল ফাইল ও সেটিংস সিস্টেম থেকে মুছে ফেলা হবে।
