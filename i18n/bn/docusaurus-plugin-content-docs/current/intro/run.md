# প্রথম প্রোগ্রাম চালানো

পূর্ববর্তী ইনস্টলেশন নথি থেকে ওয়েভ ইতিমধ্যেই ইনস্টল করা থাকলে, এবার প্রথম ওয়েভ প্রোগ্রামটি সরাসরি চালানোর পালা।
এই বিভাগে, সাধারণ উদাহরণ দ্বারা ওয়েভ প্রোগ্রাম লেখা এবং চালানোর সম্পূর্ণ প্রক্রিয়া ধাপক্রমে দেখা যাবে।

## `hello.wave` ফাইল তৈরি করা

প্রথমে কাজ করার ডিরেক্টরিতে `hello.wave` নামে নতুন একটি ফাইল তৈরি করুন।
ফাইলের নাম এবং এক্সটেনশন স্বাধীনভাবে নির্দিষ্ট করা যেতে পারে, তবে এখানে উদাহরণ হিসেবে `hello.wave` ব্যবহার করেছি।

## কোড লেখা

তৈরি করা `hello.wave` ফাইলে নিচের কোড লিখুন।

```wave
fun main() {
    println("হ্যালো Wave");
}
```

এই কোডে, `fun main()` ওয়েভ প্রোগ্রামের নির্বাহ শুরু বিন্দু নির্দেশ করে।
ওয়েভ প্রোগ্রাম সর্বদা `main` ফাংশন থেকে শুরু হয়।

`println` ফাংশন হল একটি ফাংশন যা স্ট্রিংকে স্ট্যান্ডার্ড আউটপুটে মুদ্রণ করে,
স্ক্রিনে টেক্সট প্রদর্শনের জন্য মৌলিকভাবে ব্যবহৃত হয়।

## প্রোগ্রাম চালানো

কোড লেখা শেষ হলে, টার্মিনাল খুলে ফাইলের অবস্থানের ডিরেক্টরিতে গিয়ে নিম্নলিখিত কমান্ডটি চালান।

```bash
wavec রান hello.wave
```

এই কমান্ডটি নির্দেশ দেয় যে ওয়েভ কম্পাইলার উৎস ফাইলটি কম্পাইল করার পরে,
তৎক্ষণাৎ নির্বাহ যেন সম্পন্ন হয়।

## আউটপুট পরীক্ষা করা

যদি প্রোগ্রামটি সঠিকভাবে চালানো হয়, টার্মিনালে নিম্নলিখিত আউটপুট প্রদর্শিত হবে।

```
হ্যালো Wave
```

এই আউটপুট দেখা গেলে, বুঝে নিতে হবে যে ওয়েভ সঠিকভাবে ইনস্টল হয়েছে এবং প্রোগ্রাম লেখা এবং নির্বাহ সঠিকভাবে সম্পন্ন হয়েছে।

এখন প্রথম ওয়েভ প্রোগ্রামটি সফলভাবে চালানো হয়েছে।
এখন থেকে আমরা ওয়েভের ব্যাকরণ এবং কার্যক্রম একে একে দেখে নিয়ে, আরো জটিল প্রোগ্রাম লেখা চেষ্টা করতে পারি।

সুনির্দিষ্ট কমান্ড বিকল্পগুলি (`-O*`, `--debug-wave`, `--link`, `--dep-root`, `--dep`) হল
`wavec` CLI নথিতে পাওয়া যাবে।
