# মন্তব্য

Wave দুটি প্রকারের মন্তব্য সমর্থন করে।

- এক লাইনের মন্তব্য: `//`
- ব্লক মন্তব্য: `/* ... */`

## এক লাইনের মন্তব্য

`//` পরবর্তী বিষয়বস্তু লাইন শেষ পর্যন্ত উপেক্ষা করা হয়।

```wave
var x: i32 = 10; // লাইনের মন্তব্য
x += 5;          // এখনও কাজ করছে
```

## ব্লক মন্তব্য

`/*` এবং `*/` এর মধ্যে থাকা বিষয়বস্তু উপেক্ষা করা হয়।

```wave
var y: i32 = 1 /* ইনলাইন ব্লক */ + 2;
```

ব্লক মন্তব্যগুলি বহু লাইন এবং নেস্টিং সমর্থন করে।

```wave
/* বাইরের
   /* অভ্যন্তরীণ */
   বাইরের শেষ
*/
```

## স্ট্রিং এবং মন্তব্য চিহ্ন

স্ট্রিংয়ের অভ্যন্তরে থাকা `/*`, `*/`, `//` কে মন্তব্যের শুরু/শেষ হিসাবে বিবেচনা করা হয় না।

```wave
var marker: str = "/*//*/";
```

## মন্তব্য ত্রুটি

ব্লক মন্তব্য যদি বন্ধ না হয় তবে কম্পাইল ত্রুটি (`E1002`) তৈরি হয়।

```wave
/* বন্ধ নয়
```

কম্পাইলার শুরু অবস্থান, কারণ, সংশোধনীরা একসাথে আউটপুট দেয়।
