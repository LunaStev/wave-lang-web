# অপারেটর

এই নথিটি বর্তমান কম্পাইলার ভিত্তিক ব্যবহারযোগ্য অপারেটর সংগ্রহ করে।

## অঙ্কশাস্ত্র

| অপারেটর | বর্ণনা     |
| ------- | ---------- |
| `+`     | যোগ        |
| `-`     | বিয়োগ     |
| `*`     | গুণ        |
| `/`     | ভাগ        |
| `%`     | অবশিষ্টাংশ |

## তুলনা

| অপারেটর | বর্ণনা        |
| ------- | ------------- |
| `==`    | সমান          |
| `!=`    | বিভিন্ন       |
| `<`     | ছোট           |
| `<=`    | ছোট বা সমান   |
| `>`     | বড়           |
| `>=`    | বড় অথবা সমান |

## যৌক্তিক

| অপারেটর  | বর্ণনা       |
| -------- | ------------ |
| `&&`     | এন্ড অপারেটর |
| \\`\\ | অথবা অপারেটর |
| `!`      | লজিক্যাল NOT |

## বিট

| অপারেটর  | বর্ণনা       |
| -------- | ------------ |
| `&`      | বিট AND      |
| \\`\\ | বিট OR       |
| `^`      | বিট XOR      |
| `~`      | বিট NOT      |
| `<<`     | বামদিকে শিফট |
| `>>`     | ডানদিকে শিফট |

## বরাদ্দ

| অপারেটর | বর্ণনা                 |
| ------- | ---------------------- |
| `=`     | মূল বরাদ্দ             |
| `+=`    | যোগের পর বরাদ্দ        |
| `-=`    | বিয়োগের পর বরাদ্দ     |
| `*=`    | গুণের পর বরাদ্দ        |
| `/=`    | ভাগের পর বরাদ্দ        |
| `%=`    | অবশিষ্টাংশের পর বরাদ্দ |

## একমাত্র / পয়েন্টার / কাস্ট

| অপারেটর/কীওয়ার্ড | বর্ণনা                    |
| ----------------- | ------------------------- |
| `++`, `--`        | পূর্ববর্তী/পরবর্তী বৃদ্ধি |
| `&x`              | ঠিকানা অর্জন              |
| `deref p`         | পয়েন্টার ডেরেফারেন্স     |
| `expr as T`       | স্পষ্ট কাস্ট              |

## পয়েন্টার অপারেশন

| প্রকাশ                     | ফলাফল                                        |
| -------------------------- | -------------------------------------------- |
| `ptr<T> + int`             | `ptr<T>` (GEP স্থানান্তর) |
| `int + ptr<T>`             | `ptr<T>` (GEP স্থানান্তর) |
| `ptr<T> - int`             | `ptr<T>` (GEP স্থানান্তর) |
| `ptr<T> - ptr<T>`          | `i64` (বাইট পার্থক্য)     |
| `ptr == ptr`, `ptr != ptr` | পয়েন্টার তুলনা                              |

## রিজার্ভ অথবা অবাস্তবায়িত আইটেম

সিনট্যাক্স টোকেন উপস্থিত থাকলেও এটি বর্তমান এক্সপ্রেশন অপারেশন দ্বারা সমর্থিত নয় এমন আইটেম রয়েছে।
উদাহরণ: `??`, `?:`, `in`, `is`, `!&`, `!|`, `~^`.
