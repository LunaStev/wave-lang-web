# 注释

Wave支持两种注释。

- 单行注释：`//`
- 块注释：`/* ... */`

## 单行注释

`//`后的内容将被忽略至行末。

```wave
var x: i32 = 10; // 行注释
x += 5; // 仍然有效
```

## 块注释

忽略`/*`和`*/`之间的内容。

```wave
var y: i32 = 1 /* 内联块 */ + 2;
```

块注释支持多行和嵌套。

```wave
/* 外
 /* 内 */
 外结束
*/
```

## 字符串和注释符号

字符串内部的`/*`、`*/`、`//`不会被视为注释的开始或结束。

```wave
var marker: str = "/*//*/";
```

## 注释错误

如果块注释未关闭，将产生编译错误（`E1002`）。

```wave
/* 未关闭
```

编译器会输出起始位置、原因和修正提示。
