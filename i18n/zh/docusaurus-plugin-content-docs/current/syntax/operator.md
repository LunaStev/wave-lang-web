# 运算符

이 문서는 현재 컴파일러 기준으로 실제 사용 가능한 연산자를 정리합니다.

## 산술

| 运算符 | 说明  |
| --- | --- |
| `+` | 加法  |
| `-` | 减法  |
| `*` | 乘法  |
| `/` | 除法  |
| `%` | 나머지 |

## 비교

| 运算符  | 说明   |
| ---- | ---- |
| `==` | 等于   |
| `!=` | 不等于  |
| `<`  | 小于   |
| `<=` | 小于等于 |
| `>`  | 大于   |
| `>=` | 大于等于 |

## 논리

| 运算符        | 说明  |
| ---------- | --- |
| `&&`       | 逻辑与 |
| \\\`\\ | 逻辑或 |
| `!`        | 逻辑非 |

## 비트

| 运算符        | 说明   |
| ---------- | ---- |
| `&`        | 位与   |
| \\\`\\ | 位或   |
| `^`        | 位异或  |
| `~`        | 位非   |
| `<<`       | 左移运算 |
| `>>`       | 右移运算 |

## 대입

| 运算符  | 说明       |
| ---- | -------- |
| `=`  | 기본 대입    |
| `+=` | 덧셈 후 대입  |
| `-=` | 뺄셈 후 대입  |
| `*=` | 곱셈 후 대입  |
| `/=` | 나눗셈 후 대입 |
| `%=` | 나머지 후 대입 |

## 단항 / 포인터 / 캐스트

| 연산자/키워드     | 说明       |
| ----------- | -------- |
| `++`, `--`  | 전위/후위 증감 |
| `&x`        | 주소 획득    |
| `deref p`   | 포인터 역참조  |
| `expr as T` | 명시적 캐스트  |

## 포인터 연산

| 표현식                        | 결과                                   |
| -------------------------- | ------------------------------------ |
| `ptr<T> + int`             | `ptr<T>` (GEP 이동) |
| `int + ptr<T>`             | `ptr<T>` (GEP 이동) |
| `ptr<T> - int`             | `ptr<T>` (GEP 이동) |
| `ptr<T> - ptr<T>`          | `i64` (바이트 차이)    |
| `ptr == ptr`, `ptr != ptr` | 포인터 비교                               |

## 예약 또는 미구현 항목

문법 토큰은 존재하지만 현재 표현식 연산으로는 지원되지 않는 항목이 있습니다.
예: `??`, `?:`, `in`, `is`, `!&`, `!|`, `~^`.
