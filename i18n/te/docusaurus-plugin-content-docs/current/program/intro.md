# హలోని ముద్రించండి

Wave భాషలో అత్యంత ప్రాథమిక అవుట్‌పుట్ ప్రోగ్రామ్‌ను ఎలా వ్రాయాలో ఈ పత్రం వివరిస్తుంది.

---

## ఉదాహరణ కోడ్

```wave
fun main() {
    println("Hello Wave");
}
```

---

## వివరణ
- `fun main()`

    Wave ఇది ప్రోగ్రామ్ యొక్క ఎంట్రీ పాయింట్ ఫంక్షన్. పరిగెత్తినప్పుడు మొదట అంటారు.

- `println()`

    తాత్కాలిక అవుట్‌పుట్ ఫంక్షన్‌తో స్ట్రింగ్‌ను అవుట్‌పుట్ చేస్తుంది మరియు అవుట్‌పుట్ తర్వాత లైన్ బ్రేక్ (`\n`) జోడిస్తుంది.

- `;` (సెమికోలన్)

    Waveలోని అన్ని స్టేట్‌మెంట్‌లు సెమికోలన్‌తో ముగించబడతాయి.

---

## అమలు ఫలితం

```text
Hello Wave
```

---

## అదనపు ఉదాహరణలు
Wave స్ట్రింగ్ ఇంటర్‌పోలేషన్‌కు మద్దతు ఇస్తుంది.

```wave
fun main() {
    var name: str = "Wave";
    println("Hello, {}!", name);
}
```

అవుట్‌పుట్:

```text
Hello, Wave!
```

---

> ఈ ఉదాహరణ Wave ప్రామాణిక లైబ్రరీ యొక్క ప్రాథమిక అవుట్‌పుట్ ఫంక్షన్‌ల వినియోగాన్ని ప్రదర్శిస్తుంది.
