# IF స్టేట్‌మెంట్

## పరిచయం

ఈ విభాగంలో Wave భాష అందించబడిన నియంత్రణ స్టేట్‌మెంట్‌లలో ఒకటి అయిన IF స్టేట్‌మెంట్ యొక్క వ్యాకరణం మరియు ఉపయోగించు విధానాన్ని వివరిస్తుంది.
IF స్టేట్‌మెంట్ ఒక ఆదర్శ నియంత్రణ నిర్మాణం దాని కు అంచనాలు వెడితే ఆ పరిశీలన నిజమైతే ప్రత్యేక కోడ్ బ్లాక్‌ను అమలు చేస్తుంది.

ఈ దారిలో శ్రేణిగా సులభంగా అమలయే ప్రోగ్రామ్ నాటిని, యాత్ర మరియు పరిస్థితుల వల్ల విభిన్న కార్యకలాపాలు నిర్వహించగలుగుతుంది.
IF స్టేట్‌మెంట్ దాదాపు ప్రతి ప్రోగ్రామ్కు తలగాయ్యడైన అంశం మరియు తార్కిక విభజన మరియు కార్యక్రమం వరకు నియంత్రణను అమలు చేయడానికి అనుబంధముగా ఉపయోగించబడుతుంది.

## మూల నిర్మాణం

IF స్టేట్‌మెంట్ మొదటి అంచనాలను అంచనా వేస్తుంది మరియు స్థితి చెల్లుబాటులో ఉన్నట్లయితే `{}` లో పేర్కొన్న కోడ్ బ్లాక్‌ను అమలు చేస్తుంది.
స్థితి తప్పుడు ఉంటే ఆ బ్లాక్‌ను పక్కన బగ్గుని మరియు తరువాత కోడ్ మీదకి నివసించే వేళ.

Wave లో IF స్టేట్‌మెంట్ యొక్క మూల నిర్మాణం ఈ క్రింది విధంగా ఉంది.

```wave
if (సందర్భం) {
    // పరిస్థితి నిజంగా ఉన్నప్పుడు అమలు చేయబడే కోడ్
}
```

అప్పుడు సూత్రాలలో సరిపోయే O, H వంటి మరియు కూడా లాజి తరము  యాదృచ్ఛికంగా ఉపయోగపడుతుంది.
ఉదాహరణకు `==`, `!=`, `<`, `>`, `<=`, `>=` వంటి పోల్చు ప్రసరణదేదే చేసింది.
`&&`, `||`, `!` వంటి లాజికల్ సూక్ష్మాలను గాడీ చేయవచ్చు లేదా సాహితని కాంబైన్ చేయవచ్చు.

రాలి ప్రతియాటన విశేషాలు తప్పనిసరిగా విద్యలు లేదా పూకే గలుగుతున్నారని అంచనా వేయాలి, ఇది తప్పును పద్దులు వేయనవసరం అవుతుంది.

## ఉదాహరణ

కొనసాగింపు అతనిగాను ఉండే IF స్టేట్‌మెంట్ యొక్క అత్యంత సరళమైన ఉదాహరణ.

```wave
var temperature :i32 = 30;

if (temperature > 25) {
    println("అవెన్య అనే శుభ్రత")
}
```

ఈ కోడ్‌లోత `temperature` క్రమంలో 25 కంటే ఎక్కువ ఆ నుంచి అంచనా వేస్తుంది.
ఈస్థితి నిజమైతే `"అవెన్య అనే శుభ్రత"` అనే సందేశం ప్రింట్ చేస్తారు, ఇది తప్పుడు అయితే ఎలాంటి ప్రక్రియ కూడా జరగదు.

అది జాదీని కలిగి ఉండే రోజులేదు, అప్పుడు వేరు సంఖ్య తీసుకోండి అది జరుగుతుంది, అది దాని ప్రక్రియ తప్పనిసరిగా అమలు చేయాలి.

## IF-ELSE స్టేట్‌మెంట్

ఫలితంగా యాదృచ్ఛికంగా IF స్టేట్‌మెంట్‌లో ELSE విభాగాన్ని జోడించడం.
IF-ELSE స్టేట్‌మెంట్ నేపథ్య ప్రా, రెండు కోడ్ బ్లాక్ యొక్క ఫలితాలను మీరే ఆలోచించడమే.

సాధారణ మూలకాలు ఈ క్రింది విధంగా ఉన్నాయి.

```wave
if (సందర్భం) {
    // పరిస్థితి నిజంగా ఉన్నప్పుడు అమలు చేయబడే కోడ్
} else {
    // పరిస్థితి తప్పుడు ఉంటే అమలు చేయబడే కోడ్
}
```

IF బ్లాక్‌ను కొద్దిగా అమలు చేస్తే, ELSE బ్లాక్‌ను తగిన వాక్యాలనువెంటనే అసోసియేట్ చేయకపోతే అమలవుతాయి.
ఈ రెండు బ్లాక్‌లలో ఒకదానిని మాత్రమే అమలవేస్తుంది, ఒకటి ఇదే సమయంలో అమలవడం కాదు.

తరువాత IF-ELSE విధానాన్ని ఉపయోగించాల్సి వచ్చిన ఉత్పత్తి ఉంటుంది.

```wave
var score :i32 = 70;

if (score >= 60) {
    println("మీరు ఉత్తీర్ణులయారు!");
} else {
    println("మీరు అధ్యాయించరు.");
}
```

ఈ కోడ్‌లో `score` కంటే 60 ఎక్కువ అయితే మరచినా ప్రింట్ చేస్తుంది.
లాజికల్ కొనసా prawen కొన్ని కంటే నిజమైతే `"మీరు ఉత్తీర్ణులయారు!"` అని అని అంటుంది, నేను సత్యంగా చేయనవసరం ఉంటే `"మీరు ఆసురు ప్రాస్టింగ్!"`.

## పునఃరుద్దరణ IF స్టేట్‌మెంట్

IF స్టేట్‌మెంట్ ఇతర IF స్టేట్‌మెంట్‌లో కూడా ఉపయోగించవచ్చు, దీనిని పునఃరుద్దరణ IF స్టేట్‌మెంట్ అంటారు.
పునఃరుద్దరణ IF స్టేట్‌మెంట్ అనేక దశల పరిస్థితులను వరుసగా అంచనా వేయగలయి.

తరువాత స్పష్టమైన ప్రత్యేక రీతిలో సంచించండి.

```wave
var score :i32 = 85;

if (score >= 60) {
    if (score >= 90) {
        println("మీరు అత్యుత్తమ మార్కులు పొందారు!");
    } else {
        println("మీరు ఉత్తీర్ణులయారు!");
    }
} else {
    println("మీరు చదవలేదు.");
}
```

ఈ కోడ్‌లో మొదటి `score` 60 కంటే అధికమా అని అంచనా వేయగలదు.
60 కంటే తక్కువగా ఉంటే `"మీరు ఆసికల మంచిమండతు!"` అని ప్రింట్ చేస్తుంది.
60 కంటే ఎక్కువగా ఉంటే, `score` 90 కంటే పై లాక్యం తీసుకొని, `"మీరు అత్యుత్తమ మార్కులు పొందారు!"`, లేదా మునుపటి సందేశంలో `"మీరు ఉత్తీర్ణులయారు!"` ను ప్రింట్ చేస్తుంది.

Dengan menggunakan pernyataan IF bertingkat, anda dapat menyatakan percabangan syarat yang kompleks secara bertahap.

## Ringkasan

Pernyataan IF adalah pernyataan kawalan asas yang menilai syarat untuk mengawal aliran eksekusi program.
Dengan menggunakan ELSE, anda juga dapat mendefinisikan tindakan ketika syarat tersebut salah, dan menggunakan IF bertingkat untuk menggabungkan pelbagai syarat dalam pengendalian bercabang yang kompleks.

Dengan menggunakan pernyataan IF secara tepat, anda dapat membina aliran program yang lebih logik dan jelas.
