# 주석

Wave는 두 가지 주석을 지원합니다.

- 한 줄 주석: `//`
- 블록 주석: `/* ... */`

## 한 줄 주석

`//` 뒤의 내용은 줄 끝까지 무시됩니다.

```wave
var x: i32 = 10; // line comment
x += 5;          // still works
```

## 블록 주석

`/*` 와 `*/` 사이 내용을 무시합니다.

```wave
var y: i32 = 1 /* inline block */ + 2;
```

블록 주석은 여러 줄과 중첩을 지원합니다.

```wave
/* outer
   /* inner */
   outer end
*/
```

## 문자열과 주석 기호

문자열 내부의 `/*`, `*/`, `//`는 주석 시작/종료로 처리되지 않습니다.

```wave
var marker: str = "/*//*/";
```

## 주석 오류

블록 주석이 닫히지 않으면 컴파일 에러(`E1002`)가 발생합니다.

```wave
/* not closed
```

컴파일러는 시작 위치, 원인, 수정 힌트를 함께 출력합니다.
