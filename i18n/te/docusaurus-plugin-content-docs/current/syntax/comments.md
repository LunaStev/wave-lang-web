# టిన్

Wave రెండు ఉల్లేఖనాలకు మద్దతు ఇస్తుంది:

- వన్-లైన్ వ్యాఖ్య: `//`
- వ్యాఖ్యను నిరోధించు: `/* ... */`

## ఒక లైన్ వ్యాఖ్య

`//` తర్వాత ఏదైనా లైన్ ముగిసే వరకు విస్మరించబడుతుంది.

```wave
var x: i32 = 10; // line comment
x += 5;          // still works
```

## ఉల్లేఖనాన్ని నిరోధించండి

`/*` మరియు `*/` మధ్య కంటెంట్‌లను విస్మరించండి.

```wave
var y: i32 = 1 /* inline block */ + 2;
```

బ్లాక్ కామెంట్‌లు బహుళ పంక్తులు మరియు గూడు కట్టడానికి మద్దతు ఇస్తాయి.

```wave
/* outer
   /* inner */
   outer end
*/
```

## స్ట్రింగ్స్ మరియు వ్యాఖ్య గుర్తులు

స్ట్రింగ్ లోపల `/*`, `*/`, `//` వ్యాఖ్య ప్రారంభం/ముగింపుగా పరిగణించబడవు.

```wave
var marker: str = "/*//*/";
```

## ఉల్లేఖన లోపం

బ్లాక్ వ్యాఖ్య మూసివేయబడకపోతే, సంకలన లోపం (`E1002`) సంభవిస్తుంది.

```wave
/* not closed
```

కంపైలర్ ప్రారంభ స్థానం, కారణం మరియు దిద్దుబాటు సూచనలను అవుట్‌పుట్ చేస్తుంది.
