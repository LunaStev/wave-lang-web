# సూచిక

ఈ పత్రంలో ప్రస్తుత కంపైలర్ ప్రామాణికంగా చెలామణి అయ్యే సూచకాలను సేకరిస్తుంది.

## గణితం

| సూచిక | వివరణ      |
| ----- | ---------- |
| `+`   | తిరుగుడులు |
| `-`   | గుణ        |
| `*`   | గుణ        |
| `/`   | భగడ        |
| `%`   | తిరుగుళ్ళు |

## సూత్రాలు

| సూచిక | వివరణ                    |
| ----- | ------------------------ |
| `==`  | సమానం                    |
| `!=`  | భిన్నం                   |
| `<`   | చిన్నది                  |
| `<=`  | చిన్న ఉండగా లేదా సమానమైన |
| >     | పెద్దది                  |
| `>=`  | పెద్ద లేదా సమానం         |

## గణిత

| సూచిక    | వివరణ     |
| -------- | --------- |
| `&&`     | తర్కం AND |
| \\`\\ | తర్కం OR  |
| `!`      | తర్కం NOT |

## నిబంధలు

| సూచిక    | వివరణ                |
| -------- | -------------------- |
| `&`      | బిట్ AND             |
| \\`\\ | బిట్ OR              |
| `^`      | బిట్ XOR             |
| `~`      | బిట్ NOT             |
| `<<`     | ఎడమ స desplazamento  |
| `>>`     | కుడి స desplazamento |

## అలంకరణ చేయడం

| ఆపరేటర్ | వివరణ                 |
| ------- | --------------------- |
| `=`     | ప్రాధమిక సంబంధం       |
| `+=`    | జోడించ తర్వాత అలంకరణ  |
| `-=`    | వయసు తర్వాత అలంకరణ    |
| `*=`    | గోచి తర్వాత అలంకరణ    |
| `/=`    | భాగించ తర్వాత అలంకరణ  |
| `%=`    | మిగిలిన తర్వాత అలంకరణ |

## యూనరిఅరి / పాయింటర్ / క్యాస్ట్

| ఆపరేటర్/తాళం | వివరణ                |
| ------------ | -------------------- |
| `++`, `--`   | అంతర/తరువాత పెరుగుదల |
| `&x`         | యొక్క అడ్రస్ పొందడం  |
| `deref p`    | పాయింటర్ రెఫరెన్స్   |
| `expr as T`  | స్పష్టమైన క్యాస్ట్   |

## పాయింటర్ ఆపరేషన్

| వ్యాఖ్యయం                  | ఫలితం                                       |
| -------------------------- | ------------------------------------------- |
| `ptr<T> + int`             | `ptr<T>` (GEP మార్చడం)   |
| `int + ptr<T>`             | `ptr<T>` (GEP మార్చడం)   |
| `ptr<T> - int`             | `ptr<T>` (GEP నడిపించడం) |
| `ptr<T> - ptr<T>`          | `i64` (బిట్ వ్యత్యాసం)   |
| `ptr == ptr`, `ptr != ptr` | పాయింటర్ సరిపోల్చడం                         |

## నిర్ధారించబడని లేదా అమలు చేయబడని అంశం

వ్యాఖ్య టోకెన్ ఉంది కానీ ప్రస్తుత ఎక్స్ ప్రెషన్ ఆపరేషన్‌తో మద్దతు లేదు.
ఉదా: `??`, `?:`, `in`, `is`, `!&`, `!|`, `~^`.
