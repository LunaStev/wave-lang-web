# ఆపరేటర్

ఈ పత్రం ప్రస్తుత కంపైలర్ ఆధారంగా అందుబాటులో ఉన్న వాస్తవ ఆపరేటర్‌లను నిర్వహిస్తుంది.

## అంకగణితం

| ఆపరేటర్ | వివరణ |
| --- | --- |
| `+` | అదనంగా |
| `-` | తీసివేత |
| `*` | గుణకారం |
| `/` | విభజన |
| `%` | విశ్రాంతి |

## సరిపోల్చండి

| ఆపరేటర్ | వివరణ |
| --- | --- |
| `==` | సమానం |
| `!=` | విభిన్న |
| `<` | చిన్న |
| `<=` | కంటే తక్కువ లేదా సమానం |
| `>` | పెద్ద |
| `>=` | కంటే ఎక్కువ లేదా సమానం |

## తర్కం

| ఆపరేటర్ | వివరణ |
| --- | --- |
| `&&` | తార్కిక మరియు |
| `\|\|` | తార్కిక OR |
| `!` | లాజిక్ కాదు |

## బిట్

| ఆపరేటర్ | వివరణ |
| --- | --- |
| `&` | బిట్‌వైస్ మరియు |
| `\|` | బిట్ లేదా |
| `^` | బిట్‌వైస్ XOR |
| `~` | బిట్ కాదు |
| `<<` | ఎడమవైపు మారండి |
| `>>` | కుడి షిఫ్ట్ |

## కళాశాల అడ్మిషన్లు

| ఆపరేటర్ | వివరణ |
| --- | --- |
| `=` | ప్రాథమిక కేటాయింపు |
| `+=` | చేరిక తర్వాత అప్పగింత |
| `-=` | తీసివేత తర్వాత కేటాయింపు |
| `*=` | గుణకారం తర్వాత కేటాయింపు |
| `/=` | విభజన తర్వాత ప్రత్యామ్నాయం |
| `%=` | మిగిలిన తర్వాత అసైన్‌మెంట్ |

## unary/పాయింటర్/తారాగణం

| ఆపరేటర్/కీవర్డ్ | వివరణ |
| --- | --- |
| `++`, `--` | ముందు/వెనుక పెరుగుదల/తగ్గడం |
| `&x` | చిరునామా పొందండి |
| `deref p` | పాయింటర్ dereference |
| `expr as T` | స్పష్టమైన తారాగణం |

## పాయింటర్ అంకగణితం

| వ్యక్తీకరణ | ఫలితాలు |
| --- | --- |
| `ptr<T> + int` | `ptr<T>` (GEP ఉద్యమం) |
| `int + ptr<T>` | `ptr<T>` (GEP ఉద్యమం) |
| `ptr<T> - int` | `ptr<T>` (GEP ఉద్యమం) |
| `ptr<T> - ptr<T>` | `i64` (బైట్ తేడా) |
| `ptr == ptr`, `ptr != ptr` | పాయింటర్ పోలిక |

## రిజర్వ్ చేయబడిన లేదా అమలు చేయని అంశాలు

వ్యాకరణ టోకెన్లు ఉన్నాయి, కానీ కొన్ని ప్రస్తుతం వ్యక్తీకరణ కార్యకలాపాల ద్వారా మద్దతు ఇవ్వవు.
ఉదాహరణలు: `??`, `?:`, `in`, `is`, `!&`, `!|`, `~^`.
