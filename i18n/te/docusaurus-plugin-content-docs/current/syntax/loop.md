# పునరావృత ముసుపు

## ఏర్పాటుగా

Wave భాషలో ఉండేదులను అనేక క్షేత్రాలను నిర్వహించడం మిశ్ర సం మంచి సేవలు ఈ స్మృతి.
పునరావృత సందేశం రిజగటం మాస్తఁ నిగ్గోడల సముక్షణం అనేదేమి మారవద్దే స్మృతి.

నేడక్షణం మధ్యలో ప్రయోజనాలను పొందడానికైన నివేదించడమంటాను
Wave షరతుల ప్రాతిపదిక పునరావృతాన్ని మరియు పునరావృత సంఖ్యలను మద్దతుగా ఉంది మరియు క్రమంలో నడుపడానికి పదాలను అందిస్తుంది.

ఈ విభాగంలో, మేము `while` మరియు `for` స్టేట్‌మెంట్‌ల ఉపయోగాన్ని మరియు పునరావృతం పథాన్ని నియంత్రించడానికి `break` మరియు `continue` పదాలను వివరించాము.

---

## while సందేశం

`while` స్టేట్మెంట్ నిబంధన సత్యం గా పరిగణించబడుతున్నప్పుడు కోడ్ బ్లాక్‌ను పునరావృతం చేస్తుంది.
సందర్భం తప్పు‌గా ఐతే, పునరావృతి తక్షణముగా ముగియవచ్చు.

ఈ విధానంలో పునరావృతి సంఖ్య స్పష్టంగా తెలియనిప్పుడు మరియు ప్రత్యేక సూచన పొందినప్పటికీ తిరగాలి.

### ప్రాథమిక నిర్మాణం

Waveలో while స్టేట్‌మెంట్ యొక్క ప్రాథమిక నిర్మాణం ఇది.

```wave
while (నిబంధన) {
    // పునరావృతం చేయాల్సిన కోడ్
}
```

నిబంధన తప్పనిసరిగా `bool` టైపు గా పరిశీలించాలి మరియు `{}` లో దిగువ చెప్పిన కోడ్ బ్లాక్‌లో ఒక పునః సూచనలు లో పెట్టాల్సి ఉంది.

### ఉదాహరణ: 0 నుంచి 4 వరకు ముద్రించు

```wave
var i :i32 = 0;

while (i < 5) {
    println("i {} ఉంది.", i);
    i = i + 1;
}
```

ఈ ఉదాహరణలో, `i` విలువ 5కి కంటే తక్కువ ఉన్నంత సమయాన రిపీటెడ్ అవుతుంది.
ప్రతి పదం క్రమంగా ప్రస్తుతం విలువను అచ్చుగా ముద్రిస్తుంది మరియు `i` విలువను 1తో పెంచి ప్రత్యేక నియమానికి సాకారం.

---

## for సందేశం

`for` స్టేట్మెంట్ తిరోగతి సంఖ్యలు గమనించినప్పుడు ఉపయోగించడం పైగా చక్కగా ఉంటుంది.
ప్రారంభ విలువను, నియమాన్ని, వృద్ధిని ఒక్కటిగా నిర్వచించి పునరావృతం యొక్క నడవలను స్పష్టంగా తెలియచేస్తున్నాయి.

పునరావృతం వెంట ఉన్న పదాలు ఒకచోట ఉంచేయగాన  ఇది పేర్కొనడం సులభం.

### ప్రాథమిక నిర్మాణం

```wave
for (ప్రారంభం; నగరబోరి; వెంట్) {
    // పదం రాసిన పదం
}
```

Wave యొక్క `for` ప్రారంభం చాలా రూపాల అనుకూలించుకుంటుంది.

- సోఊర్స్థ సమాచారాన్ని చిట్టి విధంగా పొందాలి.
- `var` / `let mut` / `const` ప్రకటించే ప్రాథమిక ఆదేశం.
- ఒక ప్రచార పద్ధతి పునరావృతం.

### ఉదాహరణ 1: పుల్కితం నిర్వహణ

```wave
for (i :i32 = 1; i <= 5; i += 1) {
    println("i = {}", i);
}
```

### ఉదాహరణ 2: `var` / `let mut` ప్రారంభం

```wave
for (var i: i32 = 0; i < 3; i += 1) {
    println("var i = {}", i);
}

for (let mut j: i32 = 0; j < 3; j += 1) {
    println("let mut j = {}", j);
}
```

### ఉదాహరణ 3: వ్యక్తీయ ఆధారిత ప్రారంభం (పాత వాటిని పునఃఉపయోగించడం)

```wave
var i: i32 = 99;

for (i = 3; i <= 5; i += 1) {
    println("i = {}", i);
}

println("after loop: {}", i); // 6
```

ప్రకటన ఆధారిత ప్రారంభం (`var`, `let mut`, `i :i32 = ...`) గు౹ యంతులి ప్రస్తుత యథార్ధంలో కూర్చిఉంటే, వ్యక్తీకి నిహితమయ్యాయి.

## అంతరిత్మ నావలక్కోడి

పునరావృతి మరొక పునరావృతంలో ఉంచడం విప్రత్తి తర్వాత, దీనిని అంతరిత్మ పునరావ్ర్తం అని పిలవబడింది.
అంతరిత్మ పునరావృతం 2-డ పరికరం స్థితిని తిరగడం లేదా, ప్రక్షాళన పరిమాణాలలోను ఉపయోగించగలదు.

### ఉదాహరణ: 2 స్థాయిల వైలబెల్ల

```wave
var i :i32 = 0;

while (i < 3) {
    var j :i32 = 0;

    while (j < 2) {
        println("i={}, j={}", i, j);
        j = j + 1;
    }

    i = i + 1;
}
```

ఈ ఉదాహరణలో, మధ్యంతర `while` స్టేట్మెంట్ కూడా ఆందోళన చేరకుండా విధానం ఉంటుంది.
దీని ద్వారా (`i`, `j`) కాంతిని తిన్నట్టు క్రమంమా ప్రక్రియలో ఉంచడాన్ని పెంచెలిస్తారు.

## బ్రేక్ స్టేట్మెంట్

`break` స్టేట్మెంట్ తిరుగులను ఇప్పుడు ముగించసాగుతాయి.
పునరావృతాలపై మరో అనుభవాన్ని ప్రయోజనవిషేలు చేయకపోతే వాడుతారు.

### ఉదాహరణ: నిర్దిష్ట విలువపై పునరావృతాన్ని ముగించడం

```wave
var i :i32 = 0;

while (true) {
    if (i == 5) {
        break;
    }

    println(i);
    i = i + 1;
}
```

ఈ ఉదాహరణలో, సంరక్షణించిన పునరావృతంలో `i` 5ని ఎదుర్కొత్తే `break` అమలవుతుంది.
అంచనా స్థాయిలో అంచనా ప్రకృతి పునరావృతంలో `break` ప్రక్రియకు అనుకూలిస్తుంది.

---

## కంటిన్యూ స్టేట్మెంట్

`continue` స్టేట్మెంట్ తప్పనసరిగా ఇంటిపేరు కింద.Expressions నీడ పాల్గోనడం మనకూ హార్మెలగగల హోల్ దర్ఖాన్ తీసుకుంటాయి.
అనవసర సూత్రంకి చేర్పుగా చానిడాకు ఉపయోగిస్తారు.

### ఉదాహరణ: కేవలం సమ సంఖ్యలను అచ్చు చేయండి

```wave
for (var i: i32 = 0; i <= 10; i = i + 1) {
    if (i % 2 == 1) {
        continue;
    }

    println(i);
}
```

ఈ కోడ్‌లో `i` పద్ధతి నుండి ప్రభావం పెరిగితే, `continue` అమలవండి, కాలాన్ని ఒక దాట చేస్తుంది.
ఇ మునుపటి సంఖ్య మాత్రమే చక్కబించబడుతుంది.

---

## సంక్షేపం

Wave పునరావృతం సరువులుగా, పునరావృత పరిమాణాల రూపంలో గతాలు కల్పించారు.
`while` స్టేట్మెంట్ కరారాధిని వాడుతుంది, `for` స్టేట్మెంట్ విశ్వాసానికి అనుకూలించుకుంటుంది.

`break` మరియు `continue` కలిగి ఉండటం ద్వారా, పునరావృత సమయంలో కోడ్‌ను ప్రాథమికరమైన సమాచారంలో హార్మెలిచే దులే పొంచే అనుభవాన్ని కోల్పోతాయి.
