# డేటా టైప్స్

ఈ ప్రాథమిక పత్రం Wave ప్రోగ్రామింగ్ భాష అందించే వివిధ డేటా రకాల గురించి వివరిస్తుంది.
Wave లో ఎన్నో రకాల డేటా రకాల ద్వారా విలువలను నిల్వ చేయటం మరియు గణనలు చేసుకోవటం సాధ్యం, ప్రతి డేటా రకం దాని డేటా యొక్క ప్రదర్శన పద్ధతి మరియు మెమరీ ప్రాసెసింగ్ పద్ధతిని స్పష్టంగా నిర్వచిస్తుంది.

డేటా రకాన్ని స్పష్టంగా పేర్కొనడం Wave యొక్క ప్రధాన రూపకల్పన తత్వాలకు ఒకటి.
Wave **శృతిలో శక్తి ఉన్న** వ్యవస్థ. అన్ని `var`/`fun` ప్రకటడం మరియు విభజనలు ప్రత్యేక రకం అవసరం మరియు సందర్భం ఆధారిత రాయినట్లుగా చెందదు. కాబట్టి `var x = 1` వంటి రకం లేకపోతే కంపైలింగ్ విఫలమవుతుంది.
దీనితో కోడ్ యొక్క ఉద్దేశ్యాన్ని స్పష్టంగా వ్యక్తపరచవచ్చు, కంపైలింగ్ సమయంలో పొరపాట్లను మాత్రమే కనుగొనవచ్చు మరియు సమర్థవంతమైన మెమరీ వినియోగాన్ని మరియు స్థిరమైన అమలును నిర్ధారించవచ్చు.

## పూర్తి సంఖ్య రకం

పూర్తి సంఖ్య రకం పూర్తి సంఖ్య విలువలను నిల్వ చేయటానికి ఉపయోగిస్తారు.
Waveలో పరామర్శించే `i32` (సంతి ఉన్న 32-బిట్ పూర్తి సంఖ్య) మరియు `u32` (సంతి లేని 32-బిట్ పూర్తి సంఖ్య) మొదట అవలంబించబడతాయి కానీ అవసరానికి అనుగుణంగా పూర్తి సంఖ్య యొక్క బిట్ పరిమాణాన్ని చాలా జాగ్రత్తగా పేర్కొనవచ్చు.

సంతి ఉన్న పూర్తి సంఖ్య రకం `i8` నుండి `i1024` వరకు అందుబాటులో ఉండి, సంతి లేని పూర్తి సంఖ్య రకం `u8` నుండి `u1024` వరకు ఉపయోగించవచ్చు.
దీంతో సరళమైన ప్రయోగాల నుండి పెద్ద శ్రేణి సంఖ్యల గణనలు, కృత్రిమ లెక్కలు, తక్కువ స్థాయి సిస్టమ్ ప్రోగ్రామింగు వరకు సంభందిత అవసరాలను తీర్చవచ్చు.

చివరి పూర్తించిన సంఖ్యను ఉపయోగించే సరళమైన ఉదాహరణ.

```wave
var a: i32 = 100;
var b: u32 = 200;
```

## తేలికపాటి పాయింట్ రకం

తేలికపాటి పాయింట్ రకం నిజమైన విలువలను నిల్వ చేయటానికి ఉపయోగిస్తారు.
Wave లో సాధారణంగా ఉపయోగించే తేలికపాటి పాయింట్ రకం `f32` కానీ బాగా స్పష్టమైన భాగం అవసరమైతే పెద్ద పరిమాణు రకాలను ఎంపిక చేసుకోవచ్చు.

Wave `f32` నుండి `f128` వరకు తేలికపాటి పాయింట్ రకాలు అందించి సంకీర్ణ ప్రతిదాలను మరియు పనితీరును వినియోగదారుల కీ నిండుగా ఎంపిక చేసేటట్లు తయారుచేస్తోంది.
దీని ద్వారా సాధారణా గణనలు మొదలైన ప్రాథమిక గణనలు మరియు శాస్త్రీయ గణనల యుక్తములు ఉన్నాయ్.

కింద తేలికపాటి పాయింట్ రకం ఉపయోగించిన ఉదాహరణ.

```wave
var pi: f32 = 3.14;
var e: f64 = 2.71828;
```

## అక్షరాల రకం

అక్షరాల రకం పాఠ్య డేటాను నిర్వహించటం కొరకు ఉపయోగిస్తారు.
Wave లో మీరు `str` పదాన్ని ఉపయోగించి అక్షరాలను ప్రకటించవచ్చు, అక్షరాల లిటరల్ ముక్కలు మహా సాధారణం " లో భాదితమ్.

అక్షరాలు సంఖ్యను సూచ్యం, వినియోగదారు ఇచ్చుకో వేళ, పాఠ్య ఆధారిత అంశం ఉత్పత్తి వగైరాల చర్యలో విపరీతంగా వాదించవచ్చు.

చివరి అక్షరాల రకాన్ని ఉపయోగించే ప్రాథమిక ఉదాహరణ.

```wave
var text: str = "Hello Wave";
```

## బూలియన్ రకం

బూలియన్ రకం నిజం (True) లేదా అబద్ధం (False) విలువలను ప్రకటించు డేటా రకం.
Wave లో `bool` రకం ఉపయోగించి, విలువలు `true` లేదా `false` గా అవిశ్రాంతిస్తారు.

బూలియన్ రకం క్రమ సహాయం మరియు మన్నవ పనితీరును ప్రశంసించటానికి అవసరాన్ని నిర్వహించా.

```wave
var isActive: bool = true;
var isAvailable: bool = true;
```

## అక్షర రకం

అక్షర రకం ఒకే ఒక్క అక్షరాన్ని నిల్వ చేయటానికి ఉపయోగిస్తారు.
`char` పదాన్ని ఉపయోగించి ప్రకటించిన ప్రాథమిక ఒక అక్షరాన్ని పంపించగలరు.

అక్షర లిటరల్ చిన్న నావిక చేయలా `  '  `.

```wave
var letter: char = 'A';
```

## బైట్ రకం

బైట్ రకం 1 బైట్ పరిమాణం లో ఉండే డేటాను నిల్వ కంటే ఉపయోగిస్తారు.
ఈ రకం ప్రధానంగా బైనరీ డేటాను గణించటం, ఫైల్ మాట్లాడటం, నెట్ వర్క్ ప్రోగ్రామింగ్ లాంటి తక్కువ అల్లువాత వర్తించేది.

Wave లో `byte` పదాన్ని ఉపయోగించడం ద్వారా బైట్ రకం ప్రకటిస్తారు.

```wave
var byteData: byte = 0xFF;
```

## పాయింటర్ రకం

పాయింటర్ రకం మెమరీ చిరునామాలను నేరుగా కూర్చవటానికి ఉపయోగిస్తారు.
Wave లో  `ptr<T>` రూపంలో పాయింటర్ రకం ప్రకటిస్తారు.

పాయింటర్ తక్కువ స్థాయి మెమరీ up క్రొత్త లేక ప్రకారం, వ్యవస్థ ఫంక్షన్లు లేదా అయితే ఏదో వేచ్చడం అవసరమైన కోడ్ గా వాడటం.

```wave
var ptr: ptr<T> = &someVariable;
```

## `null` లిటరల్

Wave లో `null` అందంగా లిటరల్.

- `null` ఒక గుర్తింపుకారిక కాదు. (`var null = ...` ఫార్మాట్ అనుమతించబడదు)
- `null` ని కేవలం `ptr<T>` రకాలలో మాత్రమే ఇస్తుంది.

```wave
var p: ptr<i32> = null;  // సమర్ధన

// var n: i32 = null;    // తప్పు
// var b: bool = null;   // తప్పు
```

## అనుక్రమిక రకం

అనుక్రమిక రకం అనేది సమానమైన డేటా రకం యొక్క అనేక విలువలను వరుసగా నిల్వ చేయడానికి ఉపయోగించబడుతుంది.
Wave లో `array<రకం, పరిమాణం>` రూపంలో ఫలకాలను ప్రకటిస్తాయి మరియు ఫలకాల పరిమాణాన్ని కంపైల్ సమయమునకు ఖాతా గా నిర్ణయిస్తారు.

ఇది మెమరీ నిర్మాణాన్ని స్పష్టంగా చేసేందుకు మరియు స్థిరమైన చేరువ అందుబాటులో ఉంటుంది.

```wave
var numbers: array<i32, 5> = [1, 2, 3, 4, 5];
```

ప్రతి డేటా రకం అనువర్తనాలు మరియు లక్షణాలకు తగిన పరిధులు మరియు పరిమాణాలను ఎంపిక చేసేందుకు రూపొందించబడింది.
సరైన డేటా రకం ఎంపికచేస్తే మెమరీని సమర్థంగా నిర్వహించవచ్చు మరియు కోడ్ యొక్క స్థిరత్వం మరియు చదవగలిగే మాటను కూడా పెంచుతుంది.
