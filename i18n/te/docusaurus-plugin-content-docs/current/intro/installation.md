# ఇన్‌స్టాల్ చేయండి

## స్తాపన విధానం

Wave అందించే ఇన్స్టాలేషన్ స్క్రిప్ట్ ద్వారా సులభంగా ఇన్‌స్టాల్ చేయవచ్చు.
టెర్మినల్‌లో దిగువ శ్రీను అమలు చేస్తే, నిర్దేశిత సంస్కరణ యొక్క Wave కంపైలర్ (`wavec`) ఆటోమేటిక్‌గా ఇన్‌స్టాల్ అవుతుంది.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --వెర్షన్ <వెర్షన్>
```

ఇన్స్టాలేషన్ స్క్రిప్ట్, సిస్టమ్ ఎన్విరాన్మెంట్‌ని తనిఖీ చేసిన తరువాత, Wave నిర్వహణకు అవసరమైన ఆధారాలను మరియు కంపైలర్‌ను ఆటోమేటిక్‌గా ఆదేశిస్తుంది.
ఒక సంస్కరణను పేర్కొనకపోతే, అది తాజా స్థిరమైన సంస్కరణ లేదా నిర్ధేశించిన ప్రమాణం ప్రకారం కస్టమ్ సంస్కరణను ఇన్‌స్టాల్ చేయబడుతుంది.

## ఇన్స్టాలేషన్ ఉదాహరణ.

తాజా సంస్కరణను ఇన్స్టాల్ చేసేందుకు దిగువగా అమలు చేయండి.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- తాజాది
```

కొత్త సంస్కరణని ద్రుష్తిలో ఉంచుకోవాలంటే, `--version` ఆప్షన్ ఉపయోగించండి.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --వెర్షన్ v0.1.3-pre-beta
```

నైట్‌లీ బిల్డ్ వంటి మరింత విపులమైన సంస్కరించాలను స్పష్టంగా ప్రదర్శించడం కూడా సాధ్యం.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --వెర్షన్ v0.1.3-pre-beta-nightly-2025-07-11
```

## ఇన్‌స్టాలేషన్ సమయంలో జరిగిన పనులు

ఇన్స్టాలేషన్ స్క్రిప్ట్, Wave ని సమర్థవంతంగా నిర్వహించేందుకు అనేక దశలను ఆటోమేట్ చేస్తుంది.
ప్ర మొదటగా LLVM 14 కు సంబంధించిన ఐతే అవసరమైన ప్యాకేజీలను `apt-get` ద్వారా ఇన్స్టాల్ చేయండి.
తర్వాత నిఖార్సైన పద్ధతుల‌ను దృష్టిలో ఉంచుకుని, సిస్టమ్‌లో LLVM ను నిర్ధారించడానికి `/usr/lib/libllvm-14.so` కోసం సింబాలిక్ లింక్ సృష్టించండి.

Wave కంపైలర్ LLVMని ఖచ్చితంగా కనుగొనాలంటే, `LLVM_SYS_140_PREFIX` మధ్యం క్రియ పరిచ యం చేయండి.
ఈ క్రియా తరువాత, `~/.bashrc` లో చేర్చి, తదుపరి టెర్మినల్ అవధి లో కూడా కొనసాగుతుంది.

తరువాత, వినియోగదారు నిర్దేశించిన వెర్షన్ యొక్క Wave ప్యాకేజీ (`.tar.gz`) ని డౌన్‌లోడ్ చేసి విపరీతం చేయండి.
దురోహం అయినా, `wavec` ఎగ్జిక్యూటబుల్ ఫైలు `/usr/local/bin` లో ఇన్స్టాల్ చేయబడుతుంది.
సిస్టమ్‌లో ఎక్కడ అయినా `wavec` కమాండ్‌ని వాడుకోవచ్చు.

ఇన్స్టాలేషన్ పూర్తయితే, `wavec --version` ఆదేశం ద్వారా సమర్థంగా ఇన్స్టాల్ చేసింది అనా పైకి చూపించబడుతుంది.

## ఇన్‌స్టలేషన్ ని ధృవీకరించండి

ఇన్స్టాలేషన్ తర్వాత, క్రింద ఇచ్చిన ఆదేశాలతో Wave కంపైలర్ స్థిరంగా జరిగిందాని చెక్ చేయవచ్చు.

```bash
wavec --వెర్షన్
```

కమాండ్ అమలులో ఇన్స్టాల్ అయిన Wave నిర్మాణ సమాచారం ప్రింట్ అయినప్పుడు, ఇది సంపూర్ణంగా ఇన్స్టాల్ గా ఉంటుంది.

## వేవ్ తొలగింపు మార్గదర్శిని (`uninstall.sh`)

Wave ని మీ సిస్టమ్ నుండి తొలగించడం ఇష్టమైతే, ఇచ్చిన అన్ ఇన్‌స్టాల్ స్క్రిప్ట్ ఉపయోగించవచ్చు.
ఈ స్క్రిప్ట్ ఇన్స్టాలేషన్ సమయంలో జోడించిన ఫైల్ మరియు సెటప్ ని క్లీన్ చేయటం.

### తొలగింపు విధానం

టెర్మినల్ లో క్రింద ఇవ్వబడిన ఆదేశం అమలుచేయండి.

```bash
curl -fsSL https://wave-lang.dev/uninstall.sh | bash
```

తొలగింపు పూర్తి అయినట్లయితే, wavec ఆదేశం ఉపయోగించబడదు మరియు
Wave తో సంబంధిత ఎగ్జిక్యూటబుల్ మరియు సెటింగ్ లు సిస్టమ్ నుండి తొలగించబడుతున్నది.
