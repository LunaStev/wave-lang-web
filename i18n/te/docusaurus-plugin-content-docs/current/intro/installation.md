# సంస్థాపన

## సంస్థాపన విధానం

అందించిన ఇన్‌స్టాలేషన్ స్క్రిప్ట్ ద్వారా Waveని ఇన్‌స్టాల్ చేయవచ్చు.
మీరు టెర్మినల్‌లో దిగువ ఆదేశాన్ని అమలు చేస్తే, Wave కంపైలర్ (`wavec`) యొక్క పేర్కొన్న సంస్కరణ స్వయంచాలకంగా ఇన్‌స్టాల్ చేయబడుతుంది.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version <version>
```

ఇన్‌స్టాలేషన్ స్క్రిప్ట్ సిస్టమ్ ఎన్విరాన్‌మెంట్‌ను తనిఖీ చేస్తుంది మరియు Waveని అమలు చేయడానికి అవసరమైన డిపెండెన్సీలు మరియు కంపైలర్‌ను స్వయంచాలకంగా సెట్ చేస్తుంది.
మీరు సంస్కరణను పేర్కొనకుంటే, పేర్కొన్న ప్రమాణాల ప్రకారం తాజా స్థిరమైన సంస్కరణ లేదా డిఫాల్ట్ వెర్షన్ ఇన్‌స్టాల్ చేయబడుతుంది.

## సంస్థాపన ఉదాహరణ

తాజా సంస్కరణను ఇన్‌స్టాల్ చేయడానికి, అమలు చేయండి:

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- latest
```

మీరు నిర్దిష్ట సంస్కరణను ఇన్‌స్టాల్ చేయాలనుకుంటే, `--version` ఎంపికను ఉపయోగించండి.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version v0.1.3-pre-beta
```

రాత్రిపూట బిల్డ్‌ల వంటి మరింత వివరణాత్మక సంస్కరణలను పేర్కొనడం కూడా సాధ్యమే.

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version v0.1.3-pre-beta-nightly-2025-07-11
```

## సంస్థాపన సమయంలో ఏమి జరుగుతుంది

ఇన్‌స్టాలేషన్ స్క్రిప్ట్ స్వయంచాలకంగా Waveని సాధారణంగా అమలు చేయడానికి అనేక దశలను నిర్వహిస్తుంది.
ముందుగా, `apt-get` ద్వారా LLVM 14కి సంబంధించిన అవసరమైన ప్యాకేజీలను ఇన్‌స్టాల్ చేయండి.
`/usr/lib/libllvm-14.so`కి సింబాలిక్ లింక్‌ను సృష్టించండి, తద్వారా భవిష్యత్ సిస్టమ్‌లు LLVMని విశ్వసనీయంగా సూచించగలవు.

Wave `LLVM_SYS_140_PREFIX` ఎన్విరాన్మెంట్ వేరియబుల్‌ను సెట్ చేస్తుంది, తద్వారా కంపైలర్ LLVMని సరిగ్గా కనుగొనగలదు.
ఈ సెట్టింగ్ `~/.bashrc`కి జోడించబడింది మరియు భవిష్యత్ టెర్మినల్ సెషన్‌లలో కొనసాగుతుంది.

తర్వాత, Wave ప్యాకేజీ (`.tar.gz`) యొక్క వినియోగదారు పేర్కొన్న సంస్కరణను డౌన్‌లోడ్ చేసి, అన్జిప్ చేయండి.
అన్జిప్ చేసిన తర్వాత, `wavec` ఎక్జిక్యూటబుల్ ఫైల్‌ను `/usr/local/bin`లోకి ఇన్‌స్టాల్ చేయండి.
`wavec` ఆదేశాన్ని సిస్టమ్‌లో ఎక్కడైనా అందుబాటులో ఉండేలా కాన్ఫిగర్ చేయండి.

ఇన్‌స్టాలేషన్ పూర్తయిన తర్వాత, `wavec --version` ఆదేశాన్ని ఉపయోగించి ఇది సరిగ్గా ఇన్‌స్టాల్ చేయబడిందో లేదో తనిఖీ చేయండి.

## సంస్థాపనను తనిఖీ చేయండి

ఇన్‌స్టాలేషన్ పూర్తయిన తర్వాత, దిగువ ఆదేశాన్ని అమలు చేయడం ద్వారా మీరు Wave కంపైలర్ సరిగ్గా ఇన్‌స్టాల్ చేయబడిందో లేదో తనిఖీ చేయవచ్చు.

```bash
wavec --version
```

ఆదేశాన్ని అమలు చేస్తున్నప్పుడు ఇన్‌స్టాల్ చేయబడిన Wave యొక్క సంస్కరణ సమాచారం ప్రదర్శించబడితే, అది సాధారణంగా ఇన్‌స్టాల్ చేయబడుతుంది.

## Wave రిమూవల్ గైడ్ (`uninstall.sh`)

మీరు మీ సిస్టమ్ నుండి Waveని తీసివేయాలనుకుంటే, మీరు అందించిన తీసివేత స్క్రిప్ట్‌ని ఉపయోగించవచ్చు.
ఇన్‌స్టాలేషన్ ప్రాసెస్ సమయంలో జోడించిన ఫైల్‌లు మరియు సెట్టింగ్‌లను శుభ్రపరచడానికి ఈ స్క్రిప్ట్ బాధ్యత వహిస్తుంది.

### ఎలా తొలగించాలి

టెర్మినల్‌లో కింది ఆదేశాన్ని అమలు చేయండి:

```bash
curl -fsSL https://wave-lang.dev/uninstall.sh | bash
```

తీసివేత పూర్తయిన తర్వాత, wavec ఆదేశం ఇకపై ఉపయోగించబడదు.
Waveతో అనుబంధించబడిన ఎక్జిక్యూటబుల్ ఫైల్‌లు మరియు సెట్టింగ్‌లు మీ సిస్టమ్ నుండి తొలగించబడతాయి.
