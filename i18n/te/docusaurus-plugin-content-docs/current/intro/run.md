# మొదటి ప్రోగ్రామ్ ను అమలు చేయండి

మునుపటి సృష్టించిన వస్తువు లో ఉన్నవేవ్ ఇప్పటికే మీకు తీసుకున్నట్లయితే, ఇక్కడ మొదటి వేవ్ ప్రోగ్రామ్‌ను నేరుగా అమలు చేయండి.
ఈ విభాగంలో సులభంగా ఉదాహరణల ద్వారా వేవ్ ప్రోగ్రామ్‌ను రాసి, అమలు చేసే సమగ్ర చర్యలను స్థితిలో చూడవచ్చు.

## `hello.wave` ఫైలు సృష్టించు

ముందుగా పనిచేయడం డైరెక్టరీ నుండి `hello.wave` అనే పేరు తో కొత్త ఫైలు సృష్టించాలి.
ఫైలు పేరు మరియు విస్తరణకు విముక్తి ఇచ్చినదైనా, ప్రామాణికంగా `hello.wave` ను ఇక్కడ ఉదాహరణగా ఉపయోగిస్తారు.

## కోడ్ రాయడం

సృష్టించిన `hello.wave` ఫైలుకు క్రింద ఉన్న కోడ్ ను వ్రాయండి.

```wave
fun main() {
    println("హలో వేవ్");
}
```

ఈ కోడ్ లో `fun main()` అనేది వేవ్ ప్రోగ్రామ్ యొక్క అమలుకు ప్రారంభ ఆఫ్ స్టాప్.
వేవ్ ప్రోగ్రామ్ ఎప్పుడైనా `main` ఫంక్షన్ నుంచి అమలవుతుంది.

`println` ఫంక్షన్ ఒక చిత్రాన్ని సాధారణ అవతిరేకించే వ్యవహారంగా,
స్కీన్ పైన టెక్స్ట్ ప్రదర్శించడానికి ప్రాథమికమైనది.

## ప్రోగ్రామ్ అమలు చేయండి

కోడ్ రాసిన తరువాత, టెర్మినల్ తెరిచి, ఫైల్ ఉన్న డైరెక్టరీలో క్రింది ఆదేశాన్ని అమలు చేయండి.

```bash
wavec అమలు చెయ్యండి hello.wave
```

ఈ ఆదేశం వేవ్ సంకలనం సోర్స్ ఫైల్‌ని మల్చడానికి చూపించబడ్డది,
ఇప్పుడు వేగంగా అమలు చేయవడం.

## ఊపిరిండి తనిఖీ చేయండి

ప్రోగ్రామ్ సరిగ్గా అమలవైతే, టెర్మినల్ లో క్రింది ఔట్ పుట్ చూపబడును.

```
హలో వేవ్
```

ఈ ఔట్ పుట్ ముందు చూపబడితే, వేవ్ సరిగ్గా ఇన్స్టాల్ అయింది,
ప్రోగ్రామ్ నిర్మాణం మరియు అమలును కచ్చితంగా చేస్తుంది.

ఇప్పుడు మొట్టమొదటి వేవ్ ప్రోగ్రామ్‌ను విజయవంతంగా అమలు చేసింది.
ఇప్పుడూ వేవ్ యొక్క వ్యాకరణాన్ని మరియు ధోరణులను ఒకటి ఒకటిగా పరిశీలించి, మరింత క్రమం తనిఖీ చేయండి.

సూటిగా ఆదేశాలు ఎంపికా `(-O*`, `--debug-wave`, `--link`, `--dep-root`, `--dep`) విషయాలు
`wavec` CLI ప్రామాణికంలో పరిక్షించవచ్చు.
