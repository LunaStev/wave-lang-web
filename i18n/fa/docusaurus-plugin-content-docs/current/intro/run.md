# اجرای اولین برنامه شما

اگر قبلاً Wave را طبق راهنمای نصب نصب کرده‌اید، اکنون بیایید اولین برنامه خود را اجرا کنیم!

## ایجاد فایل `hello.wave`

ابتدا یک فایل جدید به نام `hello.wave` ایجاد کنید.

## نوشتن کد

کد زیر را در فایل `hello.wave` بنویسید:

```wave
fun main() {
    println("Hello Wave");
}
```

در اینجا `fun main()` نقطه شروع برنامه است و تابع `println` متن را روی صفحه نمایش می‌دهد.

## اجرای برنامه

اکنون برنامه Wave را اجرا می‌کنیم. ترمینال را باز کرده و دستور زیر را وارد کنید:

```bash
wavec run hello.wave
```

## بررسی خروجی

پس از اجرای برنامه، خروجی زیر را مشاهده خواهید کرد:

```
Hello Wave
```

اکنون می‌توانید تأیید کنید که Wave به‌درستی نصب شده و کار می‌کند. تبریک می‌گویم! شما با موفقیت اولین برنامه خود را اجرا کردید.
