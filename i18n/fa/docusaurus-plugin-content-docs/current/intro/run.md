# اجرا کردن اولین برنامه

اگر قبلاً Wave را با توجه به مستندات نصب کرده‌اید، حالا زمان آن رسیده که اولین برنامه Wave را خودتان اجرا کنید.
در این بخش، با یک مثال ساده کل فرآیند نوشتن و اجرای برنامه Wave را به صورت مرحله به مرحله بررسی می‌کنیم.

## ایجاد فایل `hello.wave`

ابتدا، یک فایل جدید به نام `hello.wave` در دایرکتوری کاری خود ایجاد کنید.
نام و پسوند فایل به شما بستگی دارد، اما در اینجا به عنوان مثال از `hello.wave` استفاده می‌کنیم.

## نوشتن کد

کد زیر را در فایل `hello.wave` که ایجاد کرده‌اید بنویسید.

```wave
fun main() {
    println("سلام ویو");
}
```

در این کد، `fun main()` به عنوان نقطه شروع اجرای برنامه Wave عمل می‌کند.
برنامه‌های Wave همیشه از تابع `main` شروع به اجرا می‌کنند.

تابع `println` یک رشته را به خروجی استاندارد چاپ می‌کند و به صورت پایه برای نمایش متن بر روی صفحه استفاده می‌شود.

## اجرای برنامه

پس از نوشتن کد، ترمینال را باز کرده و در دایرکتوری که فایل قرار دارد، دستور زیر را اجرا کنید.

```bash
wavec run hello.wave
```

این دستور به کامپایلر Wave می‌گوید که فایل منبع را کامپایل کرده و بلافاصله اجرا کند.

## بررسی خروجی

اگر برنامه به درستی اجرا شود، خروجی زیر بر روی ترمینال نشان داده خواهد شد.

```
سلام ویو
```

اگر این خروجی را مشاهده کردید، به این معنی است که Wave به درستی نصب شده و نوشتن و اجرای برنامه به صورت صحیح انجام شده است.

حال اولین برنامه Wave را با موفقیت اجرا کردید.
از این به بعد، می‌توانید به بررسی گرامر و قابلیت‌های Wave بپردازید و برنامه‌های پیچیده‌تری بنویسید.

گزینه‌های دقیق فرمان (`-O*`, `--debug-wave`, `--link`, `--dep-root`, `--dep`) را می‌توانید در اسناد CLI `wavec` بیابید.
