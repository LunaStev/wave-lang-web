# ऑपरेटर

यह दस्तावेज़ वर्तमान कंपाइलर के मानक पर आधारित वास्तव में उपयोग योग्य ऑपरेटरों को प्रस्तुत करता है।

## अंकगणित

| ऑपरेटर | विवरण                           |
| ------ | ------------------------------- |
| `+`    | जोड़                             |
| `-`    | घटाव                            |
| `*`    | गुणा                            |
| `/`    | विभाजन (भाग) |
| `%`    | बाकी                            |

## तुलना

| ऑपरेटर | विवरण        |
| ------ | ------------ |
| `==`   | समान         |
| `!=`   | असमान        |
| `<`    | छोटा         |
| `<=`   | छोटा या समान |
| `>`    | बड़ा         |
| `>=`   | बड़ा या समान |

## तर्कशक्ति

| ऑपरेटर     | विवरण     |
| ---------- | --------- |
| `&&`       | लॉजिक AND |
| \\\`\\ | लॉजिक OR  |
| `!`        | लॉजिक NOT |

## बिट

| ऑपरेटर     | विवरण      |
| ---------- | ---------- |
| `&`        | बिट एंड    |
| \\\`\\ | बिट OR     |
| `^`        | बिट XOR    |
| `~`        | बिट NOT    |
| `<<`       | बाएं शिफ्ट |
| `>>`       | दाएं शिफ्ट |

## आवंटन

| ऑपरेटर | विवरण               |
| ------ | ------------------- |
| `=`    | मूल आवंटन           |
| `+=`   | जोड़ के बाद आवंटन   |
| `-=`   | घटाव के बाद आवंटन   |
| `*=`   | गुणा के बाद आवंटन   |
| `/=`   | विभाजन के बाद आवंटन |
| `%=`   | शेष के बाद आवंटन    |

## यूनरी / पॉइंटर / कास्ट

| ऑपरेटर/कीवर्ड | विवरण              |
| ------------- | ------------------ |
| `++`, `--`    | पूर्व/पोस्ट वृद्धि |
| `&x`          | पता प्राप्ति       |
| `डीरिफ p`     | पॉइंटर डेरेफरेंस   |
| `expr as T`   | स्पष्ट कास्ट       |

## पॉइंटर ऑपरेशन

| अभिव्यक्ति                 | परिणाम                                  |
| -------------------------- | --------------------------------------- |
| `ptr<T> + int`             | `ptr<T>` (GEP संचलन) |
| `int + ptr<T>`             | `ptr<T>` (GEP संचलन) |
| `ptr<T> - int`             | `ptr<T>` (GEP संचलन) |
| `ptr<T> - ptr<T>`          | `i64` (बाइट अंतर)    |
| `ptr == ptr`, `ptr != ptr` | पॉइंटर तुलना                            |

## आरक्षित या बिना लागू की गई वस्तुएं

व्याकरण टोकन मौजूद हैं परंतु वर्तमान में अभिव्यक्ति संचालन द्वारा समर्थित नहीं हैं।
उदा: `??`, `?:`, `in`, `is`, `!&`, `!|`, `~^`.
