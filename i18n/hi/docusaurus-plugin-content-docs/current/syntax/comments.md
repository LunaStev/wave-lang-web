# टिप्पणी

Wave दो प्रकार के टिप्पणियों का समर्थन करता है।

- एक पंक्ति टिप्पणी: `//`
- ब्लॉक टिप्पणी: `/* ... */`

## एक पंक्ति टिप्पणी

`//` के पीछे का सामग्री पंक्ति के अंत तक अप्रयुक्त रहता है।

```wave
var x: i32 = 10; // पंक्ति टिप्पणी
x += 5;          // फिर भी कार्य करता है
```

## ब्लॉक टिप्पणी

`/*` और `*/` के बीच का सामग्री अप्रयुक्त रहता है।

```wave
var y: i32 = 1 /* इनलाइन ब्लॉक */ + 2;
```

ब्लॉक टिप्पणी कई पंक्तियाँ और नेस्टिंग का समर्थन करती है।

```wave
/* बाहरी
   /* आंतरिक */
   बाहरी समाप्ति
*/
```

## स्ट्रिंग और टिप्पणी संकेत

स्ट्रिंग के अंदर `/*`, `*/`, `//` को टिप्पणी शुरुआत/अंत के रूप में नहीं माना जाता है।

```wave
var marker: str = "/*//*/";
```

## टिप्पणी त्रुटि

यदि ब्लॉक टिप्पणी बंद नहीं होती है, तो कंपाइल त्रुटि (`E1002`) होती है।

```wave
/* बंद नहीं हुआ
```

कंपाइलर प्रारंभ स्थिति, कारण, सुधार संकेत के साथ आउटपुट करता है।
