# स्थापना

## स्थापना विधि

वेव को प्रदान की गई इंस्टॉलेशन स्क्रिप्ट के माध्यम से आसानी से स्थापित किया जा सकता है।
टर्मिनल में नीचे दिए गए कमांड को चलाने पर, निर्दिष्ट संस्करण का वेव कंपाइलर (`wavec`) स्वचालित रूप से स्थापित हो जाएगा।

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version <संस्करण>
```

इंस्टॉलेशन स्क्रिप्ट सिस्टम वातावरण की जाँच करने के बाद, वेव के संचालन के लिए आवश्यक निर्भरताओं और कंपाइलर को स्वचालित रूप से सेट करती है।
यदि संस्करण निर्दिष्ट नहीं किया गया है, तो नवीनतम स्थिर संस्करण या निर्दिष्ट मानदंड के अनुसार डिफ़ॉल्ट संस्करण स्थापित किया जाएगा।

## स्थापना का उदाहरण

नवीनतम संस्करण को स्थापित करने के लिए, निम्नलिखित तरीके से चलाएं।

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- latest
```

यदि आप विशेष संस्करण स्थापित करना चाहते हैं तो `--version` विकल्प का उपयोग करें।

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version v0.1.3-pre-beta
```

नाइटली बिल्ड जैसे और भी विशिष्ट संस्करण निर्दिष्ट करना संभव है।

```bash
curl -fsSL https://wave-lang.dev/install.sh | bash -s -- --version v0.1.3-pre-beta-nightly-2025-07-11
```

## स्थापना के दौरान किये जाने वाले कार्य

इंस्टॉलेशन स्क्रिप्ट वेव को सही तरीके से चलाने के लिए कई चरणों को स्वचालित रूप से प्रक्रिया करती है।
सबसे पहले LLVM 14 से संबंधित आवश्यक पैकेज को `apt-get` के माध्यम से स्थापित करें।
इसके बाद, सिस्टम में LLVM को स्थिर रूप से संदर्भित करने के लिए `/usr/lib/libllvm-14.so` का प्रतीकात्मक लिंक बनाएं।

वेव कंपाइलर को LLVM सही तरीके से खोजने के लिए `LLVM_SYS_140_PREFIX` एनवायरनमेंट वेरिएबल सेट करें,
और यह सेटिंग `~/.bashrc` में जोड़ दी जाएगी, जो आगे के टर्मिनल सेशन में भी बनी रहेगी।

इसके बाद उपयोगकर्ता द्वारा निर्दिष्ट संस्करण के वेव पैकेज (`.tar.gz`) को डाउनलोड करें और अनज़िप करें।
अनज़िप होने के बाद, `wavec` निष्पादन योग्य फाइल को `/usr/local/bin` में इंस्टॉल करें,
ताकि सिस्टम के किसी भी स्थान से `wavec` कमांड को प्रयोग किया जा सके।

इंस्टॉलेशन पूरा हो जाने पर, `wavec --version` कमांड के जरिए यह सुनिश्चित करें कि यह सही तरीके से स्थापित हो गया है।

## स्थापना की पुष्टि

इंस्टॉलेशन समाप्त होने के बाद, नीचे कुछ कमांड चलाकर वेव कंपाइलर का सही तरह से इंस्टॉल हो जाने की पुष्टि कर सकते हैं।

```bash
wavec --version
```

कमांड चलाने पर अगर स्थापित वेव का संस्करण जानकारी प्रदर्शित होता है, तो यह सही तरीके से स्थापित हो गया है।

## Wave हटाने की गाइड (`uninstall.sh`)

यदि आप Wave को सिस्टम से हटाना चाहते हैं, तो प्रदान की गई हटाने की स्क्रिप्ट का उपयोग कर सकते हैं।
यह स्क्रिप्ट इंस्टॉलेशन प्रक्रिया में जोड़े गए फाइलों और सेटिंग्स को साफ करने में मदद करती है।

### हटाने की विधि

टर्मिनल में निम्नलिखित कमांड चलाएं।

```bash
curl -fsSL https://wave-lang.dev/uninstall.sh | bash
```

हटाने की प्रक्रिया पूरी हो जाने पर, `wavec` कमांड अब और काम नहीं करेगा,
और Wave से संबंधित निष्पादन फाइल और सेटिंग्स को सिस्टम से हटा दिया जाएगा।
