# पहला कार्यक्रम चलाना

यदि आपने पहले के स्थापना दस्तावेज़ में Wave को पहले से ही इंस्टॉल किया है, तो अब पहला Wave प्रोग्राम सीधे चलाने की बारी है।
इस खंड में, हम एक सरल उदाहरण के माध्यम से Wave प्रोग्राम लिखते और चलाते हैं।

## `hello.wave` फाइल बनाना।

सबसे पहले, काम करने वाली निर्देशिका में `hello.wave` नाम की नई फाइल बनाएं।
फाइल का नाम और एक्सटेंशन स्वतंत्र रूप से चुना जा सकता है, लेकिन यहाँ उदाहरण के लिए `hello.wave` का उपयोग करते हैं।

## कोड लिखना

बनाई गई `hello.wave` फाइल में निम्नलिखित कोड लिखें।

```wave
fun main() {
    println("Hello Wave");
}
```

इस कोड में `fun main()` Wave प्रोग्राम की निष्पादन प्रारंभिक बिंदु को सूचित करता है।
Wave प्रोग्राम हमेशा `main` फ़ंक्शन से शुरू होता है।

`println` फ़ंक्शन एक ऐसा फ़ंक्शन है जो स्ट्रिंग को मानक आउटपुट पर प्रिंट करता है,
और स्क्रीन पर टेक्स्ट दिखाने के लिए सबसे मौलिक रूप से उपयोग किया जाता है।

## कार्यक्रम चलाना

यदि आपने कोड लिखना समाप्त कर लिया है, तो टर्मिनल खोलें और जिस निर्देशिका में फाइल स्थित है वहाँ निम्नलिखित कमांड चलाएं।

```bash
wavec run hello.wave
```

यह कमांड Wave कंपाइलर को स्रोत फाइल को संकलित करने और तुरंत निष्पादन करने का निर्देश देता है।

## आउटपुट सत्यापित करें

यदि प्रोग्राम ठीक से चलता है, तो टर्मिनल में निम्नलिखित आउटपुट दिखाई देगा।

```
Hello Wave
```

यदि आप इस आउटपुट को देखते हैं, तो इसका मतलब है कि Wave सही ढंग से स्थापित है,
और प्रोग्राम लेखन और निष्पादन सही तरीके से किया गया है।

अब आपने पहला Wave प्रोग्राम सफलतापूर्वक चलाया है।
अब से, आप Wave की सिंटैक्स और कार्यों की एक करके जांच कर सकते हैं, और अधिक जटिल प्रोग्राम लिख सकते हैं।

विशिष्ट कमांड विकल्प (`-O*`, `--debug-wave`, `--link`, `--dep-root`, `--dep`) को `wavec` CLI डॉक्यूमेंट में देखा जा सकता है।
