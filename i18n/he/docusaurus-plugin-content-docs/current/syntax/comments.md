# הערה

Wave תומכת בשני סוגי הערות.

- הערת שורה: `//`
- הערת בלוק: `/* ... */`

## הערת שורה

התוכן לאחר `//` מתעלם עד סוף השורה.

```wave
var x: i32 = 10; // הערת שורה
x += 5;          // עדיין עובד
```

## הערת בלוק

מתעלם מהתוכן בין `/*` ל-`*/`.

```wave
var y: i32 = 1 /* בלוק מובנה */ + 2;
```

הערת בלוק תומכת בריבוי שורות ובקינון.

```wave
/* חיצוני
   /* פנימי */
   סוף חיצוני
*/
```

## סימני מחרוזות והערות

בתוך המחרוזת `/*`, `*/`, `//` אינם מטופלים כהתחלה/סיום של הערה.

```wave
var marker: str = "/*//*/";
```

## שגיאת הערה

אם הערת בלוק אינה נסגרת, תתרחש שגיאת קומפילציה (`E1002`).

```wave
/* לא סגור
```

הקומפיילר מציג את מיקום ההתחלה, הגורם ורמזים לתיקון.
