# הדפסת שלום ווייב

מסמך זה מסביר כיצד לכתוב את תוכנית ההדפסה הבסיסית ביותר בשפת Wave.

---

## קוד לדוגמה

```wave
fun main() {
    println("שלום ווייב");
}
```

---

## תיאור

- `fun main()`

  זוהי פונקציית הכניסה לתוכנית Wave. נקרא ראשון בהפעלה.

- `println()`

  פונקציית הוצאה זמנית שמדפיסה מחרוזת ומוסיפה שורת חדשה (`\n`) לאחר מכן.

- `;` (נקודה פסיק)

  כל משפט ב-Wave מסתיים בנקודה פסיק.

---

## תוצאות הפעלה

```text
שלום ווייב
```

---

## דוגמה נוספת

Wave תומכת בהטמעת מחרוזות.

```wave
fun main() {
    var name: str = "Wave";
    println("שלום, {}!", name);
}
```

תצוגה:

```text
שלום, ווייב!
```
