# التعليقات

يدعم Wave نوعين من التعليقات.

- تعليق السطر الواحد: `//`
- تعليق كتلة: `/* ... */`

## تعليق السطر الواحد

يتم تجاهل المحتوى بعد `//` إلى نهاية السطر.

```wave
var x: i32 = 10; // تعليق سطر
x += 5;          // ما زال يعمل
```

## تعليق الكتلة

يتم تجاهل المحتوى بين `/*` و `*/`.

```wave
var y: i32 = 1 /* كتلة داخلية */ + 2؛
```

يدعم تعليق الكتلة تعدد الأسطر والتعشيش.

```wave
/* الخارجي
   /* الداخلي */
   نهاية الخارجي
*/
```

## سلاسل النصوص وعلامات التعليقات

الرموز `/*`، `*/`، `//` داخل السلاسل النصية لا تُعامل كبداية/نهاية تعليق.

```wave
var marker: str = "/*/*/"; عبارة عن سطر نصي يحتوي على نجمية وعلامة شاطئ.
```

## خطأ في التعليقات

إذا لم يتم إغلاق تعليق الكتلة، يحدث خطأ تجميع (`E1002`).

```wave
/* غير مغلق
```

يعرض المترجم موضع البداية، السبب، ونصائح الإصلاح معا.
