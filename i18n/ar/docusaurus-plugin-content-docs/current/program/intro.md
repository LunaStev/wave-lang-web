# إظهار Hello موجة

يشرح هذا المستند كيفية كتابة برنامج إخراج أساسي باستخدام لغة موجة (Wave).

---

## كود المثال

```wave
fun main() {
    println("Hello Wave");
}
```

---

## توضيح

- `fun main()`

  هي النقطة الرئيسية لدخول برنامج Wave. يتم الاتصال به أولًا عند التشغيل.

- `println()`

  يستخدم كدالة طباعة مؤقتة لإظهار سلسلة نصية، ويضيف فاصل سطر (`\n`) بعد الإظهار.

- `;` (فاصلة منقوطة)

  كل جملة في Wave تنتهي بفاصلة منقوطة.

---

## نتيجة التنفيذ

```text
Hello Wave
```

---

## مزيد من الأمثلة

يدعم Wave تدخيل السلاسل النصية.

```wave
fun main() {
    var name: str = "Wave";
    println("Hello, {}!", name);
}
```

الإظهار:

```text
Hello, Wave!
```

---

> يوضح هذا المثال كيفية استخدام الدالة الأساسية للإخراج في مكتبة Wave القياسية.
