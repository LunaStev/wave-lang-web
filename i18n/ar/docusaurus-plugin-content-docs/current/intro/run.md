# تشغيل أول برنامج

إذا كنت قد قمت بالفعل بتثبيت Wave من المستندات السابقة، فقد حان الوقت الآن لتشغيل أول برنامج Wave مباشرة.
في هذا القسم، سنلقي نظرة عن قرب على عملية كتابة وتنفيذ برنامج Wave بسيط خطوة بخطوة.

## إنشاء ملف `hello.wave`

أولاً، قم بإنشاء ملف جديد باسم `hello.wave` في الدليل الذي ستعمل فيه.
يمكنك تعيين اسم الملف والامتداد بشكل حر، ولكن هنا سنستخدم `hello.wave` كمثال.

## كتابة الكود

اكتب الشفرة أدناه في ملف `hello.wave` الذي أنشأته.

```wave
fun main() {
    println("مرحبًا ويف");
}
```

في هذه الشفرة، تشير `fun main()` إلى نقطة بدء تنفيذ برنامج Wave.
برنامج Wave يبدأ دائمًا من الدالة `main`.

دالة `println` هي دالة تقوم بطباعة السلسلة النصية إلى الإخراج القياسي وتستخدم بشكل أساسي لعرض النص على الشاشة.

## تشغيل البرنامج

عند الانتهاء من كتابة الكود، قم بفتح الطرفية (الترمينال) ونفّذ الأمر التالي في الدليل حيث يوجد الملف.

```bash
wavec run hello.wave
```

يأمر هذا الأمر المترجم Wave بتجميع ملف المصدر ثم تنفيذه فوراً.

## التحقق من الإخراج

إذا تم تشغيل البرنامج بنجاح، يظهر الإخراج التالي في الطرفية.

```
مرحبا ويف
```

إذا رأيت هذا الإخراج، فإن ذلك يعني أن Wave تم تثبيته بنجاح، وتم تنفيذ كتابة وتشغيل البرنامج بشكل صحيح.

لقد نجحت الآن في تشغيل أول برنامج Wave.
من الآن، يمكنك التعرف على بناء ووظائف Wave واحداً تلو الآخر وكتابة برامج أكثر تعقيداً.

يمكن العثور على الخيارات الدقيقة للأوامر (`-O*`, `--debug-wave`, `--link`, `--dep-root`, `--dep`) في وثائق `wavec` الخاصة بواجهة سطر الأوامر.
